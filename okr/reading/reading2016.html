2. 驾照(驾驶技术) · 今年暑假同时考过了科目三和科目四，于8月26日拿到了机动车驾驶证，新技能get！✅
3. Git & Github · 基本掌握了Git 多人协作的用法，终于理解了如何用 Github 来帮助自己管理和展示项目！新技能get！✅
4. 扇贝英语 · 成功养成了每日打开的习惯，现在单词量已经突破 15000 词，今年考六级时 听力写作翻译 明显轻松了很多。
5. 跑步 & 健身 · 保持住了15年的好习惯：给自己带来了更好的 身体。曾练就了6块腹肌并掌握了健身的方法。
6. 阅读 & 思考 · 保持住了15年的好习惯：给自己带来了更好的 思想。懂得反思自己，懂得宽恕别人！
    ❗️感叹：“严于律己,宽以待人” 真是至理名言啊
7. 搬家入小区 · 这对于“安土重迁”的我们家庭来说，不得不说是件大事，自此之后也可以说算是通公交的城里人了。
⭐️ 感悟：遇到问题多反思反思自己，少去抱怨环境抱怨别人！以平和心态去看待这个世界上的差异性(不要做愤青)！
