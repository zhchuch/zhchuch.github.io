少有人走的路

作者：[美]M·斯科特·派克

  • 这是一本通俗的心理学杰作！
  • + - 第一部分：自律
    • = 问题和痛苦

      • 人生苦难重重！这是个伟大的真理，它的伟大在于人们一旦想通了它，就能实现人生的超越！【联想<平凡的世界>里面的孙少平孙少安的生活经历】

        只要我们知道人生是艰难的，那么我们就再不会对人生的苦难耿耿于怀了。

        - 解决人生问题的首要方案是：自律。

        • 人生是一个面对问题并解决问题的过程。
        • 所谓自律，是以积极而主动的态度，去解决人生痛苦的重要原则，它主要包括四个方面：推迟满足感，承担责任，尊重事实，保持平衡！这几个方面背后的原动力只有：爱！
    • = 推迟满足感
      • 推迟满足感意味着 不贪图暂时的安逸，重新设置人生快乐与痛苦的次序。
      • 首先我们应 面对问题并感受痛苦，然后解决问题并享受更大的快乐，这是唯一可行的生活方式(也即正确的次序)。
      • - 儿童教育：小学的孩子正确对待家庭作业，是实践 "推迟满足感" 的最佳手段：
        • 孩子满12岁时，要无需父母督促，首先做完功课，再去看电视；
        • 到了15,6岁之后，他们的实践应该更加得心应手；
        • 到了青春期，他们处理类似问题，应该形成一种习惯或常态！(这也是青少年所应具备的健康的状态)
        • 总之要让孩子养成 "先吃苦，后享受" 的意识！
        • 儿童教育(青少年教育) 所应该养成的这种意识，【家庭教育】在其中起到了相当大的作用！
    • = 父母的过错

      • 在年幼的孩子心中，父母就像上帝那样位高权重。孩子缺乏其他模仿的对象，自然把父母处理问题的方法全盘接受下来，并且将之视为金科玉律。父母懂得自律、自制和自尊，生活井然有序，孩子就会心领神会，并奉之为最高准则；父母的生活混乱不堪、任意妄为，孩子们同样照单全收，并视为不二法门！

        父母的爱心至关重要，即便家庭生活混乱，倘若有爱存在，照样可以培养出懂得自律的好孩子！

        爱，关乎心灵的健康成长！我们爱某样东西，就会乐于为它付出时间。对子女的爱也同样如此，我们可以花更多的时间

        去照顾他们、陪伴他们。让孩子学会自律，也需要时间。父母习惯用严厉的体罚教训孩子，本质上不是教育，而是发泄自己的怨气与不满！

        - 对孩子自信心的培养：

        • 天生我材必有用 这种自信须从幼年培养。孩子幼年就享受到父母的爱，成年后即便遭遇天大的挫折，幼年培养的强大自信，也会使其鼓足勇气，勇敢的战胜困难，而不致自暴自弃！
    • = 解决问题的时机
      • 我相信除非存在智力障碍，不然只要花时间学习，就没有什么解决不了的问题！
      • 忽视问题的存在，反应出人们不愿推迟满足感的心理！
      • 问题通常不可能自行消失，若不解决，就会永远存在，阻碍心智的成熟。
      • 当问题出现我们【应该尽可能早地面对它们】，这样就意味着推迟满足感！
    • = 承担责任
      • 趋利避害，逃避责任，这种心理趋向令人遗憾，有时甚至达到可笑的程度！
    • = 两类心理疾病
      • - 神经官能症：遇到困难总以为是自己的过错，把责任揽给自己；
      • - 人格失调症：遇到困难总认为是别人的错误，把责任嫁祸别人。
    • = 尊重事实
      • 人在贫穷时，是一个世界；在富裕时，活的又是另一个世界。
    • = 迎接挑战
      • 固步自封，逃避挑战，可说是人性的基本特征之一。逃避挑战是人类的本能，但不意味着它是恰当的态度，不意味着我们无法做出改变。
      • 人之所以为人，或许就在于我们可以超越本性，乃至于改变本性，尝试去做不合本性的事情。
    • = 保持平衡
      • 自律，是艰苦而复杂的工作，你需要拥有足够的勇气和判断力。你以追求诚实为己任，也需要保留部分事实和真相。
      • 你既要承担责任，也要拒绝不该承担的责任。为使人生规范、高效、务实，必须学会推迟满足感，要把眼光放远。
      • 还要尽可能过好眼下的生活，要通过适当的努力，让人生的快乐多于痛苦。
      • 要使心智成熟，就须在彼此冲突的需要、目标、责任之间，取得微妙的平衡，这就要求我们利用机遇，不断自我调整。
      • 保持平衡的最高原则就是“放弃”。放弃人生的某些东西，一定会给心灵带来痛苦，但是失去平衡，远比放弃更为痛苦。
      • 回避放弃只有一个方法，那就是永远停在原地，不让双脚踏上旅途。
      • 为了放弃，我们首先必须拥有某种事物！
    • = 抑郁 的价值
      • 抑郁 是一种显著的信号——要求你要适应新的状况，要求你做出改变！
  • + - 第二部分：爱

    • = 爱的定义

      • 自律的原动力 —— 爱！爱分成很多种：肉体之爱，精神之爱，手足之爱，完美的爱，不完美的爱等等。
      • 本书的定义：爱，是为了促进自我和他人心智成熟，而具有的一种自我完善的意愿。
      • 爱与不爱最显著的区别之一，在于当事人的意识思维和潜意识思维的目标是否一致，如果不一致，就不是真正的爱。
      • 人人都有爱他人的愿望，但很多人只停留在想法和口头上，爱的愿望不等于爱的行动，真正的爱是行动！
      • 真正的爱是基于灵魂的行动。你认为自己爱他人却没有躬身实践，就等于从来没有爱过！

      = 坠入情网

      • 只有意识和潜意识的性冲动，才会使我们坠入情网。其次，坠入情网的"爱"不会持续太久，不管爱的对象是谁，早晚我们都会从情网的羁绊中"爬出"。要了解坠入情网的本质，我们必须认识心理学上所谓的"自我界限"。

      对自我界限的认识和发展，可以解释为什么小孩子都喜欢强大的超人、太空飞侠之类的故事。

      • 无自我意识时期

        • 这是因为孩子刚刚出生下来的最初七个月里，他们还无法分辨自我和外部世界的界限。当他在地板上爬来爬去的时候，以为整个世界都跟着一起移动；当他感到饥肠辘辘的时候，以为整个世界都与他一块儿挨饿；他看见母亲活动身体，以为自己也跟着母亲一同活动；母亲哼唱着摇篮曲，他以为那是他自己的声音。在新生婴儿的感觉里，在一切移动和固定的事物之间，在他和周围的人群之间，在单个个体和整个世界之间，没有任何的界限和差别。

      • 自我意识形成时期

        • 随着婴儿慢慢成长，认识和经验不断增加，于是发现他和世界不是一回事。他感觉饥饿，母亲不见得立刻出现，并给他饮水喂食；他想玩耍时，母亲未必能及时配合，与他一同玩简单游戏。他的意愿和母亲的行为，是截然不同的两回事。在这种情况下，婴儿的"自我"产生，自我意识也就开始出现。随着婴儿的成长，这种能够区别出自己和外在世界不同的认知就是所谓的 "自我界限"。

      • 幻想时期(接受现实时期)

        • 随着自我界限的认识和发展，孩子到了两三岁的时候，更加能够认识到自己能力的有限；在此之前，他无法让母亲完全按照自己的愿望行事，但他仍然会把自己的愿望与想法，同母亲的行动混为一谈。两三岁大的孩子，往往是家里的"小皇帝"，事不顺心，就会大发雷霆，甚至闹得天翻地覆。到了三岁，孩子的态度有所收敛，对自己的能力的局限性有了更加深刻的认识，但脑海里还是幻想如何随心所欲。这样的心态只有再过几年，当他经受到更多的打击以后，才能够逐渐消失。在此之前，他还是幻想自己无所不能，强大的超人，太空飞侠之类的故事，也就最受他们的欢迎。

      = 依赖性

      • 仅仅把得到别人的爱当成最高目标，你就不可能获得成功。想让别人真正爱你，恐怕只有让自己先成为一个值得爱的人！
      • 真正的爱，不是单纯的给予，还包括适当的拒绝、及时的赞美、得体的批评、恰当的争论、必要的鼓励、温柔的安慰、有效的敦促等。

      = 爱，不是感觉

      • 爱是实际行动，是真正的付出！

      = 关注的艺术

      • 爱最重要的体现形式：是关注！
      • 体现 "关注"，一种最常见也最重要的方式：是倾听！
      • - 儿童教育
        • 六岁大的孩子，在他们大量的话语中，只有少部分需要父母的关注和反应。父母最关键的任务
        • 之一：就是在听和不听之间做出恰当的选择，找到最佳平衡点，尽可能的满足孩子的需要。
        • 而真正的 "倾听" ，父母应该把别的事情放到一边，真正把倾听的时间用于孩子，而且它也必须
        • 是属于孩子的时间。不愿把别的事情放到一边，包括你的不佳情绪或别的念头等等，都会意味
        • 着你不愿真正的倾听。对于一个六岁孩子的话，往往是相当枯燥的，事实上，没有爱，父母就难以产生倾听的动力。
        • 你对孩子全心全意的倾听，能够让孩子感受到你的尊重，他们就会觉得，你把他们看成是出色的人。
        • 这样一来，他们也就更加愿意听你的话，并给予你同样的尊重。
        • 随着孩子年龄的变化，听与不听的平衡点也会改变，但总的原则是：不论年龄多大，孩子都需要
        • 【父母的关注与倾听】，而且要注意这种关注需要的是 "给予全部注意力"。
      • 倾听 是表达关注的最主要形式之一。

      = 承诺

      • 做出承诺，可以给一个人带来安全感。做出承诺，就必须要履行承诺。
      • 婚姻，是分工与合作并存的制度，夫妻双方需要奉献和关心，为彼此的成长付出努力。
      • 爱的神秘感 P129，在论述"爱"这一题目时，所举出的一个例子。
  • + - 第三部分：成长与宗教
    • = 科学的宗教
      • 本书的主旨之一是：探讨增加认知、扩大视野的意义。
      • 爱的本质，其实是自我完善，而爱的风险之一就是必须进入未知的领域。
      • 我们应该对过去的信仰提出质疑，主动探索陌生领域，挑战某些久被视为真理的结论。
      • 只有怀疑和挑战，才能使得我们走上神圣的自由之路。
      • - 科学的宗教
        • 旨在帮助我们从实际出发，采用现实的经验、历史的教训去认识世界，更新人生观、世界观。
      • - 科学

        • 其重要信条是：宇宙是真实的存在，我们可以对它进行观察，人类对宇宙的观察具有重要价值；宇宙的运行遵循若干规律，而且是可以预测的；人类易受偏见和迷信的误导，在解释宇宙时易犯错误；人类要形成世界观，理应具有足够的怀疑精神，理应接受科学方法的指导，由此总结出观察宇宙的经验。

          科学是一种以 怀疑 为基础的宗教。

  • + - 第四部分：神奇的力量
    • = 荣格 的 "集体无意识"理论 【潜意识的奇迹】[P180(195/241)]

      • 即便没有亲身经历而获得智慧，我们也可以继承祖先经历并获得智慧。

        举例：每当我们阅读一本书，或是看到一个场景，碰到一种我们喜欢的想法或理论时，书中或场景中的内容就会立刻引起我们模糊的记忆，或者使我们感觉到极其熟悉！产生一种 "似曾相识" 的神奇感受。

        这种 "似曾相识" 的体验，似乎我们从前就知道了这一事实，只是忘记了而已，后来，我们则像遇到老朋友一样认出了它。似乎所有的知识、所有的智慧，都存储在我们的思维库里。我们学习某种新东西，实际上只是发现了一直存在于脑海中的某种事物。

         

        这种想法，也许通过"education"这个词中可以反映出来一些，"education"源自于拉丁语中的"educare"，它的意思就是"带出来，带领到"的意思。因此我们"教育"别人的时候，如果认真的加以对待，其实就不是把某种新东西强塞入他们的思维，而是我们把这种新东西从其思维中引导出来，让它从潜意识进入到意识中。

        也就是说对方早已经是这些新知识的占有者。

        已有科学实验证明，把基因物质同记忆现象结合起来进行研究，结论证明：我们的基因很可能继承了知识，并在细胞里以核酸DNA遗传密码的形式存储！信息以化学的形式存储，这一观点使人们意识到，人类思维所获取的信息很可能存储在几立方英寸的大脑物质中。

        潜意识知道的事情，永远比意识多得多。我们获得一项真理，得到一种启示，不过是重新认识潜意识久已熟知的事情。获得新的真理和启示，其实是意识和潜意识达成一致的结果，它们具有了共同的认识和了解。

        [参见P202(216/241) - 意识的进化]

    • = 心理学上的 "同步原理"
      • 这种事情，在统计学上往往被认为是完全不可能出现的情形。但是在真实世界中总会不间断的发生着，这无疑就是存在着一股 "神奇的力量"。
    • = 进化的奇迹
      • 生物进化过程本身就是最大的奇迹，依照我们对宇宙的认识，进化过程其实并不可能发生，进化现象本来不应该存在。
      • 因为依照热力学第二定律，能量自然地从有规律的状态流向无规律的状态，从高度分化状态流向低度分化状态，宇宙处于逐渐分解的过程！(这种自然地熵减的现象，被称为"熵的力量"。)
      • 但是生物的进化，却是由低处向高处发展，在此过程中，他变得更加复杂，而且更具分化性，更有组织性。
      • 从滤过性病毒这种极其简单的有机体，到昆虫和鱼类，进化的阶梯不断复杂化，走向高度分化和有组织的状态。
      • 进化的阶梯发展到人类，已经具有了大脑和极其复杂的行为模式。这种进化过程显然是违反自然规律的！
      • 这种向上进化的力量，它经过数百万代的发展，不断地击败自然规律，而这种力量本身，就是一条尚待了解和界定的自然规律。这种 "进化力量" 就是一种典型的 "神奇的力量"。
      • 这种"熵的力量"针对于人就是 懒惰，而这里违反自然力量的 "进化的力量" 针对于人的心灵的完善就是 爱！
    • = 懒惰
      • 懒惰的一个主要特征就是："恐惧感"！
    • = 意识的进化

      • 潜意识是上帝，它无所不知，而我们的意识也具有一种罕见的强大力量，它掌管生命的一切运动，它负责做出决定，并把决定付诸实施。

        假如只有潜意识而没有意识，我们的生命就像新生的婴儿，即使心灵可以实现"人神合一"，我们仍然无法采取任何行动，也无法让人感觉到"上帝"的存在。

        在印度教和佛教的神秘思想中，存在着一种容易使人走向退化的特质——它们把没有自我界限的婴儿阶段比作 "涅槃"，进入 "涅槃" 就如同返回到母亲的子宫，进而获得重生。

        你只能自行选择人生道路，忍受生活的艰辛与磨难，最终才能达到人生新的境界。

  • 读后感

    • 通过阅读这本书，使我对如何面对人生中的苦难，如何辨别自己是否正处于需要进行改变的阶段有了更多的了解。同时对  "爱" 的本质和具体的体现形式(关注)有了更深的认识。另外对如何进行儿童教育，如何更好的表达自己的爱有了更多的了解。通过本书的前两个部分，对我了解儿童的成长过程中的心理活动也有了更多的了解。
