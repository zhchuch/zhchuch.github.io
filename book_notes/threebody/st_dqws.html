三体· 地球往事

作者：刘慈欣


这是一个以《地球往事》为名的科幻小说系列的第一部。《三体》三部曲讲述了地球文明在宇宙中的兴衰历程。

后记·反思
如果存在外星文明，那么字宙中有共同的道德准则吗？往小处说，这是科幻迷们很感兴趣的一个问题；往大处说，
它可能关乎人类文明的生死存亡。我认为零道德的文明宇宙完全可能存在，有道德的人类文明如何在这样一个宇宙
中生存？这就是我写《地球往事》的初衷。
好看的科幻小说应该是把最空灵最疯狂的想象写得像新闻报道一般真实。

读后感：
        这部作品想象出的一个三体文明，被作者描述的如此真实，里面的红岸工程以及智子工程好像真的存在一样，不得不佩服
        作者的这种文学功底强大 以及 知识的渊博。它可以说完全把人给带入了，让我产生了一种原始的 求知欲与探索欲。
        同时该作品也引发了我对人类文明以及外星文明的诸多思考，对道德、人性等的思考。同时其中描述的三体人能操纵的9维
        空间，以及对高维空间向低维展开的描述，可以说是令我再一次加深了<宇宙的琴弦>以及<平面国>这两本书中对于空间维
        度的理解。越来越觉得书本之间的很多知识真的是相通的，加强了我想多读书的想法，因为通过这可以把这种零碎的知识整
        理成一个统一的思维模式，从而提升自己的思想深度，提高自己的思维水平。

第一部：地球往事
1. 疯狂年代
    = 从文化大革命时期中国人的疯狂开始讲起。毛泽东对知识分子的迫害简直是不忍直视。
    = 从批判一个物理学教授叶哲泰教授含有反动言论的理论，开始渐渐引出科学。叶哲泰最后被红卫兵迫害而死，会场上只剩下了
        他的女儿——叶文洁。叶文洁本身是 天体物理专业的在读研究生。
2. 寂静的春天 《Silent Spring》
    = 视线转到两年以后的大兴安岭。
    = 边境兵团陷入到砍伐森林建设新土地的队伍中，这真的是一种很蠢的发展所谓社会主义的办法。叶文洁经常抚摸那些被砍倒的
        大树的横截面。也由此认识了来自兵团《大生产报》的记者——白沐霖。
    = Silent Spring 给叶文洁思考人类本身的许多行为一种新的视角，她认识到人类真正的道德自觉是不可能的，要做到这一点，
        只有借助于人类以外的力量。
    = 叶文洁的妹妹叶文雪是一个最激进的红卫兵。她曾积极主动揭发自己的父亲。简直不能理解这些人，似乎已泯灭了人性。
    = 1964年中国搞了震惊世界的 两弹工程，这个工程处于中央的重点保护之下，得以避开 "文革" 的血雨腥风。
3. 红岸之一
    = 叶文洁因自己于1966年在《天体物理学杂志》中发表了一篇题目为《太阳辐射层内可能存在的能量界面 和 其反射特性》的英文
        论文，然后被 红岸基地的雷志成政委以及杨卫宁总工程师 选中，算是脱离苦海。
        叶文洁又被 红岸组织 接回到了地处大兴安岭的雷达峰上。
    = 叶文洁回想到自己家庭所遭受到的迫害，感觉对这个世界已经生无眷恋了，对这个世界留下的全是不好的印象，所以她毅然决然
        的与 红岸组织 签下了终身契约，承诺愿意帮助他们一起进行研究。
    = 红岸工程 在 雷达峰上的研究项目是 关于一个大型武器的研究，如果成功，其意义可能比两弹(原子弹和氢弹)还要大。
    = 红岸工程 是 隶属于中国人民解放军第二炮兵。
4. 三十八年后
    = 视线转向了 新人物 汪淼。
    = 新人物 汪淼(是一名教授) 是一位与 "科学边界"学会 有联系的学者，"科学边界"则是一个在国际学术界上很有影响力的学术组织，
        成员都是著名学者。汪淼本身也是一位研究 "纳米材料" 的著名学者！
    = 汪淼 被军方请过去参加一个由军方组织起来的会议，这里是军方口中的“作战中心”。
    = 一年前，汪淼是 "中华二号" 高能加速器项目纳米构件部分的负责人。
    = "科学边界"这一学术组织的宗旨：自上个世纪下半叶以来，物理学古典理论中的简洁有力渐渐消失了。理论图像变得越来越复杂、
                        模糊和不确定，实验验证也变得越来越难，这标志着物理学的前沿探索似乎遇到了很大的障碍和困难。科学似乎走到
                        了边界。
5. 台球
    = 物理学伟大的定律：物理规律在时间和空间上是均匀的！
        人类历史上的所有物理学理论 以至 人类迄今为止的一切科学发现和思想成果，都是这个伟大定律的副产品！

    = 引出问题：假如物理规律在时间和空间上不均匀，那么就意味着宇宙普适的物理规律不存在，那么物理学也就不会存在了。
6. 射手和农场主
    = 这是两个假说：
        "射手假说"：有一名神枪手，在一个靶子上每隔十厘米打一个洞。设想这个靶子的平 面上生活着一种二维智能生物，它们中的科
                            学家在对自己的宇宙进行观察后，发现了一个伟大的定律： “宇宙每隔十厘米，必然会有一个洞。”它们把这个神
                            枪手一时兴起的随意行为， 看成了自己宇宙中的铁律。
        "农场主假说"：一个农场里有一一群火鸡，农场主每天中午十一点来给它们喂食。火鸡中的一名科学家观察这个现象，一直观察了
                            近一年都没有例 外，于是它也发现了自己宇宙中的伟大定律：“每天上午十一点，就有食物降临。”它在感恩节早晨
                            向火鸡们公布了这个定律，但这天上午十一点食物没有降临，农场主进来把它们都捉去杀了。

    = 汪淼在自己拍摄开到了的胶卷上看到了 诡异的倒计时。
        他想到了 "科学边界"组织，那是一群思想深刻而且活跃的人。他想让他们帮助解答自己的困惑。他们中的申玉菲对这件事好像并不
        感到吃惊，反而让他 停止自己领导的纳米项目的研究工作。

    = 科学组织成员 潘寒 的观点：科技革命是人类社会的一种病变，技术的爆炸性发展与癌细胞的飞速扩散相当，最终的结果就是耗尽
        有机体的养分，破坏器官，导致其寄宿体的死亡。
7. 周文王·长夜
    = 汪淼找 丁仪 帮助寻找观测宇宙背景辐射的机构。
    = http://www.threebody.com 原来三体是一个游戏。
        在这个游戏的世界里，存在着 乱纪元 和 恒纪元 两种纪元，分别以 一颗飞星出现在夜空，两颗飞星出现在夜空里作为各自的标记；
        当 天空中 出现三颗飞星时，代表的是：漫长的严寒岁月，冷得能够把石头冻成粉末！
        在这个严寒岁月里：下的雪依次是由 水蒸气凝结 -> CO2 凝结 -> O2 凝结 -> N2 凝结，雪花的颜色由 白色 渐渐转为 淡蓝色。
8. 叶文洁
    = 原来，叶文洁就是 那个自杀的物理学家 杨冬 的母亲。
    = 叶文洁给汪淼联系好了用于观测宇宙背景辐射的地方——中科院国家天文观测中心的射电天文观测基地。
9. 宇宙闪烁
    = 汪淼 通过观测宇宙背景辐射(宇宙出现了闪烁)，利用莫尔斯电码破译出了宇宙一直在进行着一种可怕的倒计时。
    = 但是这种 幽灵般的倒计时 的终点是什么，他始终无法理解。
    = 但这一切好像都被 "科学边界" 以及 那些由军方组织起来的"作战中心" 中的人所了解了，他们好像对这一切都了如指掌。
10. 大史
    = 讲述 大史与汪淼 之间的谈话，大史把自己知道的全部有关这场战争的事情全部都告诉了 汪淼。
    = 并且 临走时 大史让汪淼继续去研究一下 三体这款游戏。
    = 大史 给 汪淼说的一条自己的格言： "邪乎到家必有鬼，这里的'有鬼'不是说真的有鬼，而是说是有人在捣鬼！"。

11. 三体·墨子·烈焰
    = 汪淼回到家里之后，再次登录了《三体》，遇到了墨子。
    = 墨子 提出了自己对宇宙的观测 而得出的一个 宇宙模型 —— "球壳模型"
        宇宙是一个悬浮于火海中的大 空心球，球上有许多小洞和一个大洞，火海的光芒从这些洞中透进来，小洞是星星，大洞是太阳。
        经过 修正的模型，宇宙之球是由 两层球壳 构成的，我们看到的天空是内层壳，外层球壳上有一个大洞，内层球壳上有大量小洞，
        那个外壳上的大洞透进的光在两层球壳之间的夹层反射和散射，使夹层间充满了亮光，这亮光从小洞中透进来，我们就看到了星星。
        太阳是外层壳上的大洞投射到内层壳上的巨大光斑，它的亮度如此之高。像照穿鸡蛋壳一般照穿了内壳，我们就看到了太阳。光斑
        周围的散射光较强，也照了内壳，这就是我们白天看到的晴空。

    = 人到老了就会发现：当年以为天要塌下来的那些大事，其实说到底没有什么的。杨母叶文洁开始向汪淼讲述当年红岸的事情。

12. 红岸之二
    = 视线 转向 叶文洁，回顾自己在红岸的日子。
    = 监听部 的主要工作内容 就是 对敌人的太空活动进行监视，包括监视敌人航天器与地面和航天器之间的通讯等。
                   可以说是 整个红岸工程的眼睛。

13. 红岸之三
    = 讲述 "红岸工程" 的真实情况：
        - 世界基础科学研究趋势中的一个被忽略的重要问题
            目前，北约和华约集团基础研究空前活跃，投入巨大。所以一项或多项 技术突变 随时都可能发生，这将对我国战略规划构成
            重大威胁。寻找外星文明：这是所有技术突变的可能性中变数最大的领域，极有可能产生突然性的巨大突破！

        - 外星文明探索技术突变可能性研究报告
            主要担心超级大国首先与外星文明接触并且垄断接触的危险，美国和苏联已经开始向着地球外面喊话了，外星社会只听到一个
            声音是危险的，我们也应该发出自己的声音。所以探索外星文明，这件事情我们不仅要做，而且要快做。

        - 红岸工程前期研究报告
            工程目的：探索可能存在的外星文明，并尝试与之建立联系和交流。
            红岸信息发送内容包括：地球行星概况 3.1KB，地球生命系统概况 4.4KB，人类社会概况 4.6KB，世界历史基本信息 5.4KB，
                                                   总计 17.5KB 内容。
        - 对外星文明发送信件

14. 红岸之四
    = 红岸工程，由于长时间收不到成效，最终被降级，知道最后差点被完全抛弃。
       后来在雷志成政委的努力下，红岸基地后来又引进了外星文明探索之外的项目，使得红岸基地得以保留下来。

15. 三体、哥白尼、宇宙橄榄球、三日凌空
    = 叶文洁的故事讲完了，视线转回现实。
    = 红岸的故事讲完了，汪淼又回到了现实生活中。他又打开了三体这个游戏，重新注册了一个叫"哥白尼"的 ID，
        他 想要揭示三体世界的秘密。
    = 他 提出了自己对三体世界 的 宇宙模型 —— "橄榄球模型"
        其实很简单：太阳的运行之所以没有规律，是因为 我们的世界中有三颗太阳，它们在相互引力的作用下，做着无法预测的三体
        运动、当我们的行星围绕着其中的一颗太阳做稳定运行时，就是 恒纪元；当另外一颗或两颗太阳运行到一定距离内，其引力会
        将行星从它围绕的太阳边夺走，使其在三颗太阳的引力范围内游移不定时，就是 乱纪元；一段不确定的时间后，我们的行星再
        次被某一颗太阳捕获，暂时建立稳定的轨道，恒纪元就又开始了。这是一场宇宙橄榄球赛，运动员是三颗大阳，我们的世界就
        是球！
     = 三颗飞星 —— 意味着严寒的到来；
        三日凌空 —— 意味着行星地表会在瞬间变成冶炼炉，它会熔化一切事物，这是 三体世界中最大的灾难。
    
    = 最终 哥白尼(汪淼) 成功揭示了三体世界宇宙的基本结构，三体文明也将产生第一次飞跃，三体游戏升级为第二级。

16. 三体问题
    = 讲述三体问题的由来，别人对该问题的研究。
    = 魏成 对这个问题 提出了自己的一种研究算法 —— "进化算法"
           他研究三体运动中的任何一个时间断面，在这个断面上，各个球的运动矢量有无限种组合，我将每一种组合看成
           一种类似生物的东西，关键是要确定一个原则：哪种组合的运动趋势是"健康的"和"有利的"，哪种是"不利的"和
           "有害的"，让前者获得生存的优势，后者则产生生存困难，在计算中就这样优胜劣汰，最后生存下来的就是对三体
           下一个断面状态的正确预测！
    = "科学边界" 组织 分为 "降临派" 和 "拯救派" 两大相互为敌的派别。

17. 三体、牛顿、冯·诺依曼、秦始皇、三日连珠
    = 汪淼 成功预测了三体世界里的宇宙模型是"橄榄球模型"之后，游戏升级了。他现在来到了第二级。
        这一次 他在这个世界里的角色是 牛顿！
    = 整个三体世界的历史 以及 时空 都是混乱的，牛顿、冯诺依曼 来到了中国古代的秦国。
    = 冯诺依曼 让 秦始皇派出三名士兵，然后 每名士兵一人手里拿着一面黑旗一面白旗(分别代表 0 和 1)，然后组成了一个 
        与(AND)门；然后他又找来三名士兵，使用相同的方式构件了一个 或(OR)门；然后 他又用士兵构件了 非(OR)门，与非门，
        或非门，异或门，同或门，三态门，从而集齐了计算机所需要的所有最基本的计算组建(CPU中的计算单元)。
    = 冯诺依曼 用秦国的士兵 搭建出了一个计算机，包含了[CPU(运算器)，主存(存储器)，控制器，总线(Bus)，输入设备，输出设备，
        外存(硬盘)]。
    = 然后他们在这台由三千万秦国士兵所组成的计算机上面，开始启动 太阳轨道计算软件 'Three-Body 1.0'

    = "三日连珠" 三颗太阳连成了一条直线，所带来的就是对地面引力骤增为原来的三倍，三体世界表面的一切都被吸向太阳。
        这一次进化到 秦国的文明 最终在 "三日连珠" 的引力叠加中被毁灭了。

18. 聚会
    = 玩 "三体"游戏的所有网友 举办了一个聚会。三体这个游戏的玩家居然都是这个社会的精英人士。这个聚会的组织者是 潘寒。
    = 潘寒 说 三体不仅仅只是个游戏，三体世界本身也是真实存在的！
        三体人确实真实的存在，在每一轮文明中，他们的外形也都是完全不同的。
    = 潘寒对这几个人进行了一轮面试，最后淘汰了两个人。淘汰的对象是 反对三体文明进入人类世界的两个人。

19. 三体、爱因斯坦、单摆、大撕裂
    = 汪淼 第五次进入三体游戏，此次他遇到了 爱因斯坦，爱因斯坦向他解释为什么 牛顿 的理论会预言失败，那是因为：
        牛顿 不知道要在他的经典方程中加入 引力摄动 的修正(也即自己发现的狭义相对论和广义相对论)。
    = 汪淼 看到了一个巨大的月亮，最后被告知这其实已经不是一个月亮了，而是这颗行星的一颗伴星，它是大撕裂的产物！
        这个 大撕裂 是由于受到飞过来的巨大的太阳的 潮汐力作用，而最终被这种强大的力量 撕碎！
    = 三体系统 最终被证明无解，完全没法被预测，所以 三体人 准备飞出三体星系，飞向辽阔的星海，在银河系中寻找可以移民的
        新世界。
    = 于是三体游戏的的最终目标也发生了变化，新的目标是："飞向宇宙，寻找新家园"。

20. 三体、远征
    = 此次 汪淼再度进入三体游戏，看到了 伟大的三体星际航队，三体人马上就要起航远征了。
    = 三体人远征的目标是距离他们最近的四光年外的一个拥有行星的恒星。而距离我们太阳系这个恒星系统最近的恒星也是四光年。
        也就是说他们远征的目标就是我们的太阳系！

    = 至此 三体游戏 结束了，三体游戏邀请汪淼参加三体组织的聚会！

21. 地球叛军
    = 汪淼 参加了聚会，第一次在现实中看到了三体运动。
    = 潘寒是 "科学边界" 组织中负责环境分支，属于 "降临派" 。他 与 申玉菲 这种 "拯救派"的组织成员之间就是死敌。
        环境分支的任务是：制造环境灾难，并利用这些灾难以激起人们对科学和现代工业的厌恶。

    = 原来 欧洲和北美以及中国 所从事的外星文明探索工程 是 一次对三体组织的大搜捕。
        原来 叶文洁 是 地球三体叛军的最高统帅。这个组织宣称："世界属于三体"。该组织 建立起了一个 第二红岸基地。
    = 汪淼是中科院院士，主要研究纳米材料，而这种技术是主首先要在地球上扑灭的技术。面对汪淼的困惑，叶文洁继续向他讲述
        红岸的故事。

22. 红岸之五
    = 继续讲述 红岸工程的故事。"凌日干扰"问题一直困扰着红岸的监听操作，而根据叶文洁发表的一篇论文，他们选中了她。
    = 经过叶文洁的研究发现：太阳是一个电波放大器！这个研究结果的意义在于：人类可以将太阳作为一个超级天线，通过它向宇宙
        中发射电波。这种电波是以恒星级的能量发出的，它的功率比地球上能够使用的全部发射功率还要大上亿倍。地球文明有可能进
        行 Ⅱ型文明能级 的发射！
    = 最后通过一次实验叶文洁本以为自己的理论失败了，但是这却是 地球文明向太空发出的第一声能够被听到的啼鸣。此时，
        在12000MHz的波段上，太阳已经成为整个银河系中 最亮的一颗恒星！

23. 红岸之六
    = 叶文洁 根据自己的经历 以及 她看到的外籍文献 得出的结论是：人性是恶的。
    = 人类第一次读到了来自宇宙中的另一个世界的信息。离我们太阳系最近的恒星系统就是：半人马座三星系统(相距 4 光年)！
        叶文洁在随后的信息破译中，知道了三体世界的存在，知道了一次次浴火重生的三体文明，也知道了他们星际移民的企图。
    = 这个女人太自以为是了，居然没有和任何一个人商量，就把整个人类暴露在宇宙中。她背叛了人类。
24. 叛乱
    = 讲述 大史警官 带队抓获三体组织内的 地球叛军的事情经过。
25. 雷志成、杨卫宁之死
    = 抓捕住她们之后，开始了对叶文洁的审问工作。她交代自己在1979年杀死了雷志成和杨卫宁。这个女人已经完全到了心理变态的
        程度了。
26. 无人忏悔
    = 文革结束之后，很多原来受到迫害的人的名誉和地位得到了恢复。叶文洁本来想要为父亲讨回公道，但是即使是当年的行凶者对
        于那个时代中枉死的人也是没有丝毫忏悔。这让叶文洁认定了人类 恶 的本性。

27. 伊文斯
    = 叶文洁已经坚定了自己要将 宇宙空间中更高等的文明 引入人类世界的决心。
    = 叶文洁回到学校之后，在一次课题的选址中遇到了 励志当一种鸟类的救世主的伊文斯。
    = 伊文斯信仰 "物种共产主义"，这种信仰的来源是 从东方产生的佛教，佛教强调众生平等，要普度众生；而基督教则是只重视人，
        虽然所有物种都被放入了诺亚方舟，但是从来没有给生命与人类同等的地位！
28. 第二红岸基地
    = 伊文斯 为了实现三体文明进驻地球的目标，在一艘改造过的油轮上建立了"第二红岸基地"。
    = 在该基地 他们接收到了来自三体世界的信息，得知伟大的三体舰队已经起航，目标是太阳系，将在四百五十年后到达。

29. 地球三体运动
    = 这个地球三体运动组织 像社会传播三体文化的主要途径就是通过《三体》游戏。
    = 三体组织 中的“拯救派”的最终理想是：拯救主。为了能够使主生存下去，可以一定程度上牺牲人类，但是他们中的大多数人认为，
        能够使得主在三个太阳的半人马星系中生存下去，避免其对太阳系的入侵，是一件两全其美的理想结局。
        所以他们一直致力于在物理上解决三体问题，从而同时拯救三体和地球这两个世界。
    = 三体组织 中的 “降临派”的最终理想是：引入外星文明，希望借此毁灭地球文明。它们对人类本性已经彻底绝望，他们对于人类
        只有憎恨。 他们的座右铭是："我们虽然不知道外星文明是什么样子，但知道人类"。
    = 三体组织 中的“幸存派”，他们只是寄希望于当入侵太阳系的外星舰队的存在被确认之后，在那场终极战争中能够幸存下来。

30. 两个质子
    = 视角再切换回到 对叶文洁的审判。
    = 对三体组织的总结：
        看看地球三体组织现在的局面：降临派要借助外星力量毁灭人类，拯救派把外星文明当神来崇拜，幸存派的理想是以出卖同胞来苟
        且偷生，所有这些都与你借助外星文明改造人类的理想不一致。
    = 三体飞船的推动力是：正反物质的湮灭。
    = 三体文明将两个质子加速到光速，发往地球，这两个质子就是一把锁，它锁死了人类的科学，因为这两个质子的存在，人类的科学
        注定将不可能有任何重大的进展。

    = 过滤嘴中的海绵或活性炭是三维体，它们的吸附面则是二维的，由此可见，一个微小的高维结构可以存贮何等巨量的低维结构。
        但在宏观世界，高维空间对低维空间的容纳也就到此为止了，因为上帝很吝啬，在创世大爆炸中只给了宏观宇宙三个维度。但
        这不等于更高的维度不存在，有多达八个维度被禁烟在微观中，加上宏观的三维，在基本粒子中，存在着十一维的空间。
       【联想：《宇宙的琴弦》以及 《平面国》中对 高维空间(高到 11维空间)以及低维空间(低到 0 维空间)的直观解释】
        高维度 对 低维度 的影响，高维空间生物的智能与科技估计在是身处低维空间里的人永远也想不明白的。

31. 古筝行动
    = 这里交代了在第4节汪淼所参加的会议中所描述的作战中心，真正的作战任务是："人类与外星侵略者之间的战争"。
        执行这一作战任务的首要目标就是："保证'审判日'号上的三体信息的安全并夺取到它"。
    = 消灭一个封闭结构内部的工作人员，同时对其中的其他设备又不能造成破坏，我们首先想到的就是利用 球状闪电武器，但是随后
        想到的各种方案都被一一否决了，直到史强提出了一个使用"纳米丝"构成的网状结构最终被敲定了。
    = 这次行动的代号是"古筝行动"，而这些纳米丝所构成的切割网则被称为"琴"。最终 古筝行动 取得了成功，他们取得了三体信息。
        得到了约有 28GB 的大量数据。这些数据之中描述了三体文明到底是什么样子。

32. 监听员
    = 三体世界里也存在着 一直在寻求外星文明的监听员。当他们知道了地球这种永远是恒纪元的星系存在之后，简直就是把这里当成了
        天堂。
    = 但是由于不知道这些信号是什么时候发出的，所以三体人不知道外星地球文明具体在什么位置，只知道它的方向。所以他们决定
        像那个遥远的世界发出信息，引诱地球文明上的生物做出回答，从而确定地球文明离自己所处的三体文明的实际距离！

        最终这个监听员为了使自己卑微的生命活的更有价值，像地球文明抛出了警告，但对三体文明来说却是一个错误的信号：
       “这个世界收到了你们的信息。我是这个世界的一个和平主义者，我首先收到信息是你们文明的幸运，警告你们：不要回答！不要
           回答！不要回答！！！你们的方向上有千万颗恒星，只要不回答，这个世界就无法定位发出源。如果回答，发射器将被定位，你
           们的文明将遭到入侵，你们的世界将被占领！不要回答！不要回答！！不要回答！！！ ”

    = 对于三体文明所生活的那颗仅剩的行星，随时都有可能被该星系中的三颗太阳其中的任意一颗的气层膨胀而吞没！
        所以它们必须移民外星，入侵外星文明。

33. 智子
    = 由于 叶文洁当年的一意孤行，立即回复了来自三体文明的信息，导致三体文明里的生物确定了我们就在 4光年外的太阳这个恒星
        系统里，这对它们来说简直就是个天大的好消息。
        但是地球文明里的生物具有一个可怕的能力，那就是 —— 加速进化能力。

    = 所以三体人想到的第一个任务就是：遏制地球文明的科学发展。为了完成这一任务，三体人做了一个"智子工程"(对应于地球的
        红岸工程)！“智子工程”的目标就是：把一个质子改造成为一台超级智能计算机！
        科学技术的全面发展取决于基础科学的发展，而基础科学的基础又在于对物质深层结构的探索，如果这个领域没有进展，科学技术
        整体上就不可能产生重大突破。

    = 三体文明中的 物理学家 已经能够操纵微观世界十一维空间结构中的 九个维度！他们想要将自己能够控制的九维空间展成一个巨大
        的二维平面。于是就开始对质子进行展开，0维展开就形成了黑洞中的奇点，1维展开就是一条长达几光时的线，2维展开就是一个
        巨大的平面，3为展开就会形成布满天空的杂乱的三维体。
    = 三体人在两次错误的展开实验中 掌握了从微观高维到宏观低维展开的规律！
    = 从某种意义上来说：每一个质子，都是一整个微宇宙，在这个宇宙中也可能存在着智慧或文明，而且它们可能还不止一个！

    = 当维度升高到六维时， 里面的人看外面处在三维时间的人的状态是：“能够看到其中的每个人，还能看到每个人的内脏，甚至还能
        看到你们内脏的内脏。”
       【联想：《平面国》中对于低维空间和高维空间的理解，高维空间内的生物始终能看到低维空间内生物的全部(包括它们的内脏)】 

    = 建立好了 智子一至四号之后，它们之间构成了一个 量子感应阵列！这个阵列的尺度可以达到任意大小，可以接收所有频段的电磁波，
        也就可以感知宏观世界。两个智子到达地球后，第一个任务就是定位人类用于物理学研究的高能加速器，然后潜伏于其中企图搞破坏。
        当智子被撞击后，它就会有意给出错误和混乱的结果，即使偶尔有对预定靶标粒子正确的撞击发生，地球物理学家们也不可能将正确
        的结果从一大堆错误结果中分辨出来。这样就可以使得人类对基础物理的研究停滞下来，从而锁死人类的科技发展！

    = 利用 质子之间的量子感应原理，这种作用是超距的，这样四个智子分处宇宙的两端，感应依旧可以在瞬间传递到，从而实现三体世
        界对地球的实时监控，同时，智子阵列也使得三体世界能够与地球文明中的异己分子进行实时通讯！

34. 虫子
    = 三体人把地球人视作为虫子。
    = 三体科学家的高明之处，在于他们对粒子低维展开的同时保留了高级结构的信息，使整个过程成为可逆。
        对物质深层结构的研究是其他一切科学基础的基础，如果这个没有进展，那么什么都是扯淡！就比如：如果我们不知道物质是由
        原子、分子组成的，我们就永远造不出所谓的原子弹，导弹，卫星等等。

    = 把人类看做虫子的三体人似乎忘记了一个事实：那就是 虫子从来就没有被真正战胜过！
        这句话算是 一语点醒梦中人，使得那些所谓的科学家学者们又燃起了斗志。

35. 尾声·遗址
    = 叶文洁在知道自己当初一意孤行给人类带来的重大灾难之后，自己绝望了。凭借自己生命中的最后一丝气血，她爬到了红岸基地的
        旧址，回忆了自己往昔所做过的一切，最后望着西方天际处血红色的落日，感叹了一声:"这是人类的落日"。
