高效能人士的七个习惯
作者：[美]史蒂芬·柯维

  • 前言：变化的世界，不变的原则

    • + 我对生命的一种最深刻的感悟就是：要完成最渴望的目标，要战胜最艰巨的挑战！

       

      + 在人的心灵深处，没有比渴望理解更强烈的需要了：希望他人聆听、尊重、重视自己的声音，希望自己能影响到别人！

       

      + 人的本质是思维的：身体、精神、智力(头脑) 和 情感(心灵)。

      • - 怎么利用这四个维度来提高自己，避免自己陷入停滞不前？

        • · 身体上：调整现在的生活方式，恰当的运用健康原理，预防疾病和其他健康问题；
          · 精神上：遵循高效人士的“七个习惯”，这些自然法则冥冥中自有天意，满足了我们追寻生命意义的基本要求，并鼓励我们探索生命的美好；
          · 智力(头脑)上：开拓阅读的深度和广度，保持终身学习的习惯；
          · 情感(心灵)上：尊重对方，专注于聆听，并无私地奉献他人，这将为你带来巨大的幸福感和满足感。

       

      + 人的右脑 主管直觉、创造、想象力和印象，左脑  则主管逻辑、分析与语言、计划(在规定时间内完成任务的能力)，换言之：左脑负责理性，右脑负责感性，我们自己的左右脑也需要统合综效！

       

      + 作者说道：阅读本书切记，最重要的，是在生活中践行它们！
      - 记住：只学不做等于没学，只知不做等于无知！

     

    总结：

    • + 本书对我产生最大冲击力的 Top-Five 五个观点：

      top-1. “外界刺激与自我回应之间存在着一段距离，我们的成长和幸福的关键就在于我们如何去利用这段距离”！我们要致力于跳出自己的角色，成为自己与这个世界(环境)之外的旁观者！“积极主动”的改变自我回应，“积极主动”的影响外界刺激！“由内而外地改变自己”！请记住：你是创造者，你能掌控自己的人生。


      top-2. “抱怨环境”是懦夫的行为，因为你想要把自己的责任推脱给自己所无力改变的环境，因为你不敢承担责任。“愿上帝赐给我勇气，让我有勇气改变那些我能改变的东西；愿上帝赐给我平和，让我能接受那些我无法改变的东西；愿上帝赐给我智慧，让我分得清上述二者之间的区别！”
      {我能改变的东西是什么：其实就只有我自己！我所不能改变的东西有什么：其实就是我的过去以及我现在所处的环境！(前者需要我们有勇气去改变，后者则需要我们平和的去接受，不要抱怨那些自己所无法改变的东西，而要通过由内而外地改变自己从而影响未来的环境，改变未来的环境！)}


      top-3. * 如何做到积极主动？敢于做出承诺，并确保从不食言！* 如何自主设计自己的人生？自主设计未来目标，付诸行动，许下承诺，决不食言！
      * 如何保证“要事第一”？要记住：“愿意为人所不为，纵使并非心甘情愿，但为了理想和目标，仍可以凭借毅力来克服心理障碍去完成它，不屈服于自己一时的冲动和欲望”！


      top-4. 高效能的一个秘诀：“责任型授权”或者说是“制定双赢协议”！从五个方面来具体实施：1. 预期结果是什么，2. 指导方针是什么，3. 有什么可用资源，4. 制定好考核标准和评估时间，5. 明确奖惩制度！


      top-5. 做人一定要信守承诺，为人正直。信守承诺的意思就是说：一言九鼎，一诺千金。为人正直的意思就是说：从不在人背后说人坏话。

      处人处事一定要学会移情聆听，赞美与肯定他人。移情聆听的意思就是说：要学会站在别人的立场来思考他的问题，要学会先理解别人，再尝试让别人理解自己("知彼解己")。

     

    第一部分 重新探索自我

    • + - 1. 由内而外全面造就和改变自己

      • + 要改变现状，首先要改变自己！要改变自己，则先要改变我们对问题的看法！
        + 承认自己的无知往往是求知的第一步！(所以如果学生不肯发问，不肯暴露自己的无知，不肯让老师知道他真正所到的程度，那么他也绝对学不到东西，也绝不会有任何长进！)

         

        + 思维转换的力量：科学研究的每一项重大突破，几乎都是首先打破传统，打破旧思维、旧模式才得以成功！
        + 注意：思维上的转换所带来的往往才是真正实质性的变化。
        @思考：如果我们只想让生活发生相对较小的变化，那么专注于自己的态度和行为即可，但是实质性的生活变化还是要靠思维的转换！

         

        + 什么是所谓的“由内而外”地改变？

        • - “由内而外”的改变是指：当你发现现实差强人意的时候，你要想改变这种现实，就必须学会先从自身做起！
          - “由内而外”的观点认为：个人领域的成功必须先于公众领域的成功，只有先做到信守对自己的承诺，才能做到信守对他人的承诺。
          - 我相信所谓的“高效能人士的七个习惯”中的很多原则其实早已深入人心，存在于我们的良知和常识中，但是我们要想利用这些优秀原则，就必须首先从自身做起，改变观念转换思维定式，“由内而外”的提升自己，从而让自己到达新境界！

       

      + - 2. 七个习惯概论

      • + “习惯”的定义

        • - 习惯养成的一句金言：“播下一个行动，收获一种习惯；播下一种习惯，收获一种性格；播下一种性格，收获一种人生”！
          @思考：从这句话来看其实改变人生并不难，你只需将“一个动作/一种行为”多次重复(连续重复不少于21天) 就可以了！这样你自然就可以收获到一个习惯，一种性格，一个新的人生！

           

          - 将“养成习惯”这个过程拆开来看，它是由三个部分共同组成的，三者缺一不可！


          (知识，让你知道“做什么，为何做”；技巧，告诉你“怎么做”；意愿，促使你“想要做”！)
          @记住：幸运的是，知识、技巧和意愿 它们三者都位于我们的影响圈之内(可控范围之内)，所以我们完全有能力通过改变自己，从而改善这三者之间的平衡，三者的交集越大，我们的好习惯也就会随着变得越来越多，我们就能实现高效能的生活！甚至是影响环境，改变环境！

           

          - 为什么要“改变习惯”？改变习惯的过程往往很难受，毕竟习以为常的事物比较能给人安全感，但是为了追求一生的幸福和成功(更为长远的利益)，暂时牺牲眼前的安逸和利益，也是值得的！经过一番努力与牺牲所换来的果实，将会更甜美！

         

        + “效能”的定义

        • - “效能”：产出 与 产能 的平衡。
          - 以伊索寓言中的“下金蛋的鹅”为例：产出指的就是金蛋，产能指的则是鹅。
          - 人类所拥有的资产可以分为“三类”：1. 物质资产 2. 金融资产 3. 人力资产。
          它们都是只有在保持产出和产能平衡时，才会帮你实现利益最大化！

     


    第二部分 个人领域的成功：从依赖到独立

    • + - 3. 习惯一：积极主动 (个人愿景的原则)
      (记住：“你是创造者，你能掌控自己的人生”)

      •  

        + 最令人鼓舞的事实，莫过于人类确实能主动努力以提升自己的生命价值！

         

        + 我们每个人都有选择的自由！(在 刺激 和 回应 之间，人有选择的自由)！
        + 而且作为人类(高等智慧生物) 相比于其他物种，我们最大的能力就是：能在刺激和回应之间自由选择！

         

        + “积极主动”的定义

        • - 积极主动：“看一个人的时间和精力集中于哪些事物, 由此我们可以大致判断出这个人是否积极主动”！
          - 积极主动的人专注于自己的“影响圈”(可控范围)，他们专心做自己力所能及的事，他们的能量是积极的，能够使自己影响圈不断扩大，扩张和成长！反之，消极被动的人则喜欢关注于自己的“关注圈”(私人领域)，然后就会发现关注圈里的很多事自己无能为力，结果就导致自己变得越来越怨天尤人，而且还会不断为自己的消极行为寻找借口，逃避自己本应承担起的责任！
          @联系:《财务自由之路》中的第二章“责任到底意味着什么”？(我们要学会主动承担责任，而不要去抱怨别人，因为我们要自己主宰自己的人生!) (当你想要抱怨别人的时候，请反思一下：“难道你想让你的公司，合作伙伴，健康等这些外在的东西来主宰自己的生活吗”，怨天尤人，只能证明自己的软弱无能)

         

        + 觉得自己前进的路途中遇到了阻力，怎么办？

        • - 永远记住：只要先从自己做起(积极主动，由内而外的改变自己)，就可以化阻力为动力！

         

        + 现实中我们所面对的问题通常可以分为三类：
        1. 可直接控制的(问题与自身行为有关,比如培养新习惯,改变自己的行为习惯和看法等等);
        2. 可间接控制的(问题与他人行为有关,比如以身作则,影响他人);
        3. 无法控制的(无能为力的问题,比如我们的过去以及我们现在所处的环境)。
        @思考：当我们面对自己无法控制的问题时该怎么做？其实我们要做的就只是简单地“改变自己的面部曲线，以微笑,真诚与平和的心态来接受现实”！

         

        + 如何辨别自己是在关注“影响圈”(可控范围) 还是在关注“关注圈”(私人领域)？

        • * 通过自己言谈中的语气，就可以简单做出判别！
          - 与“关注圈”(私人领域)相关的语句 多半 带有假设性质，多半是指向过去或者是自己所处的客观环境，因为这里面的很多问题我们改变不了！(比如：要是我...就好了, 如果..., 我本可以...)
          - 与“影响圈”(可控范围)相关的语句 多半 则体现了个人的能力，多半是指向未来以及那些通过努力可以改变的事！(比如：我可以更耐心/更明智/更体贴/更努力,这样我就能改变自己所处的困境...)

         

        + “抱怨环境”是懦夫的行为，因为你想要通过这个你改变不了的东西来逃避自己本应承担起的责任！

        • - 正确的做法是(积极主动的做法)：“由内而外”地做出改变，先改变那些自己能改变的东西(也即:改变自己)，再经由此对那些自己所不能改变的东西施加影响，从而达到“改变”环境的作用！
          * (也可理解为“先易后难”地做出改变，这样你才有可能达到改变环境的终极大目的)
          * (先主动改变自己的个人行为，让自己变得更充实，更具创造力，然后再经由此自然而然地对环境施加影响，最终方可达到改变环境之大目的！)
          @总结：不管快乐还是不快乐，同样积极主动！对力所不能及之事泰然处之，对能够掌握之事则全力以赴！

           

          + “愿上帝赐给我勇气，让我去改变那些我可以改变的东西”！
          (我们所能够改变的东西，其实就只有我们自己，积极主动的“由内而外”的改变自己)
          (@思考: 具体该怎样做才能有所改变？关注自己的“可控范围”，对自己或他人做出承诺，并绝不食言！这样改变自然而然地就在自己身上发生了！具体做法就是：从小事做起，对自己许下承诺并予以兑现。)
          (@思考: 如何掌控自己的人生？1. 做出承诺，并保证决不食言；2. 确立目标，并付诸实践。一次次地做出承诺，一次次地信守承诺，终有一天荣誉会超越情绪的力量！)

          + “愿上帝赐给我平和，让我能接受那些我无法改变的东西”！
          (我们所无法改变的东西，其实就是指我们的过去以及我们此时此刻所处的环境，要学会不去抱怨环境，要平和地接受它)

          + “愿上帝赐给我智慧，让我分得清上述二者之间的区别”！
          (感谢上帝，我感觉自己似乎已经渐渐明白了这二者之间的区别)
          (努力吧，年轻人，人生路还长，起步输了不要紧，在接下来的人生路上望你能赐给自己“勇气”和“平和”，努力奋斗！创造辉煌！)

         

        + 对于已经无法挽回的错误，积极主动的人不是悔恨不已，而是先主动承认自己往日的错误，并立马从中吸取教训，免得殃及未来！
        @切记：实际上伤我们最深的，不是错误本身，而是我们对错误的回应！我们一定要学会立刻承认错误并加以改正，否则它会继续殃及后来，对我们造成更大的伤害！

         

        - 影响圈(可控范围)的核心是：做出承诺与信守承诺的能力！做出承诺和信守诺言是培养高效能习惯的最根本力量！
        ( “积极主动”的 本质 和 最具体的体现 就是：对自己或别人有所承诺，然后从不食言！)
        ( “积极主动”的 具体表现就是：我们如何做出和信守承诺！)

         

        + 时时刻刻提醒自己：刺激和回应之间存在着自由选择的空间，我们要向自己承诺，要抓住选择回应的自由！

         

        @总结：当自己处于逆境(不顺)时，如果你一直认为问题“存在于外部”，那么请马上打住，因为这种想法本身就是问题！这种做法本身就是懦夫的表现！

       

      + - 4. 习惯二：以终为始 (自我领导的原则)
      (记住：任何事物都是经过两次创造之后的产物，包括你的人生)
      (记住：本章主要教你要学会并养成 “第一次创造/智力创造” 的习惯)

      •  

        + 人类所独有的天赋：自我意识，独立意识，想象力 和 良知！
        (自我意识 让我们能够积极处世，想象力和良知 则使我们能在生活中发扬积极精神并施行自我领导)。

         

        + “以终为始”的定义是指：做任何事之前，请先确定目标 并 认清方向！

        • · 先确定目标，那么你的洞察力就会大大改善，这样你不但可以对自己目前的处境了如指掌，而且不至于在追求目标的过程中误人歧途，白费功夫！

         

        + “以终为始”的重要原则基础：任何事都是经过两次创造而成的！

        • · 先在头脑中构思，智力上的第一次创造，然后是付诸实践，即体力上的第二次创造。
          · 这句话的意义在于：“让我们明白，要把制定目标看得和实践本身同样重要，这样你的影响圈才会日益扩大”！

         

        + “改写人生剧本：做自己人生的第一创造者”

        • · 每个人的人生也都是经过二次创造之后的产物，其中第一次创造，或者是自己主动设计的，或者是由外部环境、他人安排或旧有习惯所制定！
          · 如果你自己意识薄弱，不愿主动设计自己的生活，结果就会让出于你影响圈之外的人或事控制你自己。结果你的人生轨迹就会屈从于各种压力，受到各种外界条件的制约！然后你如有不满，就会抱怨外部环境，从而陷入一个恶性循环！

         

        + 拥有了使命感，你就同时拥有了积极主动的要素！

       

      + - 5. 习惯三：要事第一 (自我管理的原则)
      (本章是前面两个习惯的实施、执行 和 自然表现。它要求我们要运用独立意识 去实践/去行动  真正的创造出我们自己的人生)
      (记住：本章主要是在强调你要养成“第二次创造/行动创造”的习惯，本习惯的关键在于：“一定要身体力行”)

      •  

        请注意：一旦确定了人生方向和目标，你就应该对自己进行有效的管理，让自己的生活与设想的一致！
        (这个过程主要涉及左脑的功能：分解任务,分析问题,排序,具体运用以及在规定时间内完成任务)

         

        + 思考什么是“要事”？

        • Q1: 在你目前的生活中，有哪些事情能使你的个人生活彻底改观，但是你却一直没有去做？
          Q2: 在你目前的生活中，有哪些事情能使你的工作局面彻底改观，但是你却一直没有去做？
          @思考：这个还真有：在我目前的生活中，“把论文写好，然后成功地发出一篇A类论文” 这件事，可以使我 无论是“个人生活”(出国，奖学金，赚钱，人生自由 都可得到实现) 还是“工作局面”(优秀的工作成绩，简历，拿OFFER的资本) 都能达到立即彻底改观！

          @思考：这件事，我已经读研一年半了，却一直在逃避，一直没有主动去做！

         

        + 怎么对待“要事”？

        • - 所有成功人士身上普遍存在的共性其实就是：他们坚持 “要事第一”！
          - “成功者能为失败者所不能为，他们纵使并非心甘情愿，但为了理想和目标，仍可以凭毅力克服心理障碍”！
          - “他们愿意为自己所不愿为之事，他们能做到不屈服于自己一时的冲动和欲望”！

         

        + 时间管理, 事务管理 的经典方法——第四代时间管理手法：“时间管理矩阵”！
        ( 也即要区分 [ 紧急,非紧急 | 重要,不重要 ] )

        •  


          * 偏重第一类事务的人，往往感觉“压力大,精疲力尽,被危机牵着鼻子走,常常忙于收拾残局”；
          * 偏重第二类事务的人，通常能做到“规划自己的人生，设计自己的人生，有愿景有远见，懂得自律与自制，生活中极少发生危机情况”！因为这种人的思维方式总是“预防式”的，他们总是能够做到防患于未然！
          (只要我们着手实际行动做第二类事务，我们的生产效能便会立马增进)
          * 偏重第三类事务的人，往往容易“轻视目标和计划,容易急功近利,也容易导致人际关系肤浅甚至破裂”；
          * 偏重第四类事务的人，容易“玩物丧志，对自己完全不负责任，严重的可能连基本生活都无法保证，需要依赖他人”！

           

          + 大多数有“产出”的活动都是属于第一类事务的范畴，但大多数“产能”活动都属于第二类事务的范畴！只能两者之间达到平衡，才能到达所谓的“高效能”！
          + “管理好第一类事物的唯一途径就是：要重视第二类事物，也就要学会未雨绸缪和抓住机遇，同时要有勇气对处于第三类第四类上的事务活动说不”！
          + "以第二类事务为中心的方法来减少或减轻第一类事务的危机，结果就是要你把大部分时间用于授权和训练别人，采取重视“产能”的长效手段，你就有可能不会再被急事缠身了"！

         

        + 怎样才能做到 “要事第一”？

        • - 记住要：勇敢说不！
          - 要集中精力于当急的要务，就得排除次要事务的牵绊！此时就需要有勇气对这些次要事务说不！在紧急与重要之间，应该知道取舍！
          - 我们有时无法做到“要事第一”的主要原因：其实不在我们不知道重点或是我们没有足够的自制力，而主要是因为我们没能以原则作为生活重心，我们缺乏个人使命宣言，没有目标感，不知究竟所为何来！

         

        + 第四代个人管理方法-“如何做到要事第一,平衡效能”

        • 四项关键步骤：
          1. 确认角色：写出自己的关键角色；
          2. 选择目标：思考每一个角色下你最想做的一两件事，作为你该项的“要事”；
          3. 安排进度：为每一项目标事务安排具体的实施时间，把每项任务分配到一个星期中的某个具体日子里去做；
          4. 每日调整：根据每天的突发事件，人际关系的意外发展，或者新出现的崭新机会对自己每日的要务安排进行适当调整！

         

        + 高效能的秘诀——“责任型授权”

        • - 授权是一种高效提高效能的秘诀！授权是事必躬亲与管理之间的最大分野！
          - 指令型授权：总想手把手教学，他们关注的重点在于：做事的方法！他们自己为最后的结果负责！这种授权确实“劳心劳力”，不如自己干！但是还有一种更好的授权方法！
          - 责任型授权：(他充分认可别人的自我意识、想象力、良知和独立意识)，责任型授权的关注重点在于：最终的结果！从而提高自己的效能的同时，也锻炼了别人的效能。
          @思考：怎样才能将“责任型授权”真正的实施好？双方一定要注意事先确定好这五个方面：1. 预期成果是什么(明确并理解最终想要的结果，并制定具体的日程安排和明确的截止期限)；2. 成果的评估标准是什么(做出一些对结果的限制性规定，避免由于过度放任而扼杀了双方的主观能动性)；3. 可用资源有哪些(为了达到成果，我们可以使用哪些人力物力财力或者是组织技术支持)；4. 划分好责任归属(制定好业绩标准，确定好何时交出业绩，何时进行评估)；5. 明确奖惩(对评估的结果给予及时且重要的反馈，其具体内容包括：财务奖励(比如我的"海鸥行动"就拿到了第一笔财务奖励500元), 精神奖励, 职务调整, 改变其组织任命等)。

         

        + 切记：事有轻重缓急，一定要分清什么事是重要的事(要事)，然后切记“要事第一”。

      + - 7. 习惯四：双赢思维 (人际领导的原则)

        + 要建立一种能突出合作价值的信息和奖励机制，鼓励大家合作！

         

        + 双赢局面的构建

          2. 双赢关系：双赢的精髓在于信用，也即情感账户；
          3. 双赢协议：让双方由从属关系转为合作关系，上级监督转为自我监督；
          @联系：制定“双赢协议”类似“责任型授权”，请明确规定：预期结果(目标是什么)、指导方针(方法是什么)、可用资源(什么可以用)、任务考核标准(评估标准和完成时间)、奖惩制度(明确奖惩)。
          @思考：这种方法之所以有效就是因为：它让大家都看到了机会，也都能清楚的意识到自己只要能尽快的达到考核标准就能提高薪水！所以大家就会干劲十足！“双赢协议”和“责任型授权”也一样，它们都注重的是结果！
          4. 双赢体系：双赢赋予其中的每个个人以明确的任务，说明预期结果，指导原则和可用资源，个人要对自己所导致的结果负责，并完成自我评估。
          5. 双赢过程：从对方的角度看问题，认清主要问题和顾虑，确定大家都能接受的结果，并寻找实现这种结果的各种可能途径，这就是所谓的“双赢过程”！

          2. 下一次当你同别人的意见相左的时候，尽量要学会先理解他的立场和顾虑，以创造性、互惠互利的双赢模式来应对问题！
