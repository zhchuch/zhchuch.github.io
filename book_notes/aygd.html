        女性爱慕男性的“力”，男性爱慕女性的“美”。男性重行动，女性重感情，男性长于抽象观念，女性长于感性直觉。男性用刚强有力的线条勾画出人生的轮廓，而女性则为之抹上美丽柔和的色彩。

        - 男女之间是否可能有真正的友谊？
        异性之间的友谊不可能排除性的吸引，但它仍然可以是一种真正的友谊，性的神秘力量因客观情绪或主动努力而被限制在一个有益无害的地位，这既可以为异性友谊罩上一种同性友谊所未有的温馨情趣，有不致于像爱情那样激起疯狂的占有欲。

        爱与孤独永远都是人生中所必须面临的一个二律背反：结婚吧，失去自由，不结婚吧，又会孤独。一切爱都是基于生命的欲望，而有欲望不免就会造成痛苦。

        所以幸福本身就是一种悖论，而这个悖论的解决正存在于争取幸福法过程之中，其中可能有斗争，有苦恼，但只要希望尚存，就有幸福。
        “若将幸福分解成基本原子时，就可见它也是由斗争与苦恼形成的，惟此斗争与苦恼永远被希望所挽救而已。”

    • + 时光村落里的往事

      • 人分为两种：一种人有往事，另一种人则没有往事。
        有往事的人爱生命，对时光流逝无比痛惜，因而时常怀着一种特别的爱意，把自己所经历的一切珍藏在心灵的谷仓里。由于这爱，一个人才会真正用心在看，在听，在生活，因为此刻我所看的、听到、经历到的一切，无不转瞬即逝，成为往事！只有珍惜往事的人才是真正的在生活。

        浩渺宇宙间，任何一个生灵的降生都是偶然的，但离去却是必然的！
        忽然有一天，你心头一惊，想起时光在飞快的流逝，正无可挽回的把你、你所爱的人以及你们共同拥有的一切带走，于是，你的心中会升起一股柔情，想要保护你的爱人免遭时光劫掠！

        - 灵魂到底是什么？
        所谓灵魂就是你的那些活着的往事，这是灵魂之所以具有孕育力和创造力的秘密所在。
        - 为什么现在生活在城市里的人会感到时间过的真快？
        因为这些人太不在意生活了，在这个商业社会里，人们活的越来越忙，哪里有功夫去注意草木发芽、树叶飘落这些小事！哪里有闲心用眼睛看，用耳朵听，用心灵去感受生活！生活被简化为尽快地赚钱和花钱，天天争分夺秒，岁岁年华虚度，到头来怎么会不觉得一辈子真短？没有值得回忆的往事，一眼就望到了头的生活从来都是如此短暂。

    • + 永远未完成

      • 我们需要用一种永远未完成的心态来看待自己过去的事情，这种心态会驱使我们不断超越自己，从而取得那些自满之辈所不可企及的成就。

        - 对于人生选择的思考
        林中路分为两股，走上其中一条，把另一条留给下次，可是再也没有下次了。因为走上的这一条路又会分股，如此以至于无穷，不复再有可能回头来走那条未定的路！这就是人生境况的真实写照。但也正因为人生中所面临的种种可能性，我们才会感觉到自己是命运的主人，从而踌躇满志地走自己正在走着的人生之路。

    • + 生命本来没有名字

      • “生命本来没有名字”，这句话说得多么好！我们降生在世上，有谁是带着名字来的？但是随着我们长大，我们都越来越深的沉溺于俗务琐事。我们渐渐的被名字、头衔、身份、财产之类的所缠绕，而生命本身却被隐匿了起来。

        生命的诞生本身就是一个奇迹，你，我，甚至于每一个生命来都这个世界上都是那么的偶然，完全可能不降生，但毕竟降生了，然后又将必然的离去。所以，对于生命的珍惜和体悟乃是一切人间之爱的至深的源泉！

        你爱一个人，如果不是把她当做一个独一无二的生命来爱，而是爱她的美丽，温柔，聪明贤惠，那么你对她的爱还是比较有限，因为这些品质在别的女人身上也能找到。惟独她的生命，是她唯一与普天下女人不同的地方。

    • + 人人都是孤儿

      • 我们为什么会渴望爱？我们心中为什么会有爱？答案很简单，因为我们人人都是孤儿。父母不可能成为保护我们免受人世间种种苦难的可靠屏障，终有一天，我们会明白，凡将于我们身上的苦难，不论是疾病、精神的悲伤、还是社会性的挫折，我们都必须自己去承受，再爱我们的父母也对此无能为力。最后，当死神召唤我们的时候，世上绝没有一个父母的怀抱可以使我们免于一死！

        因此，从茫茫宇宙的角度来看，我们每一个人的确都是无依无靠的孤儿！也正因为这种根本性的孤独境遇，才有了爱的价值，爱的理由。人人都是孤儿，所以人人都渴望有人爱，都想要有人疼！

    • + 爱的反义词

      • 爱的反义词不是孤独，也不是恨，而是冷漠。
        孤独是爱没有着落，而恨则是自己的爱受到了挫折，所以孤独者和恨者都是会爱的！唯独冷漠者却与爱完全无缘。

    • + 沟通的限度

      • 人与人之间存在着隔膜本身就是一件极其正常的事情，在什么情况下，我们才会因为发现了隔膜的存在而倍感痛苦呢？只有当在我们内心深处非常在乎这个人的时候，或者非常在乎彼此之间的关系的时候。

        人的心灵是有质和量的不同的，质不同(比如说基本的人生态度和价值取向格格不入)，就是所谓的“道不同不同与谋”，沟通根本无从谈起；质同了之后，还会存在量的不同，其灵魂的内涵有深有浅，从而都会造成一些局部的沟通障碍。

        既然隔膜是必然存在的，那么任何一种交往要继续下去，就必须能够相互包容两人相互之间的隔膜。

    • + 爱使人富有

      • 只要你曾经用真心去爱过，爱的收获就必定会以某种方式保藏在你的心中，即使最后这段爱是不幸的，岁月也会渐渐抚平你的创伤，而这之后你就会发现最主要的珍宝并未丢失。

        爱是奉献同时也必是收获，通过爱我们才不断的建立了和外部世界的联系，深深地爱一个人，你借此所建立的不只是与这个人的联系，而且也是与整个人生的联系！爱的经历决定了人生内涵的广度和深度，一个人的爱的经历越是深刻和丰富，他就越是深入和充分的活出了自己的人生！

    • 也许孤独是爱最意味深长的赠品。

       

      + 论孤独

      • 孤独与沟通中都有大快乐，这两者都属于灵魂！
        和别人混在一起的时候，我向往孤独，而在孤独时，我又向往看到我的同类。

        - 区分“孤独”与“孤僻”
        心灵上的孤独与性格的孤僻是两回事。孤僻属于弱者，而孤独属于强者，虽然两者都不合群，但前者是因为惧怕受到伤害，而后者是因为精神上的超群卓越。

      + 爱与孤独

      • - 孤独是人的宿命
        因为它是基于这样一个事实：我们每个人都是这个世界上一个旋生旋灭的偶然存在，从无中来，又要回到无中去，没有任何人任何事情能够改变我们的这个命运。甚至连爱情也对此无能为力。

        我爱她，他成了我的一切，除了她之外的整个世界好像都不存在了。那么，一旦我失去了她，是否就代表着失去了一切呢？不，恰恰相反，整个世界又在我的面前展现出来了，我重新得到了一切！

      + 沟通的必要

      • 心境不能沟通，这是人类生存的基本境遇之一，所以每个人在某个时刻都会觉得自己是被弃的孤儿，每个人在经历痛苦的时候都会觉得只有自己是这个世上最不幸的人。

        - 为什么说沟通是必要的
        如果没有了沟通，那么你所获得一切的思想或是经历体验就都会变得没有意义了。因为你不可能把这些东西告诉别人，所以它对别人当然毫无意义，而对于自己，这种思想或经历会随你而毁灭，永不会流传出去，所以也就变得没有意义了。

      + 交往的限度

      • 交往是有限度的，两个人之间的外在距离稍稍大于他们的内在距离，能够使得他们之间情感上的吸引力达到最佳效果。
        在一次长途旅行中，最好的情况是有一位称心的旅伴，其次是没有旅伴，而最坏的情况则是有一个不称心的旅伴。

      + 爱情的容量

      • 爱情不论短暂还是长久，都是美好的。甚至陌生异性之间毫无结果的好感，那定睛的一瞥，那莫名的激动与惆怅，都是美好的。因为，能够感受这一切的那颗心毕竟是年轻的。

      + 我爱故我在

      • 爱情与事业，人生的两大追求，其实质相同，都是自我确认的方式。爱情是通过某一异性的承认来确认自身的价值，事业则是通过社会的承认来确认自身的价值。

      + 爱的距离

      • 亲密无间这个词其实是不适合爱情甚至是任何其他人与人交往中的感情的(包括友谊)，这些情感要的是亲密但不要无间！
        人与人之间不要有一定的距离，这种分寸感是成熟的爱的标志，这个距离意味着对于对方作为独立人格的尊重，包括尊重对方独处的权利。

     

    + - 5. 论爱情

    • + 爱情哲学

      • 看两个人是否相爱，一个可靠的尺度是看他们是否互相玩味和欣赏。两个相爱者之间必定是常常互相玩味的，而且是不由自主的要玩，越玩越有味。如果有一天觉得索然无味，毫无兴趣，那么爱就已然荡然无存了。

      + 爱情伦理学

      • 一个人对待异性的态度最能表明他的精神品级，能够表明他在从兽向人上升的阶梯上处在什么样的高度。凡正常人，都兼有爱人和被爱的两种需要，如果这两种需要不能同时在对方身上获得满足，那么便潜伏着危机。因为在这茫茫宇宙中，有谁不是一个想要被人疼爱的孤儿呢！

      + 爱情心理学

      • 爱情的发生需要适宜的情境。彼此太熟悉，太了解，则会失去神秘感；彼此过于陌生和隔膜，也不易发生爱情。爱情的发生，需要那种若即若离之状态，既有神秘感，又有亲切感。

      + 爱情美学

      • 一个漂亮女人能够引起我的赞赏，但不能使我迷恋，使我迷恋的是那种有灵性的美，那种与一切美的事物发生内在感应的美！

        不纯净的美使人迷乱，纯净的没使人安静。美是无法占有的。

      + 女性价值

      • 男人是孤独的，在孤独中创造文化；女人是合群的，在合群中传播文化。

      + 女性魅力

      • 自然、质朴、不雕琢、不做作，美丽的女孩子的天性往往都能得到比较健康的发展。也许这是因为她们从小就招人喜欢，饱吸爱的养料，而她们的错误又容易得到原谅，因而行动较少顾虑，能够自由的生长。
        犹如一株植物，她们得到了更加充足的阳光和更加开阔的空间，所以不致发生病态。

      + 女性心理

      • 按照《旧约》的传说，女人偷食禁果的第一个收获是知善恶，于是用无花果叶遮住了下体，而生育则是对她偷食禁果的惩罚！
        @联系《自私的基因》：这就是来自大自然的无情约束。不过在漫长的进化过程中，女性也发展出来了应对它的一套成熟的策略。

      + 男人与色情

      • 女人用肉体思考，男人用女人的肉体思考。只用色情的眼光看女人，近于无耻，但身为男人，看女人的眼光中不可能完全不含色情。

      + 两性比较

      • 女人比男人更相信梦，在女人的生活中，梦占据着不亚于现实的地位。男人不信梦，但也未必相信现实，当男人感叹人生如梦时，他通常是把现实和梦一起否定了。

        男人通过征服世界而征服女人，女人则通过征服男人而征服世界。男人是突然老的，女人是逐渐老的。
        男人最令人厌恶的缺点是：懦弱和吝啬，女人最令人厌恶的特点则是：粗鲁和庸俗。

      + 两性之间

      • 吸引异性的两种方式：1.显示风趣、智慧和活力，勾起对方的好奇心；2.显示忧愁、伤痛和深度，勾起对方的同情心。活力和深度的统一，吸引力达到极致。

     

    + - 6. 婚姻反思录

    • + 论怕老婆

      • 只有当老婆习惯于在自己错时偏要丈夫认错这种情况下，才会发生怕老婆的问题。而男人怕老婆也可分为三类：第一类是男人出于对妻子情感上的崇拜，认为妻子一定对，即使妻子有错责任也在自己；第二类是男人出于自己理智的宽容，认为向自己老婆认个错反正又不丢什么面子；第三类则是出于自己性格的怯懦，认为这是家丑，他害怕丢面子而向老婆认错但心里是不爽的。

      + 家

      • 家是温暖的港湾：经历过远洋航行的人都会知道，一旦海平线上出现港口那朦胧的影子时，自己寂寞已久的心会跳的多么欢快。如果没有一片这样的港湾在等待着拥抱我们，那无边无际的大海岂不太令我们失望，甚至绝望！
        在人生的航行中，我们也需要这样一个温暖的港湾，等待着拥抱我们，这就是家。

      + 调侃婚姻

      • 性别是大自然的一个最奇妙的发明，而婚姻则是人类的一个最笨拙的发明。

      + 宽松的婚姻

      • 爱侣之间应该有基本的诚实和相当的透明度，但是万事都应有个度，水至清则无鱼，苛求绝对诚实反而会酿成不信任的氛围，甚至会逼出欺骗和伪善！一种健全的爱侣关系的前提是互相尊重，包括尊重对方的隐私权，这种尊重一方面是基于爱和信任，另一方面则是基于对人性弱点的宽容。羞于追问相爱者难以启齿的小隐私，乃是爱情中的自尊和教养。

      + 心疼这个家

      • 家庭是把人与大地、与生命的源头联结起来的主要纽带。有一个好伴侣，筑一个好窝，生儿育女，恤老抚幼，会给人一种踏实的生命感觉。无家的人倒是一身轻，但只怕你无法承受这份轻(生命之无法承受之轻)，因为它很容易让你陷入一种在这世上没有根基的虚无缥缈的感觉。

        家庭被比喻成社会的细胞，是很有道理的！因为家庭本身也具有生命的特征，家也是一个活的有生命的东西，我们也需要把它作为一个有生命的东西那样去疼爱它珍惜它，否则很可能即使两人之间有爱情也无法结出一个美好家庭之果。

        两个人两条生命因相爱而结合成为一个家，在随后共同生活的过程中，他们的生命随岁月的流逝而流逝，这些流逝的岁月大部分都流入了这个家，转化为这个家的生命。从这个角度来说：家是两个人已流逝的岁月的装载体。所以可见为什么有那么多人反对拆迁，因为拆迁不仅是拆了这些有形的东西，更多的是拆了所有与之相关的岁月记忆。

      + 婚姻反思录

      • 爱情是人生的珠宝，当我们用婚姻这只船来运载爱情的珍宝时，我们的使命就是尽量绕开暗礁，避开风浪，安全的到达目的地！
        谁若故意迎着风浪上，固然可以获得冒险的乐趣，但同时也说明了他/她对船中的珍宝的不爱惜，好姻缘是要靠珍惜之心来保护的，珍惜便是缘，缘在珍惜中，珍惜之心亡则缘尽！

      + 嫉妒的权利

      • 当爱情存在时，嫉妒就不妨充当为爱情站岗放哨，履行职责的卫士。此时它应该心明眼亮，不可向假想敌人开枪，错杀无辜。一旦爱情不复存在，那么它就应当尊严的撤离岗哨，完全不必也不该为不存在的东西拼个你死我活。

      + 婚姻的悖论与现代的困境

      • 婚姻是以亲情式的爱情为基础的。

      + 婚姻中的爱情

      • 爱情应该具有一个更宽泛的含义，它不仅包括有浪漫之情(浪漫式的爱情)，而且也可以包含基于这种浪漫之情而生长出来的亲情(亲情式的爱情)。
        婚姻前和婚姻初期时的爱情就属于浪漫式的爱情，经历了婚姻之后，这种初级的爱情形式就得以生长，成为了更成熟的亲情式爱情。

      + 夫妻间的隐私

      • 在充满信任氛围的好的婚姻里，正因为夫妻间最尊重对方的隐私权，事实上的隐私往往最少。

     

    + - 7. 婚姻与孩子

    • + 婚姻与爱情
    • + 婚姻不是天堂

      • 好的婚姻是人间，坏的婚姻是地狱，别想到婚姻中寻找天堂。婚姻的稳固与其说取决于爱情，不如说取决于日常生活中小事的和谐！
        如果认为单凭激情就能对付年复一年充满琐碎的日常共同生活，那么你就太天真了！爱情仅仅是感情的事，婚姻却是感情、理智与意志三方面通力合作的结果！

        结婚是一个信号，代表着两个如胶似漆仿佛融为一体的人之间的热恋已达到极限，需要降降温了，要适当拉开距离，重新成为两个独立的人，携起手来走新的人生之路。另外牢固的婚姻，不仅需要相互信任，也需要相互宽容。

    • + 论亲子之爱

      • 付出比获得更能激发爱，爱是一份伴随着付出的关切，我们确实最爱我们倾注了最多心血的对象！
        @ 联想：《林肯传》中林肯所说过的一句名言：“如果你以太高的价格买了一件东西，那么它就会比其他东西更能牵动你的心。”

        一般认为做父母都是如何为子女受苦，奉献和牺牲，似乎恩重如山，但其实这受苦本身也是享乐，这奉献同时也在收获！父母对子女有付出，同时子女对父母也同样有回报。

    • + 孩子

      • 孩子是使得一个家成为家的依据！
        由于你的到来，我这个不信神的人也对神充满了敬意。婴儿是超凡脱俗的，因为他们刚从天国来，再庸俗的父母，生下的孩子都绝不庸俗。

        孩子的世界是尘世上所剩不多的净土之一，凡是走进这个世界的人，或多或少都会受孩子的熏陶，变得可爱很多。通常我们带孩子出去玩，同时也算孩子带我们出去玩，去逛公园，去跟踪叶草上的甲虫，泥土上的蚂蚁等等。孩子更新了我们对世界的感觉。

    总结：

    • 这本书讲述了作者对很多与爱情亲情有关的事物的理解，为什么需要爱，什么是爱，爱与孤独的关系，爱情与婚姻的关系，婚姻与孩子之间的关系等等诸如此类的问题，作者都用自己的人生体会阐述了对它们的理解！里面不乏有很多经典的人生总结。
      我相信这也是我读书的一大意义之所在：读一本书就相当于是在和一个人进行一场隔着时空的交流，它可以把别人的生活经验和智慧传授于你，你不需要再去经历这些相似的痛苦经历就可以获知别人的经验。这是一件多么值得尝试的事情。
