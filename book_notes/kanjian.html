看见

作者：柴静

  • 柴静十年历程的自传性作品。书中记录下了十年间的一些人和事，是他们的生活，也是你我的生活。

     

    全书总结：

    • □ 本书给我留下深刻印象的前五大观点是:

      top-1. [对职场的再思考]  柴静这个女孩子都可以在职场中从没有落脚点的不适应期(CH1)开始奋斗，到她跑新闻调查不同通过自己的努力和奋斗找到自己职场的“立足点”(CH4)，再到最后她在职场中搏出了自己的一片天地，开辟了自己的专栏节目(CH9)，她花了六年时间去奋斗！最终自己的事业，自己的人生都走向了成功与成熟(CH20)！
      · 所以我以后再入职场也一定要以此为鉴，她柴静一个23岁小女孩都能做到的事情，我一个25岁硕士大男人怎么能做不到呢？我也一定要花时间去奋斗，最终打拼出属于自己职场中的一片天地！再也不能像第一次实习初入职场那样被现实所吓倒，不能再退回象牙塔了！


      top-2. [对人生“苦难”的再认识] 这个世上比你遭受更大不公和苦难的人多了去了(CH2, CH5, CH9, CH11, CH12, CH14)！每个人的人生都会承受苦难(CH17)，人生苦难重重，家家有本难念的经！为什么如此？因为我们都对美好有期待，因为我们都想要获得“作为人的尊严”(CH8)。
      · 所以千万不要像药家鑫那样，总觉得这个世界上就自己活的最不好，其他人都获得很幸福。其实现实中，哪个人不是在为自己更好的期待而努力而痛苦呢？(CH18)


      top-3. [对处世哲学的思考]  我们要学会“宽容”(CH13, CH16, CH19)，学会“理解”！知道对自己主观很难尝试去完成的事情，如何提高自己的自控力("念起即觉，觉即不随")(CH10)。

      为什么要有自控力，因为自控力是我们作为一个有价值的人能真正体现自己“成长力”和“价值”的地方！


      top-4. 思考如何去表达爱，三大关键词：“赞美”(CH19)，“行动”(CH6)，“关注(倾听)”(CH3, CH18)。
      top-5. 通过CH16了解到了“农村征地问题”的缘由，历史由来！看到了在这个问题上政府，开发商，拆迁户三者之间复杂的三角关系！

    + - 1. 别当了主持人就不是人了 ( 职场初体验 00-02年)
    - 初入职场的前两年的那种惶恐不知所措而又迷茫的经历

    •  

      - 初入职场，每个人都会不适应会迷茫！甚至是不知所措！不断的被各种人批评，甚至对自己的能力都产生怀疑。
      - 但请记住：“别人批评你不可怕，对你失望才是真正可怕的东西”！

       

      - “你问一个问题的时候，期待答案吗？如果你不期待，那么你就别问了。”
      - 白岩松：“人们声称的最美好的岁月其实都是最痛苦的，只是事后回忆起来的时候才那么幸福”！
      @思考：每个职场老人都是从新人走过来的，他们现在之所以能混的风生水起就是因为他们经历过了挫折，并成功挺过了最初这段需要站稳脚跟的艰难岁月！比如柴静初到央视时的拼命到现在站稳脚跟成为知名媒体人并出了畅销书，可谓已然混得很成功了！

     
     

    + - 2. 那个温热的跳动就是活着 ( 记录03年“非典阻击战”  )

    • - 2003年全国进入“抗击非典”，整整一个春季，这种被称为“非典型肺炎”的病毒搅乱了一个中国，并波及了小半个世界。

      - 书中的句子：
      “把片子拍完之后，带子放在门口传达室，会有人来取，把带子消毒之后再交给编辑”
      “我妹来酒店给我送东西，我让她带只小音箱给我。晚上在空无一人的大街上，隔着三四米远，我让她站住：放下，走吧！”
      @联想：记录 03年 的非典，那个时候真的是全国都被搞得人心惶惶！全国各个地方都在背着药筒喷洒着消毒水，当时我还关塘小学上小学呢，现在都还能想起“那个时候我们放假了，然后教室被打上了消毒液”的场景。
      那个时候 芫荽 成为了紧俏商品，据说是用它煮出来的水可以起到杀菌消毒的作用！
      @感受：当时的我们这些小孩子根本不知道害怕，所谓无知者无畏，初生牛犊不怕虎，只知道能放假很开心，只有通过爸妈等这些大人们的那些紧张神情中好像才能理解事情的严重性，不过还是不在意。
      @回味：现在回想起来：还是觉得小时候的那种因放假而能得到的纯粹的快乐是多么的珍贵啊！长大了之后就再也无法体会到了！

       

      - “娱乐脱口秀”节目是如何流行起来的？

      • · 2001年911事件之后，美国人开始做起了娱乐脱口秀节目。其实政府就是想要通过这种娱乐节目让民众笑起来，因为人们只有在还能笑的时候，才是最不容被打败的！

        @思考：人在真正面对死亡或者是极大痛苦的时候是感觉不到对死亡的恐惧 或是极大的痛觉的！反倒是事后平静下来，人们才会发现原来死亡离自己曾经那么近过，那时的人们才会觉得后怕！
        我自己的“右臂双骨骨折经历”也验证了这个说法，骨折之时人是没有极大痛觉的，只是觉得这个地方的感觉不大对，手完全不听使唤了！在之后被送到医院接骨的过程中，那个疼痛感简直难以想象，找个地缝都想钻进去。
        所以这段经历也提醒我在以后如果遇到别人在忍受痛苦，一定要学会理解他们的选择并尽力帮助他们！要学会为他们带去笑，因为只有会笑的人才最不容易被打倒！

       

      - 揭露非典在初期没有良好隔离条件下的治疗情况，这种情况的出现也是出于对现实条件的妥协！
      @思考：想想这些初期患上SARS的人，你就知道现实往往就是残酷的，你有钱也没用！所以当我再面临来自社会的不公时，也不要去过度抱怨，因为：比你遭受到更大不公的人有的是，你既然无力改变这些，就不要太过于纠结，只能选择妥协！

     
     

    + - 3. 双城的创伤 ( 双城“小学生连续服毒自杀事件”记录-03年 )
    - 对人的认识有多深，呈现才能有多深。双城连续服毒事件调查到最好，我们发现，最大的谜，其实是孩子的内心世界。

    •  

      - 青少年的成长伴随着青春期的发展，其内心世界也变得开始复杂！
      他们懂得了什么是烦恼，但是他们却不懂得如何去倾诉烦恼并获得帮助。
      当烦恼压迫得他们喘不过来气的时候，他们居然会极端到选择死亡，以追寻所谓的“没有烦恼的世界”！

      - 这一系列的服毒自杀事件，还暴露出社会存在的另一大问题：
      整个社会，从警察到老师甚至到父母，这些成年人都不懂得如何去给这些孩子们提供帮助！反而却在不懂孩子内心的情况下在火上浇油！

      @思考：我现在已经走过青春期了，但在我人生中遇到有别人在青春期内出现心情压抑郁闷的情形，我一定要学会去帮助他们、开导他们！
      至于怎么开导，其实这些孩子已经给出了答案：“请那些大人们多多地学会倾听吧”，学会通过倾听与友好沟通的方式去打开孩子们的内心世界！
      @联系:《少有人走的路》中所说的：“爱是什么？爱最重要的体现方式就是：关注！而关注的最主要形式则是 倾听！”

     
     

    + - 4. 是对峙不是对抗 (讲述柴静在跑新闻时的经历，自己的职场打拼史)
    - 能不能采访准确，这 事关人的性命，我要是问得不准确，就不配坐在这椅子上。

    •  

      - 当记者也不是那么容易的，问问题要能抓住要害，否则根本起不到任何作用。

      @思考：读到柴静在刚做“新闻调查”跑新闻时的那种拼劲，那种担心自己因准备不足而不敢面对的心理纠结，我在想：其实每个人都有令  自己担心害怕的 事情，但是那一刻始终会如期到来，我们也只有硬着头皮上。在真正面对这个令你害怕的时刻时，其实这件事本身显得也并没有那么可怕。反倒是事后回忆一下，你甚至会感谢当时自己的坚持，让自己变得更加有信心。

      - 其实只有善良的人才能刚性，善良的人做“对抗性”采访，不会跃跃欲试地好斗，而是当他在决定守护真相之时，他就会变得与非真相的假象之间进行绝不撤步的对峙！

      @联想：在进入职场时必然会有一段不适应期(因为此时你在职场中的立足点为0)，因此自己绝对要找到合适的方法迅速调整状态以适应新的环境(尽快找出自己的立足点)！并用自己顽强的拼搏精神为自己在职场中搏出一片天地，赢得大家的认可和尊重！

     
     

    + - 5. 我们终将浑然难分，像水溶于水中  (讲述为社会所不容的那群真实存在的人的生活状态)

    • 生与死，苦难和苍老，都蕴藏在每一个人的体内，总有一天我们将与之遭逢。

      - 那些在社会中真实存在的人群：戒毒人员，超期羁押人员，同性恋人群等。
      @联想：这些人没有得到来自社会的尊重，他们可能曾经犯过错也可能没犯过，但他们却得到了来自社会的过重的惩罚。所以自己在不爽的时候，不要总觉得自己遭受到了最大的不公，其实在这个社会上，比你遭受更大不公的人群多的是！所以切忌：“不要无病呻吟”！
      @指导：以后自己再遭遇到不爽的时候，要学会即时调整自己去适应去改变，而不是一味地去抱怨。

      - 你在家里，在单位，在认识的人面前，你被当成一个人看，你被尊重！但是在一个没人认识你的地方，你可能会被当成东西对待。我想在任何地方都被当成人，不是东西，这就是尊严。

     
     

    + - 6. 沉默在尖叫 (关注家暴与留守儿童问题)
    - 家庭是最小的社会单元，门关上之后，在这里人们如何相待，多少决定了一个社会的基本面目。

    •  

      - 不仅是从农村到城市，好像这个社会一直存在着另一面，那就是家暴。由此出发，这个问题也引出了一系列的社会问题。包括：家暴引起的犯罪，子女的抚养义务，父母的赡养义务，留守儿童犯罪等一系列问题都出现了。
      @思考：这个社会中有些人犯罪是出于本性的恶，但另外还有些人犯罪却是为自己家庭不幸，为社会机制不健全所导致的。需要记住的是：人都是一样的，对追求幸福的愿望一样，对自身完整的需要一样！当我们面对自己所不能理解的现象时，我们不要以自己的经历和标准去评判别人，我们要学会宽容，学会理解！

      - 我很庆幸自己生活在一个没有家暴环境下的家庭中，虽然爸爸对妈妈很是自私，从没有明着表示出自己的心意，但起码他尊重妈妈，在我的记忆里，他从来都没有打过妈妈！
      @联想：所以我一直不知道也无法想象原来家暴问题在中国是这么样的一个大问题。我以后要记住无论在家庭方面还是生活中我必须坚持做的底线是：绝不打女人。
      @联系:《蔡康永的说话之道》中所说的，心意一定要用实际行动去表达，不要期盼对方可以心领意会。

     
     

    + - 7. 山西，山西 (讲述城市环境污染问题)
    - 山西姑娘没见过小溪青山之类，基本上处处灰头土脸，但凡有一点诗意，全从天上来。

    •  

      - 城市发展与环境污染之间的矛盾 (这个问题在全国都很普遍)
      @联想：本节讲到了她回忆自己儿时那种青山绿水的生活。其实我上小学的时候，也有一条河，环境现在想想也是好的不得了，河里鱼虾等各种水生生物多的是，我和庄子里面的玩伴一起：钓河虾，摸鱼也是玩的不亦乐乎！
      后来随着农村的发展，农药化肥的使用以及人类大量的捕捞等等原因导致水环境也是变得越来越差，河里开始长满了杂草，鱼虾基本灭绝，就连水也变得少了，河都快干涸了。简直到现在都还很怀恋小时候的那段无忧无虑的人生最美好的时光！

      - “山西多煤”引发的思考

      • · 说明在远古时期这里曾是芳草鲜美的肥美之地，在旧石器时代这里遍布河湖沼泽，有着各种繁茂的植物。到石炭纪时这些植物在地壳变动中被埋入地下，千百万年来这些植物的枝叶和根茎堆积形成了极厚的黑色腐质层，于是也就有了煤。

       

      - 直到70-80年代山西那里依旧也还算是环境宜人，但是自从开始挖煤后，那里的环境发生了翻天覆地的变化，这里不仅动植物种类和数量大幅减少，而且再也没有了燕子在屋檐下搭窝，下雨后再也看不见彩虹了！
      @思考：整个中国从改革开放一直走到现在还不足40年的时间，虽然中国经济得到飞速发展，但代价也是巨大的，以前的青山绿水和满天繁星在中国的许多大地上已然不复存在。中国遵循的策略可能就是：先发展再治理。
      @联想：这种策略似乎是在效仿工业革命时期的欧洲，那时的欧洲在工业革命的浪潮下也是环境随着社会发展而恶化严重，但如今21世纪的欧洲环境居然有奇迹般的治理好了，环境优美空气宜人！
      @思考：这环境的恶果却必然是我们这几代处在时代发展最迅速时期的人所必须承受的！我们这几代人背负着发展的重任，同时也不得不承受这发展的恶果！

     
     

    + - 8. 我只是讨厌屈服 (以公民的名义起诉所谓的权威，起诉铁道部等案)
    - 今天你的权利受到损害，你不说话，明天你就会失去更多的权利。
    - 关于对“作为人的尊严”问题的思考！

    •  

      + 引言:《飞越疯人院》中的麦克默菲。他押了十美金，搓了搓手，使劲抱住那个台子，没搬起来，再一次用力，还是搬不动。他只好腿下，突然，他大声叫起来：“去他妈的，我总算试过了，起码我试过了”！

      - 郝劲松为五毛钱状告北京地铁公司，为火车上没发票起诉铁道部和国家税务总局，他以“公民”的名义打官司。
      - 他说：“在强大的机构面前人们往往除了屈服别无选择，但我不愿意。我要把他们拖上战场，我不一定能赢，但我会让他们觉得痛，让他们害怕有十几二十几个像我这样的人会站出来，让他们因为害怕而迅速改变”！
      - 这就是所谓的“钉子户”，社会的刺头，但是我们的社会需要这样的人。

       

      + 公民和普通百姓的区别是什么？

      • · 公民能独立地表达自己的观点，却从不傲慢，对政治表示服从，却从不卑躬屈膝。能积极地参与国家的政策，看到弱者知道同情，看到邪恶知道愤怒，我认为这才算是一个真正的公民！
        · 一个公民被宪法赋予了很多的权利和义务，但是由于在现实生活中我们往往并没有意识到自己公民的身份，对所谓的权威机构所导致的不正义事件只能选择服从，这导致我们丧失了很多我们作为公民应有的权利！

      @联想：这次家里的拆迁与回迁事宜，里面充满了不公正之处，我能怎么办，家里财产就这样任由别人一两句话决定归谁。这些人真他妈心太黑了！自以为仗着有权就公然吆喝，还有一些所谓的地头蛇，以为自己很牛的样子，当众说出那些没有公正的话，“你带不走给我”，简直就是乘火打劫！
      @思考：我能有什么办法呢，面对那种小地方的地头蛇强权以及自己的弱小，只能选择屈服。只能抱怨“小城市真他妈的没指望，以后如果老子混好了，全他妈给我滚蛋”，一天到晚牛气轰天的样子，要不是有事谁他妈愿意鸟你们一句。这件事也在鞭挞着我：“以后一定要往大城市发展，一定要努力往上爬。不要再受这种窝囊气”！

      @思考：我以后为什么要努力，因为就是想让自己包括我的家人能够获得那种真正的作为人的尊严！无论是熟人还是生人，让别人把我们当做是一个人来看，而不是一件东西！
      - 你在家里，在单位，在认识的人面前，你被当成一个人看，你被尊重！但是在一个没人认识你的地方，你可能会被当成东西对待。我想在任何地方都被当成人，不是东西，这就是尊严。

    + - 16. 逻辑自泥土中剥离 (记录新闻调查六年最常见的农村征地问题以及由此而引出对中国发展中的经济问题和社会问题的思考)
    - 政府、开发商、拆迁户三者之间在土地与开发问题上所形成的复杂三角关系。
    - 这个问题自中国政府宣传“加快城市化进程”，以及加速经济发展保住GDP目标开始，一直到现在2016年，问题依旧存在，矛盾依旧突出！

    •  

      @总结：本章对农村征地问题进行了比较详细的分析，值得反复思考！因为12年开始我家也被政府征收用于开发了，也就是说前面的那些问题其实说白了和我也没多大关系，而这个问题则与我息息相关！

       

      - 首先，经济学家周其仁给我们大众看待这个问题提了三点意见：
      · 1. 不要用道德的眼光来看待经济问题；
      · 2. 不要妖魔化地方政府；
      · 3. 不管左中右，质量最重要。

       

      + 征地过程中存在什么样的经济问题？

      • - 所谓经济问题，说简单些，就是指买和卖！在征地这件事上，买的是开发商而卖的是农民！
        - 政府在其中承担什么角色呢？
        · 他其实就相当于一个中间代理人的角色，他把地从农民那儿以低价买过来，再跑到开发商那里以高价出售出去！政府在此过程中赚取足够大的溢价作为财政收入的一个重要部分！

        - 政府在此过程中所承担的角色是法律所规定的，是必须的！但是为什么是这样等一系列问题也就随之而来了[见下问]！

       

      + 农村土地归集体所有而非国有，因此为什么农民的地就必须要让政府来卖呢？

      • - 根据《土地管理法》: 任何单位和个人进行建设，需要使用土地的，必须依法申请使用国有土地[包括国家所有的土地和国家所征收的原属于农民集体所有的土地]！

        - 因此：农村集体所有土地要想变成建设用地，就必须被国家收为国有土地，否则就违法了！因此根据这条法律，政府这个中间代理人的身份就是必须的了！

       

      +《土地管理法》上的这条法律合理吗？

      • - 立法都是有依据的，这条法律也不例外，它是根据宪法而来的！
        - 具体依据是82年五大五次会议通过的《中华人民共和国宪法》规定：“城市的土地属于国家所有”！
        - 也就是说：城市土地是国有的，而农村土地不属国有，因此农村要想城市化，要想建设发展，就必须通过政府征收的方式将其土地收为国有！

       

      + 为什么“城市的土地”非得属于国家所有呢？八二年宪法中的这句话是从哪来的？

      • - 此宪法规定的历史由来：‘文革’前国家没财力搞建设，而到了82年人口又在膨胀，人开始没地儿住了，北京的四合院、上海的小洋楼都得住人。可是城里不像农村，它没土改，土地都还有所谓的地契。因此为了让人能住进去，干脆就修改法律吧，将所有城市土地收归国有，就这样简单的解决了问题！宪法这么规定其实是相当于搞了一次城市的土改！

        - 没想到这句规定 原本是为了解决城市住房问题，现在却误打误撞的给农村征地制度造成巨大麻烦！随着中国城市化的开始，城市土地都属于国家所有，因此，农村土地一旦要用于建设，必须都要经由政府之手，征收转为国有土地！

       

      - 在征收土地问题上的真正的“经济问题”来了：在市场经济中，自由买卖和等价交换是众所周知的市场准则！但是征地这件事上却完全未遵守该准则！
      - 政府征地给农民的不是等价的所谓的价格，而是补偿！
