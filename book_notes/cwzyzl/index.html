      + 什么是可控范围？为什么要扩大我们的可控范围？

      • - “可控范围”：是指那些你可以直接施以控制和影响的事物！
        - 还有一些事物，你虽然感兴趣，但你却无法对之施加影响，这些所有与你有关的事物所处的领域就是你的“私人领域”。
        - 为什么要扩大自己的可控范围？因为财富往往就产生于自己的可控范围被扩大之时！

       

      + 如何扩大自己的可控范围？(四种扩大可控范围的方法)

      • 1. 离开舒适区 (那些令我们感到安全和舒服的区域)
        * 当我们踏入新的领域，付出一切努力并获得成功时，我们就会感到自己最强大的生命力。
        2. 解决问题 (发现和解决可控范围之外的事物，把问题看做是成长的机会)
        * 问题之所以成为问题，就是因为它处于我们的私人领域之内，却又在我们的可控范围之外。
        * 注意：回避问题是永远不可能致富的，无论是谁，只要想挣更多的钱，就必须研究一系列的问题。
        3. 要会提出正确的问题 (问“为什么”不如问“怎么做”)
        * 当自己遇到问题时，问“我能行吗”，不如问“我要怎样做？才能办成这件事”，问你自己怎样才能取得成功，你才会成功，这促使你去寻找在你可控范围之外的机会！
        @思考：我现在做科研就是在不停地问自己“为什么要做Mutation-testing”，但我却一直在回避“我要怎么做,才能把它做成功”！可悲啊。
        4. 扩大自己的私人领域
        私人领域一扩大，可控范围也就会跟着扩大。

       

      + 遇到“问题”之后的思路：

      • - 遇到问题后不要仅仅是问自己：“怎样才能解决这个问题”；而是要换一个角度问自己：“我怎样才能创造一种环境，以防止这样的问题再次出现”！
        - 这样你才能扩大自己的可控范围！

       

       

      + 要如何做出一个“决策”？

      • - 当你做决策之时，应当问“是否”；一旦你做出了决定，你就不应该继续关注“是否”的问题了，而应该关注“怎么做，怎么解决”！
        - 决策的流程：
        · 首先对一件事问一个“是否”的问题；
        · 接着确定了之后，应该思考：“我应该怎么做，怎么解决当前所遇到的难题”，而不是再返回去思考是否该做的问题！
