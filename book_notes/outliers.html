异类 · 不一样的成功启示录
作者：[加] 马尔科姆·格拉德威尔

  • 引言 罗赛托之谜

    • + 什么是“异类”？

      • - 它是指 “离开 主体或者相关本体 的东西”，或者说是 “一个在价值上能与其他样本显著区别开来的统计观察值”！

       

      + 什么是 “罗赛托之谜”？

      • - “这里没人自杀，没人酗酒，没人吸毒，犯罪率也很低。他们没有人领救济金。我们甚至没有发现任何人患上胃溃疡。生活在这里的人大多都是自然死亡，就这么简单。”
        - 这一点超出了我们的日常经验，惯例在这里并不适用，罗赛托变成了一个“异类”。

       

      + “罗赛托健康之谜”的解答：

      • - “罗塞托的秘密不在于饮食，不在于运动，也不在于遗传基因和地理位置，而在于罗塞托社会自身。”

        - “这里的人们崇尚平等理念，富人不会浮夸炫耀，整个社会都愿意帮助失败者走出困境；这里的人们免受现代社会的压力，生活在一个自给自足的山区小镇”

        - 沃尔沃关于“罗赛托之谜”的研究工作，意义在于“努力揭开健康的奥秘”，而本书的意义则在于“想要揭开成功的奥秘”！


     

    第一部分 机遇 (该部分讨论成功所依赖的几种优势累积模式)

    • + - 1. 马太效应 ( 成功往往就来自于优势的不断累积,而那最初的微弱优势往往却来自于机遇 )

      • “凡是有的，还要加给他，叫他有余；没有的，则连他所有的，也要夺过来。”《马太福音》

         

        + 反思：我们现代的社会都在 一种所谓的“精英教育体系” 内谋求发展，每个人都想要成为各行各业  “金字塔顶端”的那个人，然而能成为“金字塔顶端的男人”(能成功的人)真的都是依靠 自己的表现,自身的能力 吗？

         

        + 成功到底靠的是什么？

        • -《异类》这本书想要证明：“个性作用并非个人成功的决定因素。成功人士并非白手起家，他们以某种形式获得家族的荫蔽和支持。那些最终变得卓尔不群的人看似完全依靠个人奋斗，其实不然。只关心成功人士是什么样的人这远远不够。我们还必须探究他们从哪里来，只有这样才能明白为何某些人成功，其他人则流于平庸。”

         

        + 优势会不断地积累

        • - “筛选，分组 和 区别训练” 把同组内的年龄较大者的优势不断地放大，导致他们最终能取得斐然的成绩。
          - 这种选拔机制会产生“优势累积”的不公平现象，这种现象则广泛的存在于社会的各个领域，甚至是“学校教育领域”。

          - 我们随意选取的 ‘分组日期’ 竟然会因为这套选拔机制而造成如此深远的影响，而人们对此却竟然毫不知情，也毫不关心！
          @思考：如今之中国《义务教育法》规定，小学入学年龄：“截至当年8月31日年满6周岁儿童可以报名就读小学一年级，而9月1日出生的孩子，则要等到下一年”。从这条规定看出：生孩子要赶在8月31日之前生，从“马太效应”又得出：生孩子还要尽早生！(在6周岁这一区间内，1月份生的孩子比8月份生要早发育几个月，这微弱的优势可能会由于“累积效应”变得越来越重要)！正是这种一开始的微弱领先优势造就了那些天才们辉煌的成就。

         

        + 天才并非一开始就表现出众，一开始他也只是比别人优秀那么一点点。

         

        “由于我们过于相信成功是个人的事情，所以我们损失了许多造就天才的机会；很多制度甚至过早地将一部分人划入失败者行列，阻碍了这些人成才。”

       

      + - 2. 一万小时法则 ( 成功 = 机遇+大量练习+刻意练习 )

      • + 本章讲了四个例子：Sun创始人比尔·乔伊，甲壳虫乐队(汉堡训练)，比尔盖茨，史蒂芬·乔布斯。
        @思考：有的时候真的可谓是“勤能补拙”，“时势造就英雄”，非凡的机遇加上后天的努力，这些人怎么能不成功呢？

         

        + 成功是天赋加上后天努力，随着研究的深入，我们发现天赋的作用其实很小，而后天努力的作用其实很大(10000小时的对某一特定项目的长期刻苦训练)！任何一个领域的世界级水平都需要起码长达10000小时的训练！
        @思考：训练(练习)的目的：不是为了你一次就能把事情做好，而是为了帮助你把事情越做越好~~

         

        + 研究结论表明：1.根本就没有“与生俱来的天才”(那种花比别人少的时间就能达到比别人高的成就的人)；2.也没有所谓“一生的劳苦命”(那种努力程度远比别人高但就是无法比别人更优秀)。

         

        回顾成功人士的历史，“恰恰说明，那些获得特殊机遇眷顾的人们总能努力工作，并胜任使命；与机遇相伴的人总能取得非常的成就。他们的成功并不仅仅是自己努力的成果(10000小时的练习)，更是独特的成长环境促成的结果(时势机缘)。”
        @联想：这里的一万小时理论也被拓展开了，我们花1万小时做一件简单的充分劳动并没有什么用，一定在花这1万小时的同时还需注重四个字“刻意练习”！要学会反思，学会在每一次的练习过程中不断地精炼自己！而不能仅仅是简简单单的从事重复工作。

       

      + - 3. 天才之忧(上) ( 智商对成功的影响存在门槛效应  )

      • + “仅靠智商很难区分两个聪明孩子”！

         

        + 一个很重要的认识：“智商存在门槛效应”，这就是说：智商和成功只是在一定程度上相互关联，一旦某人的智商超过了120，此时更高的智商往往并不意味着就能转化为更多的现实优势！
        @联系：当两个人都是聪明人时(智商都很高时)，智商本身对成功的影响已经就变得不再重要了！一个智商130的科学家和一个智商180的科学家都一样可以获得诺贝尔奖。

         

        + 这种智商对成功的这种“门槛效应”，有点像身高对打篮球的影响。一旦你的身高足够高，身高对成功就变得不再重要了，人们就开始关注你的速度、球感、灵活性、球技和投篮准确性等其他方面！
        + 智商超过门槛之后，成功的又一大影响因素是：想象力！
        @思考：如何评估自己的想象力？采用与智商测试这种收敛性测试完全不同的 发散性测试 进行评估 (比如：著名的“砖块-毛毯”测试)！

         

        + 从上面的认知和结论可以看出：“智力和成就之间往往真的并没有那么大的关系”。

       

      + - 4. 天才之忧(下) ( 智商超过门槛后情商变得更为重要，而这又是一大重要的环境机遇·家庭  )

      • + 成功除了智商之外的其他重要影响因素：“实践智力”(Practical Intelligence)，或者说情商(人在情绪、意志和耐受挫折等方面的品质)。
        {实践智力包括“知道该向什么人说什么话，该在什么时候说，该怎样说才能达到最好效果”等}

         

        + 思考：如何提高情商？

        • - “提高情商” 本质上就是想把 不可控情绪 转变为 可控情绪，从而增强理解他人及与他人更好地相处的能力。
          - 那么又该 如何转变？请注意理解一条重要的佛教偈语 “一切皆空”，“万事万物皆具有空性”。

         

        + 思考：实践智力 从何而来？

        • - 智商可说是天生的能力，但 社交常识(实践智力,情商)这类技能 的获得场所往往却是在 家庭(这也是一种重要的环境机遇)！

           

          - 原则上家庭教育孩子的方式只有两种(是通过阶级状况划分的)：家境富裕的父母教育孩子采用“协同培养”模式，家境贫寒的父母教育孩子则采用“自然成长”模式(或叫做“放羊”模式)。
          + 结论：父母的生活方式、父母所属阶级，对子女的成才至关重要！

           

          · 协同培养模式：“家境富裕的父母总是让孩子一刻也不能闲着，让孩子穿梭于各项活动之间，并听取孩子对老师、教练或是队友的评价。”
          [把兴趣发展成专长] [懂得与孩子商量，不会下硬性命令]
          {这种家庭培养出来的孩子：更懂 权利意识 和 平等意识}

          · 放羊模式：“而在穷人家的孩子的生活中完全没有这种紧张的时间安排，他们的活动不会是每周两次足球训练，而是和亲戚或者邻居的孩子在外面玩耍。”
          [把兴趣仅当做特点] [喜欢推卸对孩子本应承担的责任]
          {这种家庭培养出来的孩子：则 更倾向于安静羞涩，表现的十分顺从}

           

          - 这种培养模式上的“阶级优越性”到底带来了什么？富裕家庭的“协同培养”模式所培养出来的孩子往往表现得 更加机灵、沉着、衣着得体和富有吸引力！而在贫穷家庭的“放羊模式”下培养出来的孩子则天生缺乏一个可以帮助他或教他如何获得应对社会经验(实践智力)的社群！

           

          + 如何避免像兰根这样的超高智商天才的悲剧重演？说实话，真的很难，因为这要求你必须自己为自己营造一个 能帮助你学会更好地与社会沟通 的社群 (也即：要刻意练习你的情商)！

       

      + - 5. 乔·弗洛姆的三个启示 ( 成功绝不会是随机事件,走向成功绝少不了环境机遇的帮助 )

      • + 结论：成功人士不可能独自走向成功，他们总是 特定地点 和 特定环境 的产物！那些“异类”总能在成功的路上获得帮助！

         

        + 再看一个例子：世界法律事务所之翘楚 乔·弗洛姆 世达律师事务所！

        • - 他成功的机遇何在？
          1. 身为犹太人的背景：这个背景让他们身处逆境，却也最终成为他们成功的机遇之源；
          2. 出生恰巧遇到生育低潮期：生育低潮期为其带来人口优势，他可以拥有更少的竞争对手 和 更多享用社会资源的机会等。
          (从某种意义上来说，成功的可能性不来自于我们自身，也不来自于我们父母，而是我们的时代！{时势造英雄})
          3. 父母给他们带来了“自主、复合、付出就有回报”的奋斗理念！
          (他们懂得 “为了把握机遇，你必须拥有任劳任怨的美德，而犹太移民愿意苦干。他们愿意做自我牺牲，他们生活节俭，善于储蓄，对投资精打细算”)
          (所谓的复合就是指：所干的事结合了自己的意志力和想象力。)

         

        + 什么样的工作可以称得上是一份令人满意的工作？思考如何看待加班这件事！

        • - 它具备三个属性：自主性，复合性，付出与回报的关联性！
          - 一个工作只要能满足这三条标准就能被称之为“有意义的，令人满意的”工作！
          (只有不断地从事没有意义的工作，才称得上是艰苦的工作，一旦它变得有意义了，这个工作就变得不再艰苦，而是会变得令你快乐和迫不及待)
          @思考：如何看待互联网企业的加班？如果你想改变世界(自主性)，而且觉得自己做的事情就真的是正在改变世界，别人都在用(想象力)，而且你每天多工作几个小时(意志力)就能得到丰厚的回报(付出就有回报)，那么何乐而不为呢？这将是一份多么有意义的工作啊~

         

        + 成功从来就不是随机事件！成功是由一系列 可预知且强有力的优势环境和机遇 所构成！
        (要么更聪明，要么更勤奋，能抓住机遇，为了取得最后的成功可以不择手段)


     

    第二部分 文化传承 (该部分讨论个人成功是否也会受到文化传承的影响)

    • + - 6. 小镇哈伦 ( 理解文化对一个人的影响  )

      • “胆小鬼游戏又称胆量游戏，简单说来就是：两个驾车撞向对方的死对头，都希望对方会在最后一刻转向，而自己的胆量能胜对方。胆小鬼游戏无外乎有四种结果：一是在最后一刻，甲乙双方都转向；二是甲先转向，乙获胜；三是乙先转向，甲获胜；四是甲乙都不转向，冲突或战争爆发。”

         

        + 从“血染哈伦”这种复仇式的暴力冲突中，我们得出一个重要结论：“荣誉文化”推动暴力蔓延。
        @联系：理解荣誉文化：“荣誉文化植根于高地或富庶地区的边缘地带，由于这些地方很难耕种，人们大多依靠牧羊生活。游牧文明和耕种文明大相径庭。从事耕种的农民能否活下来依靠的是人与人之间的相互合作，而从事游牧的牧民则更多依靠自己。所以他们养成了好斗的性情：他们必须通过自己的言行表明自己不是弱者，要对危及他们名誉的哪怕最轻微的挑战予以最坚决反击——这就是“荣誉文化”的含义。游牧文化所在的地方，荣誉感在男性那里占据存在感与自我价值的中心。”

         

        + 荣誉文化可以用来解释很多身处偏远地区的人的思维为何如此独特！

        • - 因为 “在穷乡僻壤，暴力案件发生的原因不是经济状况，而是个人尊严，人们实际上是为自己的荣誉而战。”
          - 也就是说：一个民族的文化遗产才是这类事件背后的主导诱因。

         

        + 文化传承 具有如此强大的力量，它植入人性，影响长存。“文化直接决定了我们看待世界的方法和行为模式”。

       

      以下各章的目的是想要：揭秘不同文明中的人如何利用其特有的文化取得成功，同时也想要找寻如何利用文化传承来使人们获得成功的秘诀！

       

      + - 7. 飞机失事的族裔理论 ( 权利距离指数(PDI)对人做人行事的深刻影响 )

      • + 利用文化改革取得的成功  举例：“大韩航空是将自身从一个安全纪录糟糕的公司转变成世界最好航空公司”
        - 故事背景：大韩航空801航班失事！引发的对 韩国文化 在事故中所起到的作用的思考。

         

        + 关于空难

        • - “一个典型空难通常包括7个人为的错误。飞行员中的一位犯了错误不会有什么问题，然后他们中又有一位在前一个还没有发展成事故的错误的基础上再犯一个错误，接着是第3个、第4个、第5个、第6个直到第7个。结果正是所有错误操作的累加导致了最终的灾难。”

          - “造成飞机失事的症结主要集中在团队协作和相互沟通上。”

           

          - 反之，要想成功化解空中危机，他必须具备多种素质：1. 技术必须过硬，2. 同时也必须具备良好的沟通素质  (“这里的沟通不仅意味着向机组下达降落指令，还意味着鼓励、安抚、说服、商讨，以及用明白无误的语言与他人共享信息。”)。

         

        + 飞行过程中没有级别概念，要严禁客套！

        • - 什么是“客套”？语言学家解释为“缓和性语气”！

          • · “这一术语是指：低调处理 所说内容 以取悦听众。当我们表示礼貌、害羞、窘迫，或对权威的恭敬时，我们会使用缓和性语气。”
            · 六种缓和程度不同的语气：1.命令{最直接最清晰的表达，缓和性为零}；2.责任性陈述；3.建议；4.询问；5.偏好；6.暗示{最难解读和最容易被拒绝的对话方式}。

           

          - 可以说“缓和性语气”可以用来解释飞机失事最令人费解的部分！而“与缓和性语气作斗争，成了过去15年间商业航空业界的主要战役。”

          - “这场针对缓和性语气的战役比任何其他措施都更加有效，以至于近年来的航空事故因此大为减少。”

           

          - 由 文化传承 所导致的 东西方 说话方式的不同：
          · “西方人的沟通方式在语言学上被称为“以说话者为导向”——意思是说话者有责任将意思清晰明白地表达出来。”
          · “东方人却是“以聆听者为导向”的。也就是说，意思是否搞清楚，取决于聆听者自己。”

         

        @理解：空难的深层次原因的文化视角(心理学视角) : “跨文化心理学研究的经典理论框架-霍夫斯泰德文化维度”！

        • - 其中最引人注目的一个维度：“权利距离指数(Power Distance Index, PDI)”
          - “权力距离是指人们对待比自己更高等级阶层的态度，特别是指对权威的重视和尊重程度。”

           

          - 研究结果表明：
          “在权力距离指数低的国家中，掌权者对手中的权力感觉不好意思，并试图淡化其重要性。为了行使权力，会努力让自己看上去不那么强势。领导们以削弱自己权力象征的方式加强自己的亲民形象。”
          (美国就是典型的 低PDI 的国家，信奉“人人生而平等”的美国精神!)

         

        + 本质上：航空业 向“缓和性语气”开战，其目的就在于想要“降低驾驶舱内的权利距离指数”！“让副机长在机长面前敢于维护自己的意见，必须助他们克服所处文化的权力距离。”

         

        结论：我们取得事业成功的能力 与 我们的文化背景 紧密相连！“在通向职业成功的道路上，除个人努力以外，我们所处的文化、历史和外部环境对我们能否成功具有决定性意义”。

       

      + - 8. 稻田与数学 ( 勤劳坚韧 的稻田文明赐予亚洲人天赋异禀 )

      • + 背景问题：为什么亚洲人普遍数学都较好，智商都较高？[与种稻民族“辛勤劳动”的文化特性和习惯有很大的关系]

         

        - 一段种植稻谷的历史，诉说着亚洲人民积累种植技术的艰辛历程。
        - “稻田是修造出来的，是需要精心呵护的”，“为了防止灌溉水渗漏，需要以硬泥作底。但是种子不可能在硬泥里生长，所以在硬泥上面还要有一层软泥。对于结合硬泥软泥的黏土层要精心设计，从而使农作物浸泡在适量的水中以利于水分吸收。稻谷需要不间断施肥，这也是一门艺术。”

         

        + 东西方不同的农业发展模式

        • - 从历史来看，西方农业发展模式是“机械导向型”：在西方，农场主想提高工作效率增加产出，就需要购进更多高新技术设备，如拖拉机、打谷机、压捆机、联合收割机等，用机械代替人力。{购置机械}
          - 而东方的稻田文明则是“技术导向型”农业：你只有更精心除草，更小心施肥，密切观察水位变化，有效控制黏土层，利用好稻田的每一寸土地，你才能获得丰收。{精心呵护}

         

        + 建立在勤劳基础之上的文化：懂得“一份耕耘一份收获”的道理，都在追求过有意义的生活！养成了要“努力工作”的精神品质。
        - 反观比尔盖茨，比尔乔伊，甲壳虫乐队等那些愿意花一万小时来打造自己的世界级成功人士，“努力工作”是他们所有成功人士的共性！

         

        @思考：“既然勤奋那么重要，那么我们怎样做才能算做是 勤奋 呢”？{首先最重要的一点：要端正自己的态度！态度比能力要更重要！}
        - 只要你有意愿，你就能驾驭难题！@思考：成功是什么？“成功就是坚持不懈，就是顽强不屈，就是别人花30秒钟就放弃的事你却花22分钟去思考的坚定信念。”

         

        结论：亚洲人的数学为什么天生就具有优势？因为他们的共同点都在于：他们的文化都是 崇尚-有意义工作,崇尚-劳有所得  的稻田文明！

       

      + - 9. 玛丽塔之幸 ( 利用改变文化影响成功的例子:KIPP )

      • + 背景情况：KIPP (Knowledge Is Power Program "知识就是力量计划") 试验性公立学校KIPP学园项目启动！

        @结论：“KIPP的成功并非依靠环境、师资或制度创新，它的成功依靠的是人们对文化传承的深刻理解。”
        @思考：人际交往中的“double-SLANT-法则” 感觉可以给我在“与人相处”过程中带来很大启发：
        “Smile, Sit-up, Listen, Ask-question, Nod when being spoken to, Track with your eyes”! <==> hone each others communication skills <==> communicating with others more skillful and more EQs. {提高人际交往情商的"double-SLANT"法则}

         

        ※ 思考：“穷人家庭里的孩子相比富人家庭，成绩到底差在哪儿”？

        • ~ 一大最主要原因就在于：休假期间，穷人的孩子停滞了而富人孩子却在继续发展。
          · 研究表明：在正常上学的学期间，穷人家的孩子可能由于更勤奋反而比富裕孩子学得更多一点；但一旦放了假，这些穷人孩子们就开始大大落后了！
          · 事实上：“富裕家庭孩子与贫穷家庭孩子的差距主要是在 非在校时间内 形成的”。

           

          @深刻反思过去的自己 “为什么就总感觉自己很努力却一直也不出色”？
          - 从作者的这段话中，我终于明白了“自己一直很差”的原因--“放假的时候(休息的时候)我就完全停滞了呀”！

          + 举例：两种典型的孩子教育方式 ("拉开差距"的原因立现)

          • 1. 富人家庭的孩子“小威廉”：“小威廉的父母按照“协同培养”的原则教育孩子，带他去博物馆，在各类辅导课程中报名，参加学校的夏令营。他在家有很多书可以看，所以他不会觉得无聊。他的父母视孩子充分接触社会为己任。因此暑假过后，小威廉的阅读和数学成绩大幅提高就不足为奇了。”
            2. 穷人家庭的孩子“布林德”：“父母没有足够的钱让她参加夏令营，她也不会进入特长班学习。在家无聊的时候她也没什么书可看，能看的也许只有电视。当然她的假期很可能过得也很愉快，结交新朋友，外出野游，看电影，过一个许多人都向往的无忧无虑的暑假。现实是：然而一个无忧无虑的暑假对她的阅读与数学成绩毫无助益，经过一个暑假，她与威廉之间的距离更大了。
            - 结论：威廉并不一定就比布林德尔更聪明，但威廉学的东西比她多：威廉花了整个暑假的时间学习各类课程，而这期间，布林德尔却在看电视或者在外面玩耍。”
            {富人家庭孩子之所以比你成功或优秀的原因在于：他们(或者说是他们的父母)懂得在你休息的时候超越你。}
