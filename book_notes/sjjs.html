  • + - 第一章 我们的宇宙图像
    • =  艾萨克·牛顿  于1687年出版了他的《数学的自然哲学原理》，这部书也许是有史以来物理科学上最重要的单独的著作。
    • = 宇宙 的 "大爆炸模型"
    • = 科学的终极目的 在于提供一个简单的理论去描述整个宇宙。
    • = 广义相对论 和 量子力学
      • 广义相对论 是描述引力和宇宙的大尺度结构，从只有几公里直到大至可观测到的宇宙范围的尺度的结构；
      • 量子力学 则是描述极小尺度的现象，例如万亿分之一英寸。
      • 相对论 和 量子力学 这些理论已经给我们带来了 核能 和 微电子学的革命。
  • + - 第二章 空间和时间
    • = 牛顿三大定律
      • 第一定律(惯性定律)：任何物体都保持静止或匀速直线运动的状态，直到受到其他物体的作用力迫使它改变这种状态为止；
      • 第二定律(F = ma)：物体在受到合外力的作用会产生加速度，加速度的方向和合外力的方向相同，加速度的大小正比于合外力的大小且与物体的惯性质量成正比；
      • 第三定律(作用力与反作用力)：两个物体之间的作用力与反作用力，在同一条直线上，大小相等，方向相反。
    • = 牛顿定律引出：并不存在一个静止的唯一标准，静止是相对的，不是绝对的；

      • 缺乏静止的绝对的标准表明，人们不能决定在不同时间发生的两个事件是否发生在空间上的同一位置。“绝对空间”的想法被渐渐的推翻了！

        正如声波在空气中传播一样，光波应该是通过一种无所不在的称为 "以太" 的物质进行传播。

        爱因斯坦在1905年的一篇著名论文中指出：只要人们愿意抛弃绝对时间的观念的话，整个以太的观念则是多余的。

        • 不论观察者运动多快，他们应该测量到一样的光速！
        • 相对论限制任何正常的物体永远以低于光速的速度运动！只有光和其他没有内禀质量的波才能以光速运动。
        • 相对论终结了 "绝对时间" 的观念！
    • = 在相对论中 是按照时间和光速来定义距离，单位是光秒(表示光在一秒钟所走过的距离)。
    • = 光锥 的概念

      • 时空图中从事件P出发的光脉冲的轨迹！

        对于给定的事件P，人们可以将宇宙中其他事件Q分成三类：P的将来 [Q处在P的将来光锥之内或之上]；P的过去 [Q处在P的过去光锥之内或之上]；P的他处 [Q既不在P的将来又不在P的过去]。

        我们看到的从很远星系来的光是在几百万年之前发出的，在我们看到最远物体的情况下，光是在80亿年以前发出的。这样当我们看宇宙时，其实我们是在看它的过去！【由此联想：<苏菲的世界>里面关于时空观的看法。】

    • = 广义相对论 是 如何被提出的？
      • 由于 狭义相对论 和 牛顿的引力理论不相协调。牛顿理论说，物体之间的吸引力依赖于它们之间的距离，这就意味着如果我们移动一个物体，另一物体所受的力就会立即改变，
      • 换言之，引力效应必须以无限速度来传递，而不像狭义相对论所要求的那样，只能以等于或低于光速的速度来传递。
      • 爱因斯坦在1908年至1914年之间进行了多次不成功的尝试，企图去找一个和狭义相对论相协调的引力理论。1915年，他终于提出了今天我们称之为 广义相对论 的理论！
      •  
      • 这种新的引力理论彻底颠覆了之前对 引力 的理解！在广义相对论中，物体总是沿着四维时空的直线在运行，尽管如此，在我们的三维空间中看起来它是沿着弯曲的途径运动的！
      • 四维上的直线运动 映射成了 三维上的曲线运动，正如看一架在非常多山的地面上空飞行的飞机，虽然它沿着三维空间的直线在飞，但是在二维的地面上它的影子却是在沿着一条弯曲的路径。
      • 由广义相对论所引出的著名的 "双生子佯谬"。【联想：电影<星际穿越>里面的时空旅行中的不同星系之间时间的对比】
    • = 空间和时间
      • 没有了绝对的空间 和 绝对的时间 的概念！
  • + - 第三章 膨胀的宇宙

    • = 利用 多普勒效应 所表现出来的光谱的"红移/蓝移"现象来判断星系离我们究竟是"远去/驶近"。最后的测量结果表明 几乎所有的星系的光谱都是"红移"的，这就表明它们都在远离我们而去！

      = 宇宙不像人们原先想象的那样处于静止，而实际上是在膨胀，不同星系之间的距离一直在增加着。

      = 时间 是从宇宙大爆炸的时候开始的！

  • + - 第四章 不确定性原理
    • = 用来观测粒子的光的波长越长，则它的位置的不确定性越大，而它的速度的确定性越大；用来观测粒子的光的波长越短，则它的位置的确定性越大，而它的速度的不确定性越大。位置和速度不能同时被准确的测量。
    • = 光的波粒二象性
  • + - 第五章 基本粒子和自然的力

    • = 30年之前我们以为的 "基本" 的粒子，原来是由更小的粒子所组成。当我们用更高的能量时，是否会发现这些粒子是由更小的粒子所组成的呢? 这一定是可能的。但我们确实有一些理论的根据，相信我们已经拥有，或者说接近拥有自然界的终极构件的知识。

      = 质子是由2个带2/3正电的上夸克 和 1个带1/3负电的下夸克所组成，所以质子总体带 1 个单位正电；中子是由1个带2/3正电的上夸克 和 2个带1/3负电的下夸克所组成，所以中子总体不带电。夸克之间是靠着强核力紧紧的捆在一起的。

      = 自然中的四种基本力

      • 1. 万有引力
      • 2. 电磁力
      • 3. 弱核力
      • 4. 强核力
        • 强核力具有一种称为"禁闭"的古怪性质：它总是把粒子束缚成不带颜色的结合体。
        • 由于夸克有颜色(红，绿或蓝)，人们不能得到单独的夸克。
  • + - 第六章 黑洞

    • = 恒星的初始燃料越多，它则会被越快的燃尽。这是因为恒星的质量越大，它就必须越热才足以抵抗引力，而它越热，它的燃料就被耗得越快。

      = 具有一个太阳质量 1 M 的恒星坍缩会形成一个白矮星 [  mass ≤ 1.5 M]；

      = 具有十个太阳质量 10 M 的恒星因引力坍缩会形成一个中子星；

      = 具有三十个太阳质量 30 M 的恒星经引力坍缩会形成一个黑洞！

      当引力和泡利不相容原理所引起的排斥力能够抗衡时，就会保持半径不变，形成白矮星。

      = 广义相对论方程存在一些解，这些解使得我们的宇航员可能看到裸奇点，他也许能避免撞到奇点上去，而穿过一个 "虫洞" 来到宇宙的另一个区域。但是不幸的是，所有这些解似乎都是非常不稳定的，最小的干扰，譬如一个宇航员的存在就会使之改变，以至于他在看到此奇点之前，就撞上去而结束了他的时间。

      = 事件视界：就是时空中不可逃逸区域的边界，正如同围绕着黑洞的单向模。

      • 物体可以通过事件视界落到黑洞里去，但是没有任何东西可以通过事件视界而逃离黑洞。
      • 人们可以用诗人但丁针对地狱入口所说的话恰到好处的应用于事件视界："从这里进去的人必须抛弃一切希望"。
  • + - 第七章 黑洞不是这么黑的
    • = 黑洞毕竟不是真正黑的，它们像一个热体一样发光，它们越小则发热发光的越厉害。
    • = 其实任何物体都不能从黑洞的事件视界(黑洞的边界)中逃离出来，何以黑洞会发射粒子呢？量子理论给我们的回答是：粒子不是从黑洞里面出来的，而是从紧靠黑洞的事件视界的外面的空的空间中来的。
    • = 黑洞无毛 是指 所有的黑洞都是呈现出 完美的球形。
  • + - 第八章 宇宙的起源和命运
    • = "热大爆炸模型"
      • 在此模型中，人们发现当宇宙膨胀时，其中的物体或辐射都变得更凉。
      • 当宇宙的尺度膨胀到 2 倍时，它的温度就降低一半。
    • = 地球原先是非常热的，并且没有大气。在时间的长河里，它渐渐的冷却了下来，并从岩石中溢出的气体里得到了大气。
    • = 在我们的宇宙中，大约有1*10^80个粒子，它们是从何而来？
      • 在量子理论中，粒子可以以粒子-反粒子对的形式由能量中创生出来。
      • 物质具有正能量，引力场则具有负能量，在空间上大体一致的宇宙的情形中，人们可以证明，这个负的引力能
      • 刚好抵消物质所代表的正能量，所以宇宙的总能量为零！零的两倍仍然为零！这样宇宙可以同时将其正的物质
      • 能和负的引力能加倍，而不破坏其能量的守恒！
  • + - 第九章 时间箭头
    • = "对于任何正在运动的观察者光速总是一样的" 这一发现，导致了相对论。
    • = 时间至少有三个箭头
      • 1. 热力学时间箭头：在这个时间方向上无序度(或熵)始终趋向于增加的状态；
      • 2. 心理学时间箭头：我们始终感觉到时间在流逝，在这个方向上我们可以记忆过去，而不是未来；
      • 3. 宇宙学时间箭头：在这个方向上宇宙在膨胀，而不是在收缩。
      • 热力学和宇宙学时间箭头指向同一方向，无序度增加的时间方向正是宇宙膨胀的时间方向。
    • = 有一种更快的方法去查究 未来 会发生什么，那就是跳到黑洞里面去。
    • = 对于一个智慧生命的行为而言，一个强的热力学箭头是必须的。
    • = 为了生存下去，人类必须消耗能量的一种有序形式 —— 食物！并将其转化成能量的一种无序形式 —— 热量。
    • = 读书可以使我们大脑中的有序度不断增加！
  • + - 第十章 虫洞和时间旅行

    • = 从广义相对论中又找到其他一些合理的时空，他们允许我们旅行到过去。这些合理时空中的一个即为：旋转黑洞的内部。

      = 爱因斯坦-罗森桥(虫洞)是能连接遥远地区的虫洞，但它们不能保持通畅足够持久，以使任何东西通过。在空间飞船穿过虫洞之前，虫洞就有可能会收紧，最后缩小断裂形成两个分离的奇点(两个黑洞)，而飞船会撞到奇点上去。

      = 为什么我们可以相信能进行时间旅行？

      • 1. 从日食时的光线弯折知道时空可以被弯曲；
      • 2. 从卡西米尔效应得知时空可被弯曲成允许时间旅行的样子(具有负的时空曲率)。
  • + - 十一章 物理学的统一
    • = 物理学的统一(弦理论)
      • 在弦理论中，原先以为是粒子的东西，现在被描绘成在弦里传播的波动，
      • 一个粒子从另一个粒子发射出来或者被吸收，对应于弦的分解和合并。
    • = 弦理论的另一个问题是：它说时空似乎具有更高的维度
      • 而这个理论的确克服了广义相对论的通常限制，即人们不能旅行的比光更快或者旅行到过去的限制。
      • 这一点可以用以下方法描述，我们可以通过更高维度上的最短程来取代低纬度上的最短程，从而在更高维度上抄近路！
    • = 物理学的统一
      • 并不存在宇宙的最终理论，仅仅存在一个越来越精确的描述宇宙的无限的理论序列。
      • 即使我们找到了基本定律的完整集合，在未来的岁月里，我们仍然面临着富有智慧和挑战性的任务，那就是
      • 发展更好的近似方法，使得在复杂而现实的情形下，能作出对可能结果的有用预言。
      • 一个完整的协调的统一理论只是第一步，我们的目标是 完全理解发生在我们周围的事件以及我们自身的存在。
  • + - 十二章 结论
    • = 如果我们确实发现了一套完整的理论，它应该在一般的原理上及时让所有人所理解。那时，我们所有人，包括哲学家、科学家以及普普通通的人，都能参加为何我们和宇宙存在的问题的讨论。
    • = 如果我们对此找到了答案，则将是人类理智的最终极的胜利——因为那时我们知道了上帝的精神！
