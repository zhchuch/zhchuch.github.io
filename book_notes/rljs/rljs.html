    第一部分 认知革命 (约7万年前)

    • + - 1. 人类：一种也没什么特别的动物

      • + 在历史的发展道路上，有三大重要革命：7万年前的“认知革命”(让历史启动)，12000年前的“农业革命”(让历史加速)，500年前的“科学革命”(让历史画下句点并另创新局)。
        + 这本书主要想讲述的就是这三大革命是如何改变人类与其他生物！

        + 本书中所讲的人类指的是"Homo(人属)"的所有现存人员，而讲智人才是指"Homo sapiens"我们这个物种。
        + 人类这个大家族里在历史上存在过许多不同的人种：直立人，尼安德特人，梭罗人，弗洛里斯人，丹尼索瓦人，匠人，智人等等，他们都是“人属”，也就都属于人类。

        + 各个人种的大致分布情况：东非的智人，中东和欧洲的尼安德特人，东亚的直立人，澳大利亚澳洲的丹尼索瓦人。

        + 长期以来，人类一直只是稳定的定位于食物链的中间位置，他们会猎杀小动物同时也会遭到较大型食肉动物的猎杀。直到最近10万年左右的时间，才一跃而居于食物链的顶端！其中一个很重要的原因是：人类成功地掌握了对火的使用，他们变得会“用火”，会控制这个既听话又有无穷力量的工具！
        + 而智人在人类这一众不同人种中，最终能取胜灭掉其他所有人种，最终征服全世界的一大重要原因是：智人拥有独特的语言。

    • + - 2. 知善恶树

      • + 为什么智人能够拥有这种独特的语言，而其他人种没有？(一切纯属偶然)
        现在普遍认可的理论认为，某次偶然的基因突变，改变了智人大脑内部的连接方式，得以让他们以前所未有的方式进行思考，用完全新式的语言进行沟通！这个过程就像是吃了《圣经》里那颗知善恶树的果实一样。智人发生了所谓的“认知革命”。

        + 这种语言究竟有何独特之处，足以另智人统治世界？
        1. 这种语言很灵活，能够用有限的声音组合起来表达无限的句子和意义；
        2. 这种语言能帮助我们进行沟通，使得部落规模得以扩大；
        3. 这种语言能让我们“讨论虚构的事物”，能让我们“一起想象”，从而让我们得以集结大批人力，灵活合作。

        + 由认知革命所引发出来的这一套独特的语言系统，是如何起作用的？
        1. 第一层功能是用语言灵活的传达信息，这可以帮助我们形成一个小团体，团体内部大家彼此熟悉互帮互助，但一旦这个团体过大，这个团体内部的社交秩序就会因为大家之间相互无法通过语言而变得亲密从而崩坏，使得团体分裂；
        2. 此时第二层功能：用语言进行相互沟通的作用就显现出来了，这个大团体内部开始相互讨论彼此，语言此时充当八卦的功能，于是这个团体可以变得更大，但借由八卦来维持的最大“自然”团体大约只能是150人("150定律"一直到现代人类社会依旧起作用)；
        3. 此时第三次功能对智人能够战胜所有其他人种起到了最关键的作用，那就是智人可以用这种语言构造虚构的故事，以使得大批互不相识的人，只要同样相信这个故事，就能共同合作。(例如：神明，民族，国家，人权，甚至是有限公司等等，这些全都是人所虚构出来的大批人所共同相信的故事，正是有了这些故事，大批互不相识的人就可以彼此信任彼此合作，形成一个更大的团体)。

        + 人之所以为人，一个很重要的原因就是人类拥有“想象”的能力, 智人能够传达关于虚构概念的信息。人类和黑猩猩之间真正不同的地方就在于那些虚构的故事，它像胶水一样把千千万万的个人、家庭和群体结合在一起！这种胶水，甚至让我们成为了万物的主宰。

    • + - 3. 亚当和夏娃的一天

      • + 本章试图带领读者从时间的帘幕后面探头偷看一眼，看看从认知革命到农业革命这之间的6万年左右时间内人类是怎么生活的。想要了解人类的天性，历史和心理，我们就得看看我们的这些狩猎采集的祖先们头脑里是怎么想的。

        + 为什么我们总是一碰到最甜最油的事物就难以抵抗？
        从演化心理学的角度来看，我们的大脑和心灵还是以狩猎和采集的生活方式在思维。当时他们住在草原上或森林里，高热量的甜食非常罕见，永远供不应求，所以一旦碰到，最明智的做法就是立刻吃到吃不下为止，否则就会被别的生物给抢掉。

        + 初窥智人部落狩猎采集的生活方式
        他们不断追逐着食物从一地迁移到另一地，但他们的生活似乎比农业革命之后的人类来的更舒适也更有意义。他们拥有健康和多样化的饮食，拥有相对较短的工作时间，也少有传染病的发生，因此很多专家把这种社会定义为“最初的富裕社会”！
        但这种生活方式所形成的社会也有自己的弊端，他们的世界里残酷无情，常常物资匮乏，同时还有儿童死亡率很高等等问题。

        + 总结
        讲到远古的采集生活，无论是精神和心理生活，还是他们的社会政治世界等，我们都难以重建，这幅沉默的帷幕就这样的罩住了这几万年的历史。这帷幕是如此的厚重，我们连很多事情是否曾经发生都难以断定，更不要想着去详细描述。

    • + - 4. 毁天灭地的人类洪水

      • 整个动物界从古至今，最重要的也最具破坏性的力量，就是这群四处游荡、讲着故事的智人。

         

        + 人类的每一次扩展，都会伴随着一波灭绝浪潮！
        在认知革命之后，智人而得到新的技术、组织能力甚至是眼界，能够走出亚非大陆前往外面的世界，这些采集者也开始扩张起来，从亚非大陆到澳大利亚再到美洲，智人看起来就是一个生态的连环杀手！大量的物种以及巨型动物都惨遭灭绝！可以说智人的第一波殖民正是整个动物界最大也是最快速的一场生态浩劫。

        + 那么这些智人究竟是怎么搞出这场生态浩劫的呢？
        1. 狩猎：大型动物本身繁殖就十分缓慢，伴随人类的猎杀，它们必然面临低出生率高死亡率的魔咒；
        2. 火耕：这些智人掌握着火耕技术，他们会可以烧毁那些阻碍到他们的灌木丛和森林，从而改变了这些物种所赖以生存的生态环境；
        3. 气候：气候变迁也是这些物种灭绝的一个因素，伴随智人的扩张脚步变得愈发严重。

        + 总结
        每一次的人类扩张，都伴随着大量物种的灭绝浪潮。第一波是由于采集者的扩张，第二波是由于农业的扩张，而第三波灭绝浪潮则是由于现代人类所发起的工业活动的扩张。人类可说是坐上了生物学有史以来最致命物种的宝座！

     

    第二部分 农业革命 (约1.2万年前)

    • + - 5. 史上最大骗局

      • 人类有长达250万年的时间靠采集和狩猎为生，但在大约1万年前，人类的生活方式发生了质的改变：由采集走向了农业！这场改变的缘由就是：农业革命！

         

        + 农业革命可说是史上最大的一桩骗局！
        人类以为是自己驯化了植物，但其实是植物驯化了智人！这场阴谋的背后主谋就是那些原本在自然界中只占极少数的植物物种(比如：小麦，稻米和马铃薯等)！

        + 从小麦的观点来看，农业革命把它从一种野草变成了传遍世界各地的植物。从物种生存和繁衍的角度来看，小麦可说是史上最成功的植物！
        小麦取得成功的秘诀就在于能操纵智人为其所用！农业革命之后人类只能被迫从采集者转变成永久定居在麦田旁边的农民！

        + 智人为什么会被它们所操纵？
        分析之后，你会发现农业革命并没有给人类带来更多的什么好处。它只是使得每单位土地能够提供更多的食物，从而使得智人的数量呈现指数级增长！但智人这一物种数量的壮大，对于该物种内的每一个人而言其实活的更加不幸福了。农业革命的本质就是：“让更多的人以更糟的状况生活下去”！

        + 什么是所谓的“奢侈生活的陷阱”？
        原本的奢侈品往往会成为必需品，而且带来新的义务！农业革命就是这种陷阱的一个例子：种麦子原本看来是一个划算的选择，它最初貌似可以给我们提供更多的食物，但它最后会演变成为一个不可逆转的越来越沉重的负担！一连串为了让生活更加轻松的“进步”，最后却像是在这些农民的身上加了一道又一道沉重的枷锁！

        + 这场革命的最大受害者：那些被驯化后的家畜
        为了把牛羊马驴等训练成为听话的役用动物，人类打破了它们原本的天性，强迫它们与社会进行连接。

        + 我们从农业革命中能学到的最重要的一课很可能就是：物种演化上的成功并不代表着个体的幸福！这不仅适用于那些驯化物种，而且也适用于人类本身！
        每当人类整体的能力大幅增加，看似大获成功的时候，个人的痛苦也总是在随之增长！

    • + - 6. 盖起金字塔

      • 本章重点想谈论一下：“想象” 在人类经历农业革命之后对人类社会的构建所再一次发挥的极大作用。我们的整个人类社会，都依赖于“由想象所构建的秩序”来进行合作，从而得到质的发展！

         

        + 认知革命后智人所发展出来的“想象”的能力，对智人之所以能发展成为今天这副主宰万物的模样所起到的两次重大贡献：

        • 1. 利用虚构的故事，突破人际交往之间的“150定律”，形成更大团体之间的协作，从而得以战胜人类其他各种人种，取得自己在人类的完全垄断地位；
          2. 利用想象的秩序，为人类提升了合作效率，从而可以形成更大的团体，甚至是达到几亿人口级别的社会，同时这种秩序也能够帮助我们创造一个稳定繁荣的社会。

        + 农业时代的来临，给人类所带来的主要变化：

        • 1. 人类由采集者，转变为定居的农民 (生活方式的转变)；
          2. 人口少，转变为人口数量的剧增 (人口的增多)；
          3. 人类生活 空间的缩小以及时间的变长！
          (人类第一次对“未来”这个概念变得如此在意，人类此刻不仅时刻都在想着未来，而且几乎可以说是为了未来而服务。人类开始变得“担心未来”，从而不断努力工作)
          @ 联想：即使是到了现在，人们也还是在不停的为“担心未来”而不敢停歇脚步，不停的在努力向前奔着！

        + 由想象所构建的秩序

        • - 为什么人类史上有如此多的诸如战争的灾难？

          • 其根源在于“人类其实还未发展处能够进行大规模合作的本能”！所以一旦人群大了，就会有诸多事项无法达成合作协议，从反引发冲突不断！

          - 有哪些著名的虚构故事？

          • 1. 公元前1776年的《汉谟拉比法典》；
            2. 公元1776年的美国《独立宣言》；
            3. 神，民族，国家，法律，金钱，公司等等诸如此类。

          - 对想象的秩序(或称为想象的现实) 的经典解读：

          • 1.《汉谟拉比法典》vs.《美国独立宣言》，一个强调严格的阶级制度，一个强调人人生而平等，到底孰对孰错？其实，它们都错了，因为它们本质上都是一种想象的现实，想象着自己所说的就是这个世界放之四海而皆准的真理。

            2. 美国《独立宣言》中所强调的“平等”的观念，其实是源自于基督教，是基于“每个人的灵魂都是由上帝创造，而所有灵魂在上帝面前一律平等”这一套基于“上帝、创造与灵魂”的神话故事！
            所以如果从生物学抑或是科学的角度来解读《独立宣言》，你就会发现原有的独立宣言就是一种想象出来的秩序。

          - 想象的秩序不像自然界的客观秩序，它本身也需要维持，那么它是如何维持的呢？

          • 1. 该秩序要拥有 真正的坚信者，这些信徒是维持该秩序的核心力量；
            2. 对于违反该秩序的人，必须要掺入 暴力和胁迫的成分来迫使这些人维持秩序。

          - 这些 想象的秩序 如何才能做到让人相信？

          • 1. 对外一定要坚持自己千真万确，绝非虚构；
            2. 在教育上一定也要彻底贯彻同一套原则！

            举例：有三大原因让人类不会发现组织自己生活的种种秩序其实是虚构的：
            1. 想象构建的秩序深深的与真实世界相结合；
            2. 想象构建的秩序塑造了我们的欲望；
            3. 想象构建的秩序存在于人和人各个主体之间思想的连接上。
            (最后一个原因使得任何一个独立的个体即使知道这个秩序是虚构的，也都无力撼动这个秩序！因为就算使我自己下定决心不再相信美元，人权和美国，但它们仍旧存在于数十亿人的共同想象之中，我个人无法造成任何改版！)

        + 总结：

        • 身为人类，我们不可能脱离想象所构建的秩序！每一次我们以为自己打破了监狱的高墙，迈向了自由的前方，其实只是到了另一间更大的监狱，我们只是把活动范围稍稍的加以扩大。

    • + - 7. 记忆过载

      • 本章讲述农业革命后，整个人类的协作网络变得越来越复杂，人类已经无法仅凭大脑就能处理由此而产生的大量的信息(包括合作的规则，数字等)，史上第一次出现了“记忆过载”现象。文字符号及使用它们的方法应运而生！

         

        + 为什么不能依赖人类的大脑来记忆信息？

        • 1. 人类大脑的容量有限；
          2. 人总有一死，大脑因此也会随之死亡，而信息的传递如果仅依赖口口相传，可想而知传几次之后原本的信息就可能变得面目全非；
          3. 人类的大脑经过演化，只习惯存储和处理特点类型的信息(例如人类大脑就不习惯处理数字，这可能就是因为数字只是自农业革命以后的近几千年来才渐渐为人类所发明和使用，而在之前的几百万年里，人类历史上从未有过训练大脑对数字的处理能力)。

        + 部分表象和完整表象的文字

        • 部分表象是指这套系统只能呈现特定种类的信息，它局限于特定领域的活动；而完整表象则是它能够大致完整表达出人类的口头语言。

        + 发明文字的必然结果就是还需要发明能管理这些文字的技巧！

        • 我们还需要组织工具，快速的复制工具，较为快速准确的检索工具等等。同时还需要够聪明的负责人员，能够了解这些工具的使用方法！而这些工具的发明往往比发明文字本身会难上很多！

      + - 10. 金钱的味道

          解决这种局限性的两种方案：1. 建立起集中的以物易物系统，统一收集产品统一分配产品(前苏联和中国都对此做出过尝试-公有制)，后来实践证明大家的生产积极性受到极大打击，社会根本难以维持；2. 发明“钱”的概念，在大家交换各自所需时加入一个中间的媒介物，完美的解决了这一难题(私有制)。

        + 宗教的发展

        • - 泛神论阶段：远古采集者信仰的是泛神论，他们认为人类只是地球上众多生物的一种，动植物与人类一样都不过是自然界中的普通生物；
          - 多神教阶段：农业革命对宗教的这次革命性转变起到了至关重要的作用！正因为有了农业革命，大量的人得以聚集起来，宗教革命也伴随而来！农业革命最初的宗教意义就是：让动植物从与人类平等的生物，变成了人类的所有物。由此“神”这个角色就变得格外重要起来，多神教信仰应运而生！
          对比：对泛神论者来说，人类只是地球上众多生物的一种，但对多神教徒来说，整个世界就像是反映了神和人类的关系！多神教不仅提高了神的地位，也提高了人在自然界中的地位！
          - 一神教阶段：随着时间的流逝，某些多神论者开始对自己所信仰的某位神灵越来越虔诚，也慢慢远离了基本的多神论概念，开始相信只有那位神灵是唯一的神，一神教由此得以发展！
          对比：多神论与一神论的不同之处在于，多神论认为主宰世界的最高权力不带有任何私心和偏见。

        • - 其实不然，每次人类即将因为能源或原料短缺而使经济增长趋缓的时候，就会有资金投入到科学研究，以获得新知，从而另辟蹊径解决短缺问题。
          - 而且这种做法屡屡奏效，它有时候可以让我们更有效地利用现有资源，有时则更直接的为我们找出了新能源和新原料以供我们使用。

          - 工业革命就是我们人类发现新知并运用新知，从而极大帮助人类解决能源和原料问题的范例！

        + 工业革命的本质

        • - 工业革命实质上就是找到了一种新方法来进行能量转换和商品生产！工业革命的核心其实也就是能源转换的革命！
          -  在工业革命之前，能量转换主要靠人力和畜力，我们将植物从太阳处所取得的能量转换成我们肌肉的力量，再依靠肌肉的力量，将原始的能量转换成机械能，动能，热能等等。
          -  在工业革命之后，几大重要的发明：蒸汽机，内燃机，电力等无疑都在尝试怎样能把大自然中的各种能量进行直接转换，不需要再经过人力，从而解放自己！

          - 人类深深的着迷于如何使用机器和引擎来转换各种能量！

        + 工业革命为人类带来了前所未有的种种能源和原料！但是其实它本质上可说是人类历史上“第二次农业革命”。
        因为工业革命之后，农业的支柱也变为了工业化生产。工业化的畜牧业以及机械化的种植业，成为了现代社会经济的基础。

        + 工业的巨轮所带来的负面影响

        • 1. 与第一次农业革命一样，那些被驯化的家畜承受着更加悲惨的命运，它们现在可能由于人类的冷漠而变成了人类眼中的“活的生产机器”；
          2. 工业革命将很多人力从农业生产中释放出来，虽然生产和经济得到了很大的发展，但是这一套游戏规则一旦玩起来就停不下来，生产的多了必须就提倡大家多消费，“消费主义”应运而生。
          “消费主义”向之前的“资本主义”一样也提出了一种新的道德观：“放纵对你有益而节俭就是自我压抑”，这种新道德观已然不再以“节俭”为美德！基于这种新道德观究竟会给人类带来什么样的后果，我们现在还不得而知。

       

      + - 18. 一场永远的革命

      • 工业革命找出了新方法来进行能量转换和商品生产，于是人类对于周遭生态系统的依赖大减，人类社会由此而带来巨大的变化。

         

        + 工业革命为人类带来了时刻表和生产线的概念。从工业革命以后，人类社会相比之前最大的变化之一就是：采用了工业化的时间概念！

        + 工业革命带来的其他重大社会变化有：都市化、农民阶级的消失、工业无产阶级的兴起、对自由化，民主化的这种自由主义的追捧、父权社会的渐渐解体等

        + 工业革命带来有史以来人类最大的一次社会革命：家庭和地方社群的崩溃！这种传统的“家庭/社群”形式的社会正在被一种叫做“国家/市场”形式的社会所取代！

        • - “国家/市场”形式的社会中的两大主要想象社群：“民族”和“消费大众”。

         

        + 工业革命也让我们现在所处的这个时代进入到一种真正的“和平”！

        • - 核武器的研发成功，使得大国之间不再可能真正开战，因为这无异于等于集体自杀，现代战争的成本因此也变得极高。现代社会正是在这种核武器的威慑下变得和平。

          - 工业革命使得财富的形式发生了变化：从传统的土地、牛、奴隶和黄金，正在转变为人力资本、科技知识以及经济结构！
          这种财富通过武力已然很难占有，而通过和平的方式反而能获得更多的红利。

          - 总结：这种“和平”是有四大因素所共同保证的：
          1. 核子末日的威胁 促进了 和平主义；
          2. 和平主义 大行其道，于是战争退散贸易兴旺；
          3. 贸易的兴旺，也让和平的利润更高，战争的成本也变得更高；
          4. 国际网络变得日渐紧密，使得多数国家无法再维持全然独立，所以其中任何一国片面宣战的机会大幅降低！全球帝国正在形成，这个帝国会努力维持其疆域内的和平！

       

      + - 19. 从此过着幸福快乐的日子

      • 过去500年间，人类经历了四次科学革命，三次工业革命，整个地球因此也在生态和历史上逐渐整合为一个单一领域！人类社会的经济，政治，日常生活和心理都有了极大的改观，但人类在这整个过程中真的变得更加快乐了吗？

         

        + 过去的几百年内，智人虽然取得了空前的成就，但是代价却是赔上几乎所有其他动物的命运，数百亿的动物在遭受现代工业制度的剥削！另外随着人类社会工业化的发展，环境问题日趋严重，地球上的各个物种都在受其影响。所以我们要评估全球幸福程度，不应该之着眼与人类，而应该做到真正的放眼全球！

        + 快乐到底如何计算？

        • 1. 社会科学的研究方法：发问卷调查主观幸福感，运用统计学分析方法将其结果与社会因素相结合，从而得出的研究结果是人的快乐与哪些社会因素相关！[外部因素]
          2. 生物科学的研究方法：直接去检测人的神经，神经元、突触和各种生化物质，发现人能快乐的原因，不过是因为血液中开始流过各种激素，脑中开始闪现各种小小的电流！[内部因素]
          3. 宗教看待快乐的方式：最看重快乐这个问题的宗教就是佛教，但佛教教导我们得到快乐的根本因素在于：“放下我执”，让我们不要太看重自己的感受，它强调这些主观感受都只不过是一瞬间的波动，都是“空象”，当我们真正放下这些东西的时候，我们就达到“涅槃重生”之境，获得永恒的快乐。[非外非内]

        + 如何得到快乐的几种观点：

        • 1. 快乐其实并不完全取决于外部客观条件，而在于客观条件与主观期望之间是否相符！要懂得“知足常乐”！

          2. 快乐这种东西从生物演化的角度来看向来都是被当做奖赏！凡是有利于生存和繁衍的东西都会在演化的过程中被保留下来，比如为了繁衍生物之间交配会有快感，为了生存生物交配完了之后这种快感会很快消退。

          3. 总结：对于快乐，我们去在意究竟是否与期望被满足，抑或是否主观感受到快乐，其实都没有意义，真正重要的问题在于：人类是否真的更了解自己！
