All + All -

富爸爸穷爸爸
作者：[美]罗伯特·T·清崎

  • + 前言：

    • -《富爸爸穷爸爸》的愿景：给任何有兴趣提高自己的 财商 和 改善财务状况 的人提供一个很好的教育工具！
      - 富人购买资产，而穷人把负债当做资产买入！
      - “富爸爸”观念：不要做金钱的奴隶，而要让金钱为我们而工作。
      - “金钱并非真正的资产，它只是代表着一种大家都认同的契约关系，一些人仅凭思想和所谓的合约就能从一无所有变得一夜暴富”

       

      + 上天赐予我们每个人两样伟大的礼物：思想 和 时间。
      · 这两样都是无价之宝，关键就是看我们如何能利用这两大宝贝创造出我们自认为价值最大的事情；

      · 你要学会利用这两种珍贵的礼物去做你愿意做的事情！

       

      + 你赚取的每一元钱，你都有权选择它的去向，而你的选择又将决定你的人生方向：

      • 1. 愚蠢的用掉它，你就选择了贫困；
        2.  把它用在负债项目上(还贷款)，你就选择了进入中产阶级；
        3.  把它投资与自己的大脑，学习如何获取更多的资产，那么你就会变成为富人(财富将成为你的未来)。

       

      + 罗伯特讲述 “老鼠赛跑”  的故事，简直就是我从出生到现在(深造读研)一路历程的总结，以及对我将来要走的路的一个预言。
      @思考：由于这个故事对我的前20年的回顾都对，如果我现在不作出改变，那么我相信无疑这个故事对我的后半生的预言也将是正确的！

       

      + 如何跳出 “老鼠赛跑” 的陷阱？

      • - “老鼠赛跑” 的陷阱还可以这样表述：
        · 先说恐惧感再说贪婪和欲望：没钱的恐惧会刺激我们努力工作，当我们得到报酬时，贪婪和欲望又开始让我们去想所有钱能买到的东西，于是我们就陷入了一种模式；
        · “起床，上班，付账，再起床，再上班，再付账... ...”  这种模式下的生活就是在无穷无尽的为这两种感觉而奔忙：恐惧 和 贪婪。
        · 给他们再多的钱，他们也只会以更高的开支重复这种循环，这就是所谓的“老鼠赛跑”！(也可说是‘恐惧和贪婪’的陷阱)

         

        - 理论上的解决方法是：证明你在 会计、投资 上是在行的，要知道在这两个困难的领域成为高手是一件多么困难的事情。
        - 实际的解决方法就是：在继续上班的同时一定要关注自己的事业，学会利用知识分清资产和负债，然后购置资产，让我们的钱为我们而工作。

       

      - 富爸爸 vs. 穷爸爸

      • · 穷爸爸建议他为企业而工作，富爸爸则建议他拥有自己的企业；
        · 穷爸爸鼓励他成为聪明人，富爸爸则鼓励他去雇用聪明人。

       

      - 人们在财务困境中挣扎的主要原因是：他们在学校里学习多年，却没有学到任何关于金钱方面的知识，人们只知道为金钱工作，但从来不学着让金钱为自己工作。于是大家就陷入到了“老鼠赛跑”的困境中了。
      @联系:《小狗钱钱》要学会渐渐的养一只会下金蛋的鹅，让我们的钱也能为我们工作。

     

    + 读后感 本书给我留下深刻印象的几个观点：

    • 1. “老鼠赛跑陷阱 or 恐惧贪婪陷阱”，大部分人都因不懂财务知识而身陷其中，苦于无法摆脱自己的财务困境。

      2. “富人的逻辑”：富人不为钱而工作，他们倾向于学到东西，通过不断地学习和思考，让钱为他们而工作。

      3. “作者的致富哲学”：把“种子”播种在我的资产项下！从小额开始播种，有些长成了，有些则没有！
      · 致富规则1：弄清资产和负债之间的差别，然后尽可能多的购置资产；
      · 致富规则2：关注自己的事业并继续你每天的工作！

      4. “知识就是力量 and 书中自有黄金屋”：运用你的财商，运用你的知识智慧以及对游戏的喜爱来降低所谓的“风险”吧！好机会都是用脑子而不是用眼睛所看到的！
      · 要想快速建立起自己的财富大厦，切记：一定要先打牢大厦的地基(也即要多学习财务知识)！

      5. 什么是财富？怎么定义财务自由？怎么看待风险？怎么看待奢侈品？什么是赌博？

     


    + - 1. 富爸爸，穷爸爸

    • - 本章讲述作者的两个爸爸给他提供的两种截然不同的金钱观的选项，以及作者是如何选择听从哪个选项的。

       

      + 当自己面临几种截然不同的路可走时，到底该如何做选择？

      • - 这两个观点截然不同的爸爸迫使我对每一个有分歧的问题进行思考，并最终形成自己的想法。这种自己去思考和选取而不是简单地全盘接受或全盘否定的过程，在后来的漫长岁月中被证明对我是非常有益的。

        - 选择不同，命运也是不同的。一条看似艰难的路其实会越走越顺，而一条看似很顺的路可能就会越走越艰难。
        @联系:《解忧杂货店》人在每个自己人生中的岔路口，都会遇到选择问题，但无论你选择了哪一条路，切记一点: 只要自己能“认真地对待生活，认真地去生活”，那么你的人生都会同样精彩，那些烦恼和忧虑也就会自然而然的得到解决。

       

      - “脑袋越用越活，脑袋越活，挣钱也就越多”，遇到一个需要花大钱的事，一句“我负担不起”这类的话其实说白了就是一种精神上的懒惰，是不愿去开动脑筋，取而代之一句“我怎样才能付得起呢”则是一个疑问句，它有助于帮助我们开动大脑，想方法解决问题。
      - “经常性的头脑运动可以增加你获得财富的机会” -- 富爸爸。

       

      - 穷爸爸努力存钱，富爸爸不断投资；
      - 穷爸爸说：挣钱的时候要小心，别去冒险，而富爸爸则说：我们要学会管理风险。

     

    + - 2. 第一课：富人不为钱工作

    • + 穷人和中产阶级为钱而工作，但富人让钱为他们工作。

       

      - “9岁时，你就有了为钱而工作的体验，只需把上个月重复50年，你就知道大多数人是如何度过自己的一生了”。
      - “学习为钱而工作很容易，但学习让钱我们工作则显得很困难。”
      - “正是出于恐惧心，人们大多害怕失去工作，害怕付不起账单，害怕遭到火灾，害怕没有足够的钱，害怕挨饿，大多数人期望得到一份稳定的工作。为了寻求稳定，他们回去学习某种专业，或做生意，拼命为钱而工作，大多数人也因此变成了钱的奴隶，然后把怒气对准他们的老板。”
      @思考：思考一下我的前20年，我在学校勤工助学以及大三暑期华为实习的经历都是在“为钱而工作”，所以我会觉得这些工作无聊无趣而且没意思，于是我不向往去工作，最终导致现在2016年的我依旧待在大学的象牙塔内！我真的该考虑自己为什么会这样了！希望这本书能给我一个解答。

       

      + 如何跳脱这种由无知所带来的“恐惧和贪婪”的陷阱？

      • - 首先你要 “学会让感情跟随你的思想，而不是让你的思想去跟随你的感情”。
        @联系：这句话等价的描述可以是：1. 要学会用大脑去控制身体，2. 要学会按自己所想地去生活，而不是按自己的生活去思考。

       

      + 生活的艺术，生活是什么？

      • - 在生活中，人们通常会做两件事：
        1. 一些人在生活推着他转的同时，抓住了生活赐予的每个机会；
        2. 另一些人则听任生活的摆布，不去与生活抗争，只会抱怨生活的不公平，其实他们不知道生活其实也赐予了他们机会。

        - 大多数人认为这个世界上除了自己外，其他人都应该改变。其实我们更需要记住的一句话是：改变自己比改变他人更容易。

       

      - 生活中的场景：

      • · 大人们总是急急忙忙的去工作，这事看起来真是无趣，而且他们看上去也不快乐，但好像总是有些什么东西使得他们不断地去急着工作。
        @分析：这种现象最好的诠释了什么是“为钱而工作”，这里的不知是什么东西的东西，其实是：那些大人们对未来的恐惧  以及 对消费的欲望。(说白了这些大人们还是生活在了 “恐惧和贪婪”的陷阱(老鼠赛跑) 中。)
        · 人们总是为了实现愿望而最终变成为钱而工作。

       

      - 怎么样才能使自己不会变成金钱的奴隶？

      • - 简单地的说就是：要学会支配钱，而不是害怕钱 (要学会思考)。
        - 要学会用合适的方式去利用自己的“恐惧和欲望”，别让恐惧和欲望控制了你，让你自己落入陷阱。

       

      + 学会看见别人看不见的东西！

      • - 其实我们的生活中遍布着很多机会，但是由于我们没有明确的目标，没有去刻意寻找，所以这些机会大多时候就这样悄悄的流失过去了。
        - 无代价的工作，让罗伯特他们很快发现了能挣钱的方法，他们开始看见了那些平常他们所看不见的东西。
        - 由于没有从商店的工作中得到报酬，所以他们不得不发挥自己的想象力，去寻找赚钱的机会。

     

    + - 3. 第二课：为什么要教授财务知识

    • - 因为知识才能解决问题并创造财富，不是凭知识挣来的钱很快就会消失。

       

      + 什么是财务自由？

      • - 除非发生意想不到的大事，否则我们完全可以选择工作也可以选择不工作，并且我们的财富能避开通货膨胀而且还在不断地增加！
        - 当你的资产已经多到可以自我增值，不再需要你精心的照顾时，你就能开始享受它的树荫了！

       

      + 致富规则 你必须明白资产和负债之间的区别，并且尽可能地去购置资产，而不是负债！(这是富人致富的唯一法则)

      • - 简单地说：
        · 什么是资产？资产就是能把钱放进你的口袋里的东西！
        · 什么是负债？负债就是把你的钱从你的口袋里取走的东西！

        - 如果你想变富，只需在一生中不断地买入资产就行了！

      - 我们应该思考一些问题：

      • · 挣了钱之后该怎么办？· 怎样防止别人从你手中拿走钱？
        · 你能多长时间的拥有这些钱？· 你如何能让你的钱为你而工作？

       

      + 发财梦变成噩梦的故事：(没有认真考虑上述四个问题导致的)，这很可能也是大多数中产阶级家庭的故事

      • - 努力学习成为聪明人，然后找到好工作，然后结婚。
        · 一对刚结婚受过高等教育的新婚夫妇租住在一套拥挤的公寓里，然后开始省钱，梦想有一套自己的大房子，然后就贷款买了，紧接着又有了自己的孩子！
        · 有了孩子之后两个人开始专心于事业，收入开始增加，但同时支出也跟着增加(比如：税，买房子，买车，买新家具，生孩子这一系列过程所带来的越来越多的贷款债务)！
        · 渐渐地，他们发现自己已经身陷贷款和信用卡贷款的债务中，他们已然落入了“老鼠赛跑”的陷阱中去了。他们必须开始更认真而又努力的工作，钱挣得多但永远也不够用！这种“恐惧与贪婪”的情感突破了理智，使得自己永远无法逃离陷阱。

        - 为什么会出现这种问题？又能怎么解决呢？
        @原因分析：他们真正的问题在于他们所选择的支出方式存在问题，这就是他们苦苦挣扎无所托的真正原因！
        · 这种支出方式：就是在不停的购置负债，而没有将自己收入中的一部分用于购置资产，所以他们始终发现自己身陷“老鼠赛跑”的陷阱之中！
        @解决方法：联系《小狗钱钱》

        1. 尽可能少的偿还贷款；

        2. 要学会先支付自己，然后再按比例支付其他。

        · 最好每个月先定一个比例，把自己收入的一部分存起来，用于购置资产！要把自己的“鹅”养大。用本书的话来说就是：要花精力关注自己的事业！

       

      + 为什么不要急于贷款买房子？

      • · 首先房子车子这些都是属于 负债项，它们会把我们的收入从我们的口袋中拿走；
        · 其次如果你贷了很多钱，它给你带来的最大损失其实是机会损失！如果你把钱投进了房子里，那么你就不得不努力工作，而且你的现金将会不断地从支出项流出，而不是流入资产项。
        · 你会因此而失去其他投资增值的时机、失去受财务知识教育的机会、无法获得投资经验 (而这些是富人之所以 越来越富 的根本原因)。
        · 早年缺乏必要的财务知识教育 以及 收入来源只有工资，生活完全依赖于自己的雇主，这就是：中产阶级为什么无法摆脱财务问题的根本原因。
        @思考：要想摆脱财务问题，甚至是早日实现财务自由，只靠工资是肯定不可能的，还需要关注自己的事业，让自己的钱学会生钱(让自己的钱为自己打工)！

       

      + 走上致富之路的好办法 买入能带来收入的资产，让资产所产生的现金流抵消开支，然后当钱开始变得越来越多的时候，你就可以考虑进行一些投机性的投资了！
      @注意：这所谓的“投机”，或者说“高风险”，该怎么看待？其实这只不是因为你缺乏某些很重要的财务知识，而不知道究竟应该怎样去看待这些投资机会！只要你能拥有足够的财务知识，你就不必害怕这所谓的“冒险”！

       

      + 到底什么是“财富”？财富的定义！

      • - 财富就是支持一个人生存多长时间的能力，或者说如果我今天停止工作了，我还能活多久！
        - 根据这个定义，财富的单位应该是时间 (单位为：年月日)，而不是货币单位。
        - 所以当 你的财富值 大约 你的寿命  时，那么你就实现财务自由了！

     

    + - 4. 第三课：关注自己的事业

    •  

      + 致富规则 关注自己的事业(让自己的钱为自己工作)，让你在停止工作后依旧能取得收入，也即可以积累财富！
      @联系：这里所指的“自己的事业”，狭义上指的是创业开公司，但广义上却指代一切利用钱生钱的事业！也即：积累我们的资产项，也就等于是在做自己的事业！

       

      + 为了财务安全，为了早日实现财务自由，我们需要去关注自己的事业！
      (我们的事业，指的是：我们的资产，而不是我们的收入)
      + 只有你把增加的收入用于购买可产生收入的资产时，你才能获得真正财务上的安全。

       

      + 为什么大多数人都是“财务保守”的？

      • - 因为他们对财务知识的匮乏，导致他们不敢承担风险！
        @解读：缺乏 “财务知识”，对风险的理解不正确，从而导致这些人不敢去承担风险，也就变得财务保守起来了！
        （我在没看《小狗钱钱》之前对财务知识也是一片空白，对风险的理解也有误，始终担心害怕风险，所以一直不敢去投资股票和基金，现在回首看一下真的发现浪费了好多赚钱的机会啊！）
        （我现在所做出的改变，无疑也是我发誓要养成 看书-读书 的习惯，之后带给我的意外收获！所以说 “书中自有黄金屋” “知识就是力量” 这些话不是没有道理啊！）

       

      + 走上致富之路的好办法  关注自己的事业并继续你每天的工作！拿着正常工作的工作收入(从事别人的事业取得一份收入)，首先确定一个比例来购入资产项(利用这些钱来做自己的事业)，然后再想着去还贷款之类的其他负债项！这样就可以保证积累到财富啦~~

       

      - 如果你仅仅依赖雇主的工资作为自己的唯一收入，那么你永远都只能是一头乖乖待挤的奶牛！你也就是别人实现自己事业路上的一块最普通的垫脚石而已。(所以切记：一定要学会关注自己的事业！)

      - 有自己事业的好处：不久我买了一辆保时捷，我的同事都以为我是用自己的工资买的，可事实上，我正在不断地把工资投资于我的资产项，而用资产项为我生产出来的钱购买我想要的东西！
      · 我的钱为我挣回了更多的钱，在我的资产项中，每一块钱都是我的雇员！

       

      + 什么样的东西可归入真正的资产呢？

      • · 不需我到场就可以正常运作的业务 (我拥有他们)；
        · 股票、债券、基金；
        · 产生收入的房地产；
        · 借据，票据；
        · 专利权，如音乐、手稿、专利等；
        · 任何其他有价值、可产生收入或可能增值的能很好流通的东西。

        · 大部分人之所以穷，是因为在他们想要投资的时候，周围到处都是跑来跑去的“小鸡”，叫嚷着“天要塌了，天要塌了”。
        · 我们常常需要极大的勇气，不让那些谣言和杞人忧天式的怀疑加剧我们的恐惧心理和对自己的疑虑！

        3. 懒惰

        (他们常常把忙碌的工作当作自己逃避现实问题的途径)

        · 要坚信人的精神能够战胜一切，你要知道你能做成任何事情！

        (好消息是：习惯是可以养成的，而且只需短短的21天，这样算下来一年就可以轻轻松松的养成12个习惯)

        · 如果你知道自己在某一方面欠缺知识，不要试图掩饰，因为那只是在欺骗你自己，你真正应该做的是去找一位这一领域的专家或者找一本有关这一问题的书，马上开始教育自己！
