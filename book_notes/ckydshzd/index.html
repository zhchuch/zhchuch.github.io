蔡康永的说话之道

作者：蔡康永

  • - 本书强调的观点是："透过说话，懂得把别人放在心上"，这就是所谓的说话之道！

    - 本书的一大特色：“配的漫画 很切题 且 很幽默搞笑”！

     

    + 总结：

    • 本书看完之后给我留下印象最深的几个观点，或者说几条建议是：

      · 1. 与自己关心的人之间(包括亲人、情人等)表达感情，切记不要只会说而是要学会“用心”，要通过实际行动让别人感受到你的心意！

      (举例：情人节礼物的选择，亲人同事间如何懂得调整自己的心态去凑趣，KTV内如何提升相互之间的关系等等)

      · 2. 怎样让自己变成一个有聊的人？切记：你问的越具体别人回答就越省力！

      (先从“是非题/选择题”之类具体问题开始摸清别人感兴趣的话题，再用“问答题”的方式深入聊天！另外为了让话题可以持续，你要学会“问问题留有退路”)

      · 3. 怎么让自己变成一个上道的人？

      (学会去倾听，尽量不要让自己说出“我”这个字)

      · 4. 一定要学会赞赏别人，不要吝惜赞美之词

      (当别人赞美你的时候，也要学会回一句赞美)

     

    + - 01-10

    • - 转述别人的赞美，也不用刻意去捏造这类的赞美！
      (有时候，转述别人的赞美可能比你直接去赞美更加合适~)
      - 因为每个人都想要得到别人的赞美~

       

      - 有时候我们说什么话就代表了我们是一个什么样的人！(You are what you say.)

      - 切记：少给别人找麻烦的方法，就是把麻烦在自己手上解决掉！
      (你如果不喜欢别人做的，你有新想法，那么你就去做，不要再麻烦别人去做)
      - 与人打交道切记：别人并不是为了伺候你而存在的！

       

      + “传说··· 大家一起沉默时，是天使降临··· 的时候···”
      @学习：可以学一下，以后再面临沉默时，这句话可以做为打破沉默的开场白~
      - 沉默本身也很正常，有时候也不必刻意营造话题去打破沉默，而是也可以“学会享受正在做的事情”本身！

       

      - 把无谓的胜利让给对方，要学会懂得认输。
      + 智者说“每个人，都是自己那片小领土的国王”！
      - 每个人都喜欢别人同意自己！只要认清自己的立场，在认个输不会伤害到自己的原则的时候，就一笑置之吧！
      @联系:《重新定义公司》里面说道“Google在达成共识”时的经验，认同和赞美别人也是一个很重要的经验~

       

      - 每个人都希望自己被重视！
      而被重视这个关键词本身也有一个关键字，那就是“视”！

       

      + 请学会讲那些“不会 GAME OVER，让人能接得下去的话”~
      记住：不管对方说的在你心目中是多么无聊的事情，请记住给别人留有余地。要学会巧妙地引导你们之间的谈话到你们都感兴趣的话题上去。
      - 不要试图和女生秀智商，有的时候你的智商不一定有她高！
      因此：当一个女生跟你说这个东西很难懂很难理解的时候，通常就是表示对这个话题不太感兴趣，你要试着转到其他话题，而不要试图继续跟她谈这个东西，还试图想把她说明白~

       

      + 问的问题越具体，回答的人就越省力！
      · 别人发现回答你的问题很省力，那么别人就会愿意且有力气和你聊下去~
      · 建议发问：先问“是非题”或“选择题”这种具体问题，而不要试图直接用“问答题”这种需要费太多脑筋的问题~ 等到摸清楚了对方感兴趣或有聊的话题时，再采用“问答题”或“申论题”这种方式深入谈话~
      + 同时为了让话题可以持续下去，你要学会“问问题留有退路”！
      · 就是说你自己要有与你问的问题相关的有趣的事情可以讲，要不然，你的问题就应该换成其他你有话可说的题目~
      · 比如我的话题可以是：
        - “你上学期间有没有离退学只有一步之遥，就是那种差一点就退学了的特殊经历”
        - “你想知道为什么我从没有复读过，却比同班人年龄要大一岁吗”(制造悬念)

       

      + 怎么让人感觉到和你有很多“共同话题”，和你总是有“说不完的话”，和你聊天怎么那么“轻松” 呢？

      • - 切记：一开始就能制造出双方都很想讲话的气氛~ 注意用合适的方式探讨出双方都有聊的话题。

       

      + 聊天时，每个人都想聊自己！(这个世界的真相是：每个人都活在自己的世界里)
      @联想:《把时间当作朋友》中所说的"每个人在自己遇到痛苦时，都会觉得自己是这个世界上最不幸的人"！这正是因为我们每个人都是活在自己世界里的人，我们感受不到其他人的痛苦！

      - 所以，我们如果在乎某个人，就学会“去倾听他对自己所遇到的事情的倾诉”吧。
      - 要想扮演一位“最上道”的朋友的角色，其实有一个很简单的最高原则：那就是“尽量别让自己说出‘我’这个字”！

      • · 这个规则很简单，你也可以试试看，跟朋友聊天的时候不要说出“我”字！
        当你要说“我”字的时候，都改成“你”字或是“他”字~

     

    + - 11-20

    • + 问题如果很尖锐，那就试着倒推回去两三步！

      • - 不要总是认为“我才是正常人”，“我才是社会的中坚”。不要总让自己充满优越感。

      + 适度的挑衅，能让谈话活络起来

      • - 这是因为每个人都希望自己的意见被重视，被探讨，而不是被一个完全没有原则的人敷衍了事，或是被点头称是对付过去~

       

      - 人生处处有地雷，如果你还没有被炸到，那只是因为你还没有踩到！而不是因为你那片区域这么好都没有地雷~
      - 切记不要“交浅言深”，因为这会让你一不小心就踩到地雷~ 有些深言只有在大家交情够了的前提下，才能够被触及~

       

      + 避免硬生生的报出数字，因为这确实很难让人记住。(同时这样也能挑起话题，并避免尴尬)

      • - 举例：
        “小戈，你是哪一年出生的？”
        “我出生那年，迈克尔·杰克逊出了那首<Beat It>哦！”(顺便也可唱两句)
        “真的！这首歌超棒的！”
        “对啊，可惜歌红我人不红！”
        “哈哈，又不是你唱的”

       

      + 悬疑式说话：“想知道后来怎么了吗？”

      • - 大部分人心中对一件自己不知道的事情充满问号，想知道后来怎么样了！
        这是因为：人从原始时代开始，就学会必须知道很多事情后来是怎么发展的又是怎么结束的，从而向同伴们学习生存之道！
        举个例子：有两个穴居人小甲和小乙，有一天小甲对小乙说：“我今天遇到一只牙齿这么尖、满身都是毛、比大树还高的野兽”。小乙就会赶快接着问“后来呢？”。因为他最好知道小甲是怎么应付这个局面，才活下来的，以备他下次自己遇到这类野兽的时候，他才会知道该怎么去做。

        - 怎样判断自己说的东西“吸不吸引人”呢？那就是每讲几句话，就稍微停顿一下，看你的朋友会不会问“然后呢，后来呢”之类的话。

    • + 初次见面，切记：学会主动报上自己的名字！防止别人不知道怎么称呼你，给别人造成尴尬或麻烦。

      + 赞赏，观察对方最渴望的部分。而且最好是往具体里面夸，这样就显得更加真诚~

       

      + 演讲的人，要设身处地的以台下观众的立场来想，才能做出最精彩的演讲！

      • - 不管他是自己花钱买票，还是被老师逼着来听你的演讲。只要你体谅听讲者容易恍神，给他一些乐趣，以及两到三样有用的讯息，那么他就会觉得你是个非常好的演讲者，而且下次也还会乐意听到你的演讲。

      + 调整心态，学会以别人的快乐为快乐。这是凑趣的有效途径！
      (对自己在意的人，要做到 只要你开心，我就开心)

       

      + 一个人很难了解另一个人所受的苦，一定要能识别出所谓的“引蛇出洞”式的诉苦，这种诉苦通常不是真的在诉苦，而是希望你夸她们并没有她们自己所说的问题，比如：

      - 如果她的诉苦不是在“引蛇出洞”，而是真的在诉苦，那么你需要做的就是“认真去倾听”~
