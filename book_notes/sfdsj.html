      = 自然派哲学家

      • 没有一件事物可以来自空无

        Q3: 万事万物是否由一种基本的物质组成？

              水能变成酒吗？

              泥土和水何以能制造出一只活生生的青蛙？

        早期哲学家相信，大自然是由四种元素所组成的，他称之为四个"根"，这四个根就是：土、气、火、水。

        并指出大自然所有的变化，都是因为这四种元素相互结合或分离的缘故，因为所有事物都是由泥土、空气、火和水混合而成，只是混合的比例不同而已。

        早期哲学家将自然界的变化解释为四个"根"的分合聚散，但是是什么原因导致这种聚合和分散呢？于是由此又认为自然界中存在两种力量，"爱"和"恨"，爱使得事物聚合，而恨则使他们分散。

      = 柏拉图

      • 回归灵魂世界的渴望

        Q6: 面包师如何能做出五十个一模一样的饼干？

              必须肯定的回答人的灵魂是否不朽？

        柏拉图认为世界是分成两个领域的，分别是 感官世界 和 理型世界。

        柏拉图认为：世界上的每一种生物都是理型世界中永恒形体的不完美复制品。理型世界内的永恒形式就是所谓的"不朽的灵魂"。

      柏拉图的理想国

      • 所谓理想国，就是一个虚构的理想的国度，简而言之，我们可以说柏拉图认为在这个国度应该由 哲学家 来治理。
      • 在柏拉图的构想中，一个国家应该像人体一样，由三个部分组成。就像人有头、胸、腹一般，
      • 一个国家也应该有 统治者、战士和工匠(农夫)。也正如一个健康和谐的人懂得平衡和节制一般，
      • 一个"有德"之国的特色是：每一位国民都明白自己在整个国家中所扮演的角色。
      • 在这个理想国之中，统治者和战士都不能享受家庭生活，也不许拥有私人的财产[避免了滋生腐败等诸多现象]。同时，由于养育孩童的责任重大，因此不可由个人从事，而必须由政府来负责。
      • "宪法国家" 是仅次于 理想国的最好国家。在这个国家中，在上位者可以拥有个人财产与家庭生活，也因此妇女的自由较受限制。但是仍然需要教育并训练其女性国民。

      = 思考 石头，植物，动物 和 人类 之间的区别

      = 斯宾诺莎

      • 上帝不是一个傀儡戏师傅

        一个真正的傀儡戏师傅，拉动所有的绳子操纵一切的事情，他是从外面来操纵他的木偶，因此他是这些木偶做出各种动作的 "外在因"；但是上帝并非以这种方式来主宰世界。上帝是透过自然法则来主宰世界，因此上帝(或自然)是每一件事情的 "内在因"。

        对上帝的这种主宰世界的看法的理解：可以想一想非洲的一只狮子，你认为是它自己决定要成为一只野兽的么？它是因为成为野兽才攻击一只跛脚的羚羊么？它可不可以自己决定要吃素？答案是No，因为狮子会按照自己的天性来做！所谓【天性】就是"自然法则"，上帝就是通过这种所谓的天性来主宰整个世界!

        史宾诺莎强调世间只有一种存在是完全自主的，且可以充分自由行动的，那就是上帝(或自然)。

        史宾诺莎也指出，使我们真正无法获得幸福与和谐的原因是我们内心的各种冲动，例如我们的"野心和欲望"，但如果我们体会到每一件事情的发生都有其必然性，我们就可以凭借直觉来理解整个大自然，我们就会很清楚的领悟到每一件事情之间都有关联，每一件事情相互之间都是一体的！

        最后的目标是：以一种全然接纳的观点来理解世间的事物，只有这样，我们才能获得真正的幸福和满足！

        一件事情跟着另外一件事情发生，并不一定表示两者之间必有关联，哲学的目的之一就是：教人们不要妄下结论！

    • = 康德

      • 头上闪烁的星空 和 心中的道德规范

        康德[1724-1804]于 1724年 诞生于普鲁士东部的哥尼斯堡，父亲是一位 马鞍师傅。康德是一位 哲学教授。

        康德认为我们对于这个世界的观念是 我们同时透过感官与理性而得到的，不过他认为理性主义者将理性的重要性说的太过了，而经验主义者则过分的强调了感官的经验！

        康德的概念是：时间和空间属于人类的条件，时空乃是人类感知的方式，并非物质世界的属性。康德认为："事物本身" 和 "我眼中的事物" 是不一样的。

        康德的伦理学有时也被称为"善意的伦理学"，他认为一个人的行为是否合乎德正 取决于你是否出自善意而为之，而并不取决于你的行为后果。

        康德的墓碑上的墓志铭是：

        "有两件事物我愈是思考愈是觉得神奇，心中也愈充满敬畏，那就是我头顶上的星空与我内心的道德准则。它们向我印证：上帝在我头顶，亦在我心中。"

        人生的三阶段—— 美感阶段、道德阶段 和 宗教阶段。

        活在"美感阶段"的人只是为了 现在 而活，因此他会抓住每个享乐的机会。只要是美的、令人满足的、令人愉快的，就是好的。这样的人完全活在感官的世界里，是他自己的欲望与情绪的奴隶；

        活在"道德阶段"的人的特色是 对生命抱持认真的态度，并且始终一贯的做一些符合道德的抉择，人应该努力的遵循道德法则而生活，在这个阶段的人做事情会开始在意事情的是非对错，而不像活在美感阶段的人则只注重一件事是否有趣；

        活在"宗教阶段"的人通常会选择信仰，而不选择美感的愉悦和理性所要求的责任。

    • = 马克思

      • 在欧洲游荡的幽灵

        一世人劳苦奔忙有何益？到头来终究须把眼儿闭。

        马克思是一位 辩证的唯物主义者。

        关于帕特农神庙：看到这座建筑有一个 非常优雅、精巧的屋顶，当你看到这座神殿时，也许第一眼看到的就是这个屋顶和它前面的山形墙，这就是我们所说的"上层结构"，可是屋顶不会飘浮，它必须有柱子支撑，这也就可以解释上层建筑是需要经济基础来作为强有力的支撑的。

        而这些柱子又是立在一个由三层台阶所组成的基座上，这代表着我们可以把整个社会的'基础'分成三个阶层，最根本的阶层代表一个社会的 '生产条件'，也就是这个社会可以利用的自然条件和资源，这个基础决定了这个社会的生产种类，同时也决定了这个社会的性质与它的整体文化；在此之上的一个台阶代表社会的 '生产工具'，这里指的是那些生产设备、工具和机器这些东西；再上面的一级台阶代表的是社会的 '生产关系'，也就是说它代表着那些拥有生产工具的人之间分工的方式，以及财产的分配关系。

        关于阶级斗争：阶级冲突乃是存在于那些拥有生产工具的人 和 那些没有生产工具的人之间。既然上层阶级不会自愿放弃权利，因此唯有透过 革命 才能改变社会现况。

        关于人的工作：年轻的马克思专心一意的研究 人在工作时所发生的现象，最终发现人与自然之间有一种互动的或辩证的关系，当人通过劳动改造大自然时他本身也就被改造了，从某个角度来看，一个失业的人就是一个 "空虚" 的人。

        所以工作是一件具有正面意义的事情，并且与人类的本质有密切的关系。

        本我、自我、超我

        对 "梦" 的理解：

    • = 那轰然一响

      • 我们也是星尘

        在银河系中，离我们最近的一颗恒星距离地球有四光年，也许它正在我们这个岛的上方。此时此刻，如果那颗星球上有一个人正用一个强大的天文望远镜对着柏客来山庄看的话，他看到的将是四年前的柏客来山庄。他也许会看到一个十一岁的女孩正坐在秋千上晃动她的双腿。而此时此刻地球上的我们，正坐在这里探讨这他们。

        我们只要眺望太空，所看到的就一定是以前的太空，我们永远无法知道现在的宇宙是个什么模样。我们只知道它当时如何。当我们仰望一颗有几千光年的星球时，我们事实上是回到了几千年前的太空。

        宇宙目前依旧还是不断膨胀的，原因是大约在150亿年前，宇宙间所有的物质都集中在一个比较小的范围内，由于物质密度极高，再加上重力的作用，使得这些物质的温度高的吓人，温度日趋上升的结果就是这一团紧密的物质终于爆炸了，我们称这个现象为 "宇宙大爆炸"，宇宙大爆炸使得宇宙中所有的物质都向四面扩散，当这些物质碎片逐渐冷却后，就形成各个星球、银河系、卫星和恒星等等。

        关于未来的宇宙，有两种观点：一种就是宇宙会一直扩张下去，使得各个银河系之间的距离越来越远；另一种就是相信宇宙会再度由于引力和惯性的原因而收缩。

        如果是后一种可能，在我们的想象中，那就像是有一颗 “宇宙的心脏" 不断在跳动的情景。

        在晴天的夜晚，我们可以看到几百万、甚至几十亿年前宇宙的面貌。所以，我们可以说正在回家的路上。我们也是在大爆炸时开始，因为宇宙所有的物质整个是一个有机体，在万古之前，所有的物质都聚合成一大块，质量极其紧密，因此即使是小如针头般的一块，也可以重达好几十亿吨。在这样大的重力作用下，这个'原始原子'爆炸了，就好像某个东西解体一样，所以说当我们仰望星空时，我们其实是在试图找寻回到自我的路。

        宇宙中所有的星球和银河都是由同一种物质做成的。

        我在想那些遥远的星球上是否也有人！
