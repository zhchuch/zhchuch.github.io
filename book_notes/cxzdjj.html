创新者的窘境
作者：[美]克里斯坦森

  • The Innovator's Dilemma: When new technologies cause great firms to fail!

     

    前言：

    • + 领先企业在遇到破坏性技术变革和市场结构变化时，走向失败的例子不胜枚举。
      本书主要想讨论：那些原本出于行业领先地位的企业在遭遇某种形式的市场变化和技术变革为什么无法继续保持自己的行业领先地位呢？

       

      + 本书总结出了一种规律 “破坏性创新原则”，希望能够帮助管理者判断何时应采用和实施什么样的管理原则。
      本书的宗旨是：在缓慢发展和快速发展的环境下，为制造业和服务业的许多管理者、顾问和学者提供帮助！

       

      + 延续性技术 和 破坏性技术

      • - 延续性技术注重推动产品性能的改善；而破坏性技术则给整个市场带来了与以往截然不同的价值主张；
        - 破坏性技术相比延续性技术往往具有：产品结构更加简单，产品价格更加低廉，产品体积变小，毛利率更高等特点，但伴随这些特点还有一些缺点就是：性能可能更低，市场不可知，不能为大企业带来预期的增长率等！

       

      + 创新者的“窘境”：
      - 1. “好”企业在大举投资能给企业带来最大利润的消费者所希望获得的产品和服务时，通常也会同时埋下导致日后失败的种子！
      - 2. 良好的管理决策也正是导致这些成熟企业从行业龙头地位落马的根本原因！

       

      + 本书组织结构：
      第一至二章：以硬盘驱动器行业为背景建立失败框架；
      第三至九章：探讨失败框架的外部效度，失败框架可能带来的有用的启示；
      第十章：利用失败框架对电动汽车案例进行分析，总结经验教训，并分析如何利用规律取得成功；
      十一章：则对全书的发现做一个总结。

       

      + 本书的部分目标：提出破坏性技术的五大法则或原理并总结出规律

      • 1. 企业的资源分布取决于消费者和投资者；
        2. 小市场并不能解决大企业的增长需求；
        3. 无法对并不存在的市场进行分析("基于发现的规划")；
        4. 机构的能力决定了它的局限性；
        5. 技术供应可能并不等同于市场需求。

     

    第一部分：为什么大企业会失败

    • + - 1. 大企业为什么会失败？(从硬盘驱动器行业获得的启示)

      • + 创新者的窘境：盲目地遵循“好的管理者与消费者要始终保持密切联系”这一箴言。

        + 如何区分“成熟企业”和“新兴企业”？
        主要看该企业是什么时候进入到这个行业的，如果是在一项新技术变革发生之前进入的那么它就是“成熟企业”，而如果它是在这种新技术变革发生时或不久进入的那么它就是“新兴企业”！

        + 延续性技术变革

        • - 这类技术变革注重的是：沿着既定的轨道去延续或是强化产品性能的改善过程！
          - 在硬盘驱动行业内的几乎每一个延续性技术变革中，成熟企业都在研发和商业化运作中处于领先地位。
          - 在延续性技术变革中，成熟企业对那些研发风险大，复杂度高，其售价昂贵的组件技术具有先天优势，所以它们总是领先于新兴企业！

        + 破坏性技术变革(创新)

        • - 破坏性创新往往并不需要涉及到特别复杂的技术变革，它主要的表现形式就是：将成品元件组装在一起，但相比之前的产品，产品结构通常会变得更加简单。
          - 破坏性创新最初会引起新兴市场的关注，而不被主流市场所接纳！但随着这种新技术的不断发展，最终它又可以通过自己强有力的竞争力以及简单的结构而侵占主流市场。最终，使得大企业走向衰败。

          - 破坏性创新找寻新兴市场，新兴市场中出现大批的新兴企业，而这些新兴企业本身就是在引领着这些破坏性创新技术变革！它们形成了一个圈，让那些一心专注于主流市场的客户要求而不采用新技术的成熟企业渐渐失去优势，甚至是走向失败！

        + 为什么大企业不领先去做那些所谓的“破坏性创新”？

        • - 很重要的一大原因就是：大企业受制于客户！
          它们以客户为上帝，而客户所代表的主流市场是不会鼓励你去尝试做这种创新的。

       

      + - 2. 价值网和创新推动力

      • - 本章想要解决的问题：为什么领先企业似乎被它们的客户牵绊住了手脚？为什么那些具有攻击性的新兴企业具备可乘之机？提出了价值网理论！

         

        +价值网的概念：企业在一种大环境下确定消费者的需求，并对此采取应对措施、解决问题、征求消费者意见、应对竞争对手的竞争，并争取自身利润最大化！
        - 价值网反映了产品结构，典型的嵌套式商业体系就是一个价值网！
        - 不同的价值网为某种特定的性能属性会确定出截然不同的价值！成熟企业往往只是按照主流市场的价值网作为自己发展和决策的依据，所以它们很难追逐新兴市场，搞出破坏性创新；
        - 不同的价值网提供了它所定义下的“有价值的产品和服务”，并制定出了特定的成本结构。这种成本结构对一个成熟企业的灵活性和最终命运有着重大影响。

        +之所以成熟企业和新兴企业两者在延续性和破坏性创新各有千秋，很可能真正的原因在于：它们本身就处在不同的价值网中！

         

        + 管理决策过程和破坏性技术变革

        • - 企业所属的价值网内的竞争在很多情况下，决定了企业的赢利模式。因此在价值网内外企业之间的管理决策之间可能毫不相干。
          -  典型的由企业所处的价值网所确定的决策模式【大企业的失败框架】：
          1. 破坏性技术首先在成熟企业研制成功；
          2. 市场营销人员然后收集公司主要客户的反馈；
          [因为价值网中客户群的关系，这种来自旧客户对新技术的反馈往往是负面的，使得企业做出决策将这种新技术束之高阁，但它们却没意识到这种新技术此时正在新兴市场上攻城略地]
          3. 成熟企业加快对延续性技术的开发步伐；
          4. 新企业出现，破坏性技术市场在反复尝试中逐渐成形；
          5. 新兴企业向高端市场转移；
          [你要知道，人都是利益驱动的，是会变的。成熟市场也不是傻子，这些消费者最终会出于新产品的优势而接受那些曾经被他们拒绝过的东西，这些新产品容量和速度达到需求且具备更小的体积和更简单的结构，最主要的优势更可能是它们往往还有更便宜的价格]
          6. 成熟企业在维护客户基础这一方面棋慢一招，这就是它们失败的主因！
          [成熟企业的所谓主流市场正在被蚕食]

          - 总结：“密切关注消费者”往往并不总是箴言，这些消费者往往只是充当“延续性创新”进程中的引导者，但在“破坏性创新”进程中他们却只是充当一个“利益驱动的变色龙”，在这个进程中他们反而常常会去误导供应商！

         

        + 技术S形曲线

        • - 技术S形曲线经常被用于预测新兴技术是否有可能取代成熟技术！

        + 价值网框架所带来的启示

        • - 启示1：成熟企业关注自己所处的价值网中的主流客户的需求，所以一旦他们的客户有新的需求出现，他们也能够利用自己的各种资源来开发甚至是收购那些所谓的“破坏性”新技术。所以很多预言其实都有可能预言失败。
          - 启示2：只要“破坏性技术”与原来延续性技术所针对的价值网不同，两者所面向的主要市场不同，那么传统成熟企业也不会妄想去取得自己在新市场中的领先地位，因为这与它本身所处的价值网(抑或说是客户群)的利益不符！

        + 全章总结

        • 一. 价值网体系对创新的5点建议 [运筹帷幄，纵横捭阖]
          1. 企业的价值网对其本身的技术和能力有着深远的影响(价值网的界限很重要)；
          2. 创新是否能成功的一个重大因素是：该创新能够在何种程度上满足价值网内已知参与者的已知需求；
          3. 成熟企业千万不能忽略那些无法解决当前消费者需求的新技术，因为很可能在将来它就会变成消费者的新宠；
          4. 新兴企业所谓的冲击优势，不过是因为这些新技术在已成熟的价值网中不会创造价值；
          5. 新兴企业要想击败成熟企业，取得领先地位，就必须学会灵活的制定战略计划来冲击现有的成熟价值网，并发展新兴市场(或称之为“价值网”)。

          二. 分析技术创新的新思路 [立足当下，展望未来]
          - 面临破坏性技术创新的企业，必须学会分析该创新对其相关价值网的影响，该创新可能为自己的目标市场所带来的冲击！
          - 面对破坏性创新，需要多思考思考以下3个问题：
          Q1. 这些创新在新兴企业已经建立的价值网中是否被得到重视？
          Q2. 该创新为了实现自己的价值，是否必须进入其他价值网(或是否必须建立新的价值网)，而这又会不会给我的价值网带来冲击？
          Q3. 我的主流市场和这些技术创新的轨线是否有可能最终交汇？(其实是在思考：我的主流市场未来的需求会不会随着这种新技术的发展而变化？)

       

      + - 3. 机械挖掘机行业的破坏性技术变革

      • 本章带领我们看看在挖掘机行业破坏性技术变革(液压技术)是怎么把成熟企业给干倒的！
        [整个液压新技术逆袭的过程中，原先那些大企业所采取的措施完全符合我们的失败框架，最终确实那些传统的大企业走向了失败]

         

        + 那些原本领先的成熟企业在“液压技术”还只是“星星之火”时忽略了它的存在，最后当它成就“燎原之势”的时候，成熟企业已然无力抗拒失败的命运！

        + 新兴企业是如何一步步逆袭传统成熟企业的？

        • - 一开始传统成熟企业(生产 蒸汽->内燃铲土机) 在延续性技术变革中始终处于领导地位！
          - “液压新技术”这种破坏性技术变革的出现，刚出现时这种技术在传统价值网的属性价值来看，简直就是垃圾，性能极差！所以传统成熟企业不屑于去采用这种新技术，又或者说成熟企业所针对的用户群对该技术没有需求，所以成熟企业不愿去做；
          - 于是，采用这种新技术的新兴企业不得不为它们的产品开发新的应用领域(新的价值网)，最后它们成功找到了一个面向小型工业用和农业牵引车的应用领域，建立了一个新的价值网，重新定义了铲土机行业的性能评估指标！
          - 接着，“液压技术”铲土机得到快速发展，并有资格和实力向传统价值网发起进攻，此时的成熟企业已然回天乏力，它们现在所做的最后的抵抗无非就是“致力于在成熟市场内引入这种破坏性技术”，但是那些成功的新兴企业想的则是“发现并攻占自己的新市场”，最终传统成熟企业走向败亡！

        @思考：为什么破坏性技术会使得创新者陷入窘境？

        • 1. 因为这些破坏性创新技术，在刚开始出现的时候，对成熟企业的价值网来说毫无价值；
          2. 成熟企业迫于现有主流市场上的竞争压力，不得不着眼于全心全意为主流市场上的客户需求服务，于是它们选择了忽略这项在它们的价值网看来毫无价值的新技术；
          3. 新兴企业得以出现，并极力找寻适合这种新技术的“新价值网”，并开始全力发展自己以求在新价值网下变得更有价值；
          4. 新兴企业发现新技术发展的很快，并开始向传统价值网发起攻击；
          5. 成熟企业此时再想重头引进这种新技术已经为时已晚，相比新兴企业它们渐渐的失去了优势；
          6. 主流市场在利益的驱动下，转向新兴企业，成熟企业彻底败北！
          总结：这个窘境是怎么造成的？说白了就是主流市场在利益驱使下转变了阵营所造成的！真的是“成也萧何，败也萧何”啊！

       

      + - 4. 回不去的低端市场

      • 看到这个标题：第一反应是联想到了《人类简史》里面对农业革命的一句评论：“人类已经陷入到了一个奢侈品的陷阱”。陷进来容易，你想再走回去基本就不太可能了！
        引申到这里就是：“那些好的大企业已经陷进了高端市场的陷阱”，他们打入高端市场容易，但是再想做低端市场的生意可就没那么简单了！

         

        + 价值网会阻碍成熟企业向低端移动，阻碍成熟企业进入由破坏性技术所催生的低端市场。
        -> 本章想要总结出一套框架来解释：为什么进入高端市场价值网具有如此大的吸引力，我们来分析一下这种力量。

        + 价值网与成本结构
        价值网要求企业确定其相适应的成本结构，这也促使企业建立一种特别的提高自己盈利能力的模式！这种成本结构和盈利模式促使企业向高端市场迁移！
        + 总之，推动那些企业不断向上流动，限制它们向下流动的根本原因在于：那些企业的理性资源分配流程！

        + 好的资源分配体系的初衷就在于：如何摒弃那些不太可能找到大规模、有利可图和具有包容性市场的提案！所以：一个成熟企业往往在自己良好的资源分配体系的作用下，不自觉的走向高端市场同时摒弃低端市场！

        + 最令人困惑的管理特征：最理想的增长和盈利捷径是向上流动，而最致命的冲击却来自于低端市场！

         

        + 为什么成熟企业想要尝试破坏性新技术，想要赶上下一轮破坏性技术浪潮，但实际却很难实施？

        • - 一个很重要的原因在于：它的实施不仅受到来自客户需求的制约，也更加受到这个企业本身由价值网所确定下来的财务结构和企业文化的制约！
          简单的说：“客户不希望你这么做，公司员工也不希望你这么做”，请问你这么做还能成功吗？

        + @总结：

        • 1. 致使成熟企业不断进入高端市场的这股强大吸引力，主要有三大因素所致：a. 高端市场的利润率很高，会自然吸引成熟企业去做；b. 成熟企业本身所针对的主流市场上的客户也在向高端市场移动，所以为了提高自己的竞争力，它们也不得不跟着向上移动；c. 削减成本进入低端市场并最终获得能够媲美高端市场上的利润，这个本身难度真的很大。

          2. 为什么成熟企业回到低端市场这么难？
          由于增长型企业每年需要大幅提高收入水平才能维持它们所期望的增长率，因此作为一种增长手段，这些企业越来越不可能通过小型市场来获取它们所希望获得这部分新收入！

     

    第二部分：管理破坏性技术变革

    • 本部分详细分析了在面临“破坏性技术变革”时一些成熟企业的 成功案例 以及 更多企业的 失败案例！
      + 目的是：
      * 帮助我们应对“破坏性技术变革”困惑的 五大基本企业管理原则！
      {1. 资源依赖性(依赖于消费者)；2. 小市场与大企业的增长需求；3. 破坏性技术的最终用户或价值网的不可预知性；4. 机构的能力独立于机构内的工作人员能力；5. 技术供应并不等同于市场需求}
      * 同时分析一下成功企业是 如何利用 这些原则来建立自己的竞争优势，从而获得成功的！
      + 结论是：
      在遭遇破坏性技术时，所有的这些行业中导致那些成熟企业最终成败的因素都是一样的！(我们可以总结出一套规律来帮助我们更有效地应对破坏性技术)

       

      + - 5. 把开发破坏性技术的职责赋予存在客户需求的机构

      Rule-1: 资源依赖理论：真正决定企业未来发展方向的是消费者，而非管理者，成功企业必然是以消费者为导向！[所以：不论企业面临什么样的技术，在企业管理的过程中，对服务对象(或目标用户)的选择对于该企业本身来说将具有重大的战略意义]

      •  

        + 如何利用 Rule-1 ？

        • - 面对破坏性技术，企业管理者的两种选择：
          Select-1. 说服企业内的每一个人，要把目光放长远，然后在企业内去开发这种新技术；
          Select-2. 另外新建一个独立机构，让它按照创业机构的那一套新模式去运作，让它自己去主动找寻新兴消费者群体，被借此开发新技术。
          - 事实证明 Select-2 取得成功的概率要远远高于 Select-1。
          这是因为第二种方法实际上是选择了“顺从并利用”Rule-1，而第一种选择则是在尝试“抵抗”Rule-1。

        + 消费者何以取得如此重要的决定性作用？

        • - 因为消费者决定了资源分配流程！
          一个企业的资源分配流程决定了哪些提案可以得到人员和资金等资源的支持，哪些提案会被否决。而这又决定着该企业未来的发展方向。

        + 成功案例：
        1. 顺从 Rule-1 成功的案例：昆腾公司(Plus公司)、数据控制公司；
        2. 对抗 Rule-1 成功的案例：Micropolis公司(为成功付出了巨大的代价)。
        + 对抗 Rule-1 更多的失败案例：
        1. IBM在微型计算机领域的失败；
        2. 数字设备公司在台式计算机领域的失败；
        3. 伍尔沃斯公司在折扣零售领域的失败 等等。

        @总结：“顺者昌逆者亡”，不要试图在同一家机构内采用两种不同的赢利模式，或者说：不要试图将新兴消费者的需求(或新价值网)强加给现有成熟企业的员工，否则他们一定会反噬你的！

       

      + - 6. 如何使得机构与市场规模相匹配

      Rule-2: 规模匹配原则：在面对破坏性创新技术项目的时候，我们必须让市场规模与机构本身规模相匹配！


      • - 引领延续性技术变革成为先行者可能并没有多大意义，只有在破坏性技术中处于领先地位能够创造出巨大价值！
        - 相比增长陷入停滞的企业，增长型企业的新产品和流程技术开发项目更容易获得投资！而伴随着企业的发展壮大，维持企业的高增长率难度变得越来越大，所以这些企业在应对破坏性新技术时陷入窘境！

        + 如何利用 Rule-2 ?

        • - 面对这一窘境，企业管理者的三种选择：
          Select-1. 推动新兴市场的增长率，把它变得足够大以至于达到企业原有成熟市场增长率的水平；
          [可说真的非常难，几乎不可能。因为小市场终究不能满足大机构在短期内的增长需求。]
          Select-2. 等到市场发展到一定规模之后再进入；
          [这样新兴公司极可能已改变新市场的游戏规则，你发现你已经在这个市场上失去先行优势。]
          Select-3. 让小机构去利用小机遇，将破坏性技术的发展职责交给规模足够小的机构。
          - 事实证明 Select-3 取得成功的概率要远远高于前两个。
          这是因为 Select-3符合了“规模匹配理论”，小机构的利益和着眼点都在探索和发展新兴小市场上，因此是应对破坏性技术的极佳选择。

        + 成熟企业具体要怎么做？

        • - 具体的方法可以是建立一个独立的分拆机构，抑或是直接收购一家与破坏性市场规模相匹配的小企业！然后让它们去独立的研究和发展这项新技术！

       

      + - 7. 发现新的新兴市场

      Rule-3: 市场不可预知性：破坏性技术的市场应用领域不仅是未知的，而且还是不可预知的！


      • + 延续性创新的特点是：企业能事先基本掌握消费者需求，也就是说所面对的市场基本是已知市场；
        + 破坏性创新的特点则是：这会带来新兴市场，而这个市场在事先不可预知！需要不断地去学习与尝试！

        + 延续性创新市场是怎么做预测的？

        • - 常见措施：采访主要客户和行业专家，进行趋势分析，建立经济模型等等。

        + 如何利用 Rule-3 ？

        • - 成熟企业针对破坏性新技术的两种选择：
          Select-1. 一开始就对市场做出预测，并始终坚信自己已经找到了正确的战略方向，然后孤注一掷投入巨大精力去开拓这个自己所预测的市场；
          Select-2. 在制定详尽计划前采取行动，时刻铭记该技术的市场应用领域是不可知的，接着在实践中不断摸索和学习出正确的市场方向 (这种方法又被称之为“发现驱动型”市场探索方法)。
          - 事实证明 Select-2 是一个成熟企业面对破坏性新技术在开拓新兴市场过程中一条捷径！

        @总结：在面临破坏性新技术时，要注意不要相信那些所谓专家的预测，市场永远是在实践摸索学习的过程中不断探索出来的！要知道最初的战略随时可能会转变，要为接下来的转变和尝试保留足够的资源！

       

      + - 8. 如何评估机构的能力和缺陷

      Rule-4：机构能力匹配原则：机构本身也具有自己的“核心能力”，只有让机构能力与机遇规模相匹配，才能最好的开发新技术，发掘新市场。


      • + 什么东西在影响着一个机构(或企业)本身的核心能力？

        • + RPV框架(Resource-Procedure-Value, 资源-流程-价值观)

          1. 资源：表现为事物和资产的形态，比较直观，对机构能力的影响也比较初级；
          2. 流程：是指人们相互之间所采取的互动、协调、沟通和决策的模式，是为了解决特定任务而固化下来的一套行之有效的经验；
          3. 价值观：企业的价值观是一个企业据以确定决策优先级时所遵循的标准，它帮助一个企业内的所有员工学会遵照企业的战略方向和业务模式来自主确定决策的优先级别。

        + 一个成功企业的价值观会受到什么因素的影响而发生改变？

        • 1. 受到可接受的毛利率的影响而改变；
          2. 受到该企业发展规模的影响而改变。

         

        + 一个企业核心能力的转移

        • - 决定一个机构能力和缺陷的最为有利的因素会随着机构的发展不断发生变动：从 资源 到 流程和价值观，再转化为 企业文化！企业文化是一个企业内最强大的管理工具，它能让所有员工在需要做什么和如何做的问题上达到一致，它能够保证所有员工持续不断地开展正确的工作！
          - 一个企业的核心能力最终会根植于其流程和价值观中，而不是仅仅依赖它的资源！

        + 如何利用 Rule-4 ？

        • - 注重一个企业本身能力(流程和价值观)对无论是延续性创新还是破坏性创新技术的影响力，管理者必须仔细分析机构或企业自身已固化下来的流程和价值观是否能和任务相匹配！
          - 如果两者不匹配，那么在该企业内研发该任务就会极大地受到流程和价值观的牵制，甚至极有可能被搁置！

          - 如果管理者确定机构的能力不适应新的任务，那么他通常会采取如下三种方法来创造新能力：
          Select-1: 通过收购来壮大自己的能力；
          [收购另一家流程和价值观与新任务极为匹配的公司]
          Select-2: 在企业内部创造这种新能力；
          [试图在本企业内部改变当前的流程和价值观]
          Select-3:  通过分支机构创造新能力。
          [成立一个独立机构，并在该机构内针对新问题开发出一套新的流程和价值观]

          - 事实证明：
          1. 收购一家公司时必须要考虑自己最看重这家公司的资源还是能力(流程和价值观)，然后根据实际情况决定是合并还是给它提供资源让其独立发展；
          2. 不推荐，因为机构本身的流程和价值观是它最基本的能力，是已经固化下来的东西，很难改变，而且如果改变的不好，甚至会拖累原有的那一套流程和价值观；
          3. 推荐，因为这样就做到了“机构能力匹配原则”。理想的状况是：每一家企业都根据每一个项目对流程和价值观的具体要求来量身打造团队结构和机构定位！

        + 什么时候需要成立独立的分支机构去完成特殊使命？

        • - 当你发现这家企业主流的价值观已经致使其无法将资源集中投入到这个创新项目时，就到了需要成立与该项目相匹配的独立分支机构的时候了！

        + 独立机构的“独立”二字实际上是指的什么？

        • - 这里的“独立”，其实并不是看重其形式上的独立，而真正重要的是看它能否独立与原本成熟企业的常规资源分配流程！而是看它能否顺利的得到自己发展过程中所需要的资源！
          - 这里的“独立”，就是想要让新企业形成一种应对新挑战的新流程和价值观。

        @总结：当机构遭遇变革时，管理者要考虑：1. 他们本身是否具备应对变革成功所需的资源；2. 机构是否具备应对变革成功所需的流程和价值观！
        这两个问题都很重要，而很多人却往往忽略了对第二个问题的考虑！

       

      + - 9. 产品性能、市场需求和生命周期

      Rule-5: 技术供应与市场需求的博弈：当发生性能过度供给时，破坏性技术的机遇也呼之欲出，然后开始从价值网的下方冲击成熟市场！性能过度供给还可能会导致产品市场的竞争基础发生根本性变化！


      • + 性能过度供给引发竞争基础的变化(以硬盘驱动器行业为例)

        • - 当消费者对某项属性的需求已经得到满足时，该项属性在消费者选择产品时也就会变得不再重要，他们会将选择的标准转移到市场需求尚未得到满足的新的产品属性上！

          - “容量”属性性能的过度供给，使得竞争基础转移到了“体积”；当“体积”属性得到过度供给，竞争基础又转移到了“可靠性”；当“可靠性”需求得到满足，“价格”又成为新的竞争基础！

          - 总结：当产品特色和功能已经超出市场需求时，你在原有竞争基础上与别人的产品之间的差异化也就失去了意义！因为新的竞争基础已经形成！

         

        + 购买等级原理

        • - 一个产品的演变过程分成四个阶段：功能性、可靠性、便捷性和价格！
          - 当没有产品能够满足市场对功能性的需求时，竞争基础和选择产品的标准就是：产品的功能性；
          - 当一旦市场上有两种或以上的产品都能满足市场对功能性的需求时，此时消费者选择产品的标准就发生了变化：关注产品的可靠性；
          - 相似的，后面两个阶段也是由于处在它前面的属性的需求已经得到了满足！

          推动购买等级从一个阶段向另一个阶段转变的因素就是：性能的过度供给！

         

        + 破坏性技术的一些特征：

        • - 它预示着竞争基础即将发生改变；
          - 在主流市场几乎没有价值的属性通常会成为它们在新兴市场上的最大卖点；
          [新兴企业关注的是去学习和探索新市场，关注的是进行市场营销挑战，而不像成熟企业那样把破坏性技术定义为一种由成熟市场所带来的技术性挑战]
          - 它相比成熟产品，一般更简单、跟便宜，也更可靠，更便捷！

        + 性能超出市场需求的产品将被迫接受与商品同等的价格(差异化失去了意义)，而重新定义竞争基础的破坏性产品则可以获得更多的溢价。

         

        + 如何利用 Rule-5 ？

        • - 成熟企业如何管理竞争基础发生的转变？
          Select-1: 进入高端市场，吸引更高端的消费者；
          Select-2: 始终与消费者保持一致；
          Select-3: 改变市场对功能性的需求。
          - 事实表明：
          所有成功的战略实施者都有一个共同点，那就是不管他们是有意识的还是出于本能，他们都深刻的理解：消费者的需求曲线 和 技术人员的供给曲线 两者之间的微妙关系！

       

      + - 10. 管理破坏性技术变革之案例分析 (电动汽车)

      • + 利用前几章所提出的理论和原则来阐述管理者怎样才能成功地应对破坏性技术变革！以“电动汽车”这个创新项目为例进行实例分析！

        + 管理者面对一项新技术需要思考的问题：

        • Q1. 我们要如何判断出一项新技术是否具有破坏性？[方法参见A1]
          Q2. 如果是破坏性技术，那么它的市场会在哪儿？(我的市场营销战略是什么)[方法参见A2]
          Q3. 我们应关注什么样的产品技术(技术战略是什么)以及它的经销策略(经销战略是什么)？[方法参见A3]
          Q4. 什么样的机构最适合进行破坏性创新？[方法参见A4]

        + 针对“电动汽车”行业思考的过程和结果：

        • A1: 通过绘制“市场需求性能轨线”与“新技术供应性能轨线”的轨线图，对两者性能的发展进行比较，是确定该项新技术是否具有破坏性的最佳方法！
          - 当发现电动汽车的性能改善曲线表明其日后有可能会发展为部分主流市场上具有竞争力的产品时，我们才能说电动汽车是一种破坏性产品！
          - 具体方法：分析两条轨线的走势，看这两条性能曲线最终会不会相交！
          - 结论：电动汽车具有破坏性！

          A2: 市场营销战略的三大基石：
          a. 尽快找到一个能使用电动汽车的市场(猜测一个新价值网)；
          b. 通过不断地测试和探索，通过不断的反复尝试，来确定最终真正的新兴市场(不要试图去问消费者，因为对于这个新市场大家谁也不比谁知道得多)；
          c. 整个创业计划是一个学习计划，而不是一次孤注一掷的博弈！(所以每一次尝试都需要保留足够的资源和实力，以备后续的再次尝试)。

          A3: 从产品设计谈起：
          - 破坏性产品设计的三个原则：
          a. 必须具有简单、可靠和便捷的特性；
          b. 必须设计一个产品变更平台(因为我们时刻准备着要根据市场改变产品)；
          c. 必须确定一个较低的价格点。
          - 破坏性产品的技术战略：
          不要试图涉及新技术(不要试图取得技术突破)，破坏性技术只是将成熟组件组合在一起，形成一种全新的产品结构，给消费者提供一种全新的体验！
          (技术突破交给拥有市场之后的延续性创新去解决吧)
          -  破坏性产品的经销战略：
          为电动汽车寻找或创造新的经销网络，因为已成熟的经销渠道都会出于利益的考虑不愿付诸努力。

          A4: 在有了努力寻找新市场的现实方向，有了产品设计、技术、经销网络等的总体指导方针之后，接下来就要考虑要采用什么样的机构？
          -  总原则：“规模匹配、机构能力匹配原则”！该机构必须独立于原有成熟机构的资源分配流程之外！
          - 针对破坏性创新技术：推荐设立一个独立的分支机构，该机构的成败取决于它对破坏性技术进行商业化的成败！这个小机构的首要任务就是：尽快实现资金的净流入(要能保证自己存活下来)，而不要去关心流入资金的规模！
          - 切记：成立分公司只有在面临破坏性创新技术时才是一个好的策略，在延续性创新过程中，它不会带来优势反而可能会给企业带来损失！

        + 总结：管理者要确保面临破坏性创新技术时，自己的团队是一个尊重而不违背破坏性创新原理的团队！

       

      11. 总结：创新者的窘境

      • + 企业必须意识到自己的机构所形成的流程，价值观和企业文化只在某些特定的条件下才具有价值(必须意识到并不是在所有条件下都有价值)！

        + 本书给应对破坏性创新技术的七点建议梳理

        • 1. 市场的需求和技术的供应两者的速度存在差异！
          （所以我们不能指望消费者会引导我们发展他们目前并不需要的创新）
          2. 一个企业的资源分配流程或者是价值网是固化的，很难改变！
          （所以管理者往往很难集中各种资源来发展破坏性技术）
          3. 市场必须要和技术相匹配！
          （所以破坏性新技术的首要任务就是去发掘与自己相匹配的新兴市场）
          （破坏性技术应当被看作是一种市场营销挑战，而非技术性挑战）
          4. 机构的能力依赖于具体的环境！
          （所以成熟企业由于其本身成熟价值网的限制，并没有具备那些破坏性新技术所要求的新能力）
          5. 在面对破坏性技术时，被没有可供借鉴的参考信息！
          （所以我们要不断去尝试，不断去学习）
          6. 企业采取一成不变的技术战略是不明智的！
          （因为破坏性和延续性新技术，本身就要求企业学会采取截然不同的战略）
          7. 市场准入和市场流动经常会遭遇强大的壁垒！
          （所以在面对破坏性创新时，发掘新市场需要成熟企业勇于争先，奋力拿下领先优势）
