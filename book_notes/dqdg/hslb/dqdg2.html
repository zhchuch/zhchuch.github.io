+ - "第一章 铁腕平乱"

  1. 义渠大牛首接受了羊皮血契

    • 世族阶层在车裂了卫鞅之后，进一步实现自己的'乱国'计划。老太师甘龙笼络西戎部落义渠，试图让其出兵靖难咸阳，意在 废商鞅之新法恢复其所谓的祖制。
      (其实其背后也是不安好心，加入真的靖难成功了，甘龙也想着再灭了义渠部落，从而把它当枪使。)

    @其实仔细想想这个老太师是多么的幼稚，这两件事任何一件单独拿出来都是很难解决的问题，他现在反倒去想这两件事做成之后自己一家独大，太天真了！

  2. 百骑扬威 震慑草原

    • 新君嬴驷 召来商君临终前向他推荐的大臣(樗里疾,山甲) 实施自己"铲除复辟,维护新法"的执政目标。他交给两位大臣的使命是：拆散戎狄部族与世族元老之间可能产生的叛乱同盟，釜底抽薪防患于未然！
    • 这次出使只有一个目标：不管你用什么办法一定要切断戎狄与咸阳世族之间的任何盟约，一定要稳住戎狄部族。

  3. 北阪痛歼牛头兵

    • 第一次见识到了什么叫真正的"痴人做梦"，什么叫真正的"阴毒老枭"，居然自己给自己的如意算盘起了个名号'穆公定国之变'；托穆公之名，引进戎狄，铲除新法，再将“杀戮乱国”的罪名加于戎狄而剿灭之！
    • 在此次造反变乱之中，嬴驷再次展现出了什么叫"虎父无犬子"，此等不慌不乱谋定而后动的遇事能力，令我大为佩服！嬴驷有意在此战中培植属于自己的新势力，司马错则是其重点培育对象。

  4. 咸阳老世族的最后时刻

    • 牛头兵和西部戎狄两股势力都被摆平之后，嬴驷终于名正言顺的"借为商君昭雪"之由成功的将矛头指向了 老旧世族 和 山东六国。
    • 嬴驷果断地做了两件事：1.铲除世族,诛灭复辟；2.以"逼杀商君"为由向山东六国宣战！

  5. 犀首(公孙衍)挟策入咸阳

    • 嬴驷在完成了"诛杀商鞅,平息戎狄,铲除世族,恢复民心"这一连环高难度斡旋之后，也算是真正体会到了"运用权力纵横捭阖"这种感觉的美妙。
    • 作为权力中心，身为国君的嬴驷只要用权有道，那么自己将永远是正义的化身！民心失去了也没关系，因为它还可以再拿回来，只要再给他们以真正的好处！
    • 嬴驷完成这一系列动作之后，才发现自己苦于无大才之人携手共创大业。于此同时诸子百家中却出现了新流派："策士流派"。该流派专一的钻研揣摩列国形势格局，游说诸侯，为自己所向往之邦国谋划王霸之策。他们不再专习一家学问，而是融诸子百家于一体；他们锤炼辩才，专注策士谋国。

+ - "第二章 山东雄杰"

  1. 洛阳苏庄的故事
    • 讲述苏秦的身世背景，家里原是世代行商，是洛阳苏庄苏亢的儿子。苏亢觉得自己身为商人，身虽富人却贱，暗下决心一定要让儿子们入仕做官。
  2. 双杰聚酒评天下
    • 苏秦张仪两师兄弟聚首洛阳苏庄，良马择主煮酒论天下。苏秦看中了变法后秦国之蓬勃发展朝气，张仪则选择了看似已如日中天的战国老强国-齐。
  3. 洛阳试剑 苏秦成名不成功
    • 苏秦听闻老父之言，出山后首战选在了洛阳周室。
    • 试图游说周显王采用自己"聚众抗强"之中兴大略，没想到周王却兴致全无，此时的周王自从自己上位以来种种举措受阻已经觉察到周室的衰败，放弃治疗放弃折腾了。
  4. 安邑郊野的张家母子
    • 视线转动出山后的张仪，张仪的家世也不是简简单单之辈，也算是有祖辈的荫蔽也算得上是世族了。
    • 张母对张仪晓以大义，让张仪出门去而游说天下，系家族兴旺于张仪一身，张仪也由此怀揣大义走上了自己的名士之路。首站：魏国大梁。
  5. 张仪第一次遭遇挑衅 (与孟子对骂成名)
    • 张仪入大梁见魏惠王，没想到恰遇孟子此时也在魏王宫殿。张仪本想献出自己的纵横之策，却没想到首先竟被魏王因着重而蔑视一番。其后又与孟子就到底哪家才是真正的经世之学进行了雄辩。
    • 【孟子与张仪的第一次对骂】孟子发挥自己一如往常的刻薄言语说纵横家不过是擅长机变以攫取高位的妾妇，而张仪也不堪示弱直接回之以"惶惶若丧家之犬的乞国老士子"，骂其不过是那卖来卖去人老珠黄却依旧是处子之身的"娼妇处子"，言辞之机警刻薄不愧是号称战国之口活最好的人之一。
  6. 函谷关外苏秦奇遇
    • 苏秦这边从洛阳王城回来之后，对周显王的昏聩有了新的看法，同时也更加坚定了自己去秦地谋求霸业的决心。
    • 临行前家人对其倚重并为其安排好了一切，及至秦地函谷关外却没成想被 楚姬 拦了下来询问是否有意到楚国去。
    • 苏秦说出了自己对楚国的看法: 燕国太弱且了无生气，不能成就王霸大业。楚姬因此也无意再邀，反倒是二人互表爱意畅谈了一夜。

+ - "第三章 西出铩羽 - 苏秦的秦地之行游说失败" $思考-利心互换

  1. 新人新谋弃霸统

    • 上将军 vs. 国尉
      • 国尉是负责处置日常军务的武职大臣，寻常时日：主要负责 征兵，训练新兵，筹备军资，打造兵器装备，统筹要塞防务等；战时则只是打仗部队的后方官署(相当于后勤部队)而已。
      • 上将军则是在战时才有的职位，一旦遇有战事上将军才是真正的军队统帅，其爵位与统摄国政的能力与丞相相等，可谓是真正的地位显赫。
      • 所谓 "出将入相" 就是指的 上将军 和 丞相 两职。
    • 司马错在自己不太喜欢的职位上所做出的选择是："转变思想(将之视为对自己的历练)，绝不懈怠自己的本职工作(军政工作-深究谋划)"！
    • 终于在自己的苦心奋斗下，超出国君的期望(不仅指出国家当前寻求突变的不利，而且为国君未来的发展建言献策)，获得了国君大大的喜爱和肯定。这种人怎么可能会不成功呢？(能屈能伸，脚踏实地且不好高骛远)！
    • 嬴驷参照并考核通过了司马错的建议，同时也就宣示着放弃了犀首那十六字图霸大策。
    • 犀首与嬴驷相互交心，之后犀首彻底被这个新君的个人魅力所折服了，同时也弄明白了秦之所以变法成功的核心原因："利心互换"！

    @思考："利心互换" 治国处世巨匠之终极大法："天下之要，一曰利，一曰心。轻利者必得大义，专利者必失人心。 "士为知己者死，无非知己者以利换之；鲍叔牙当年不慷慨，何来管仲之高义？周厉王若不专利，何得失国出走？商鞅变法能使愚昧蛮荒的老秦人也能在短短几年之内移风易俗，能使秦人死保拥护化解各种阻力，归其根本也在于："国让利于民，民衷心于国"。

  2. 关西有大都

    • 讲述苏秦第一次来到函谷关以西见识到咸阳时的震惊。今日之咸阳已经发展成为一个实至名归真真正正的西部大都了。在住店时与犀首意外会晤，两人就时下秦发展之对策发表了一些简单地言论。

  3. 夤夜发奇兵

    • 司马错开始了自己的长途奇袭计划：先奔袭房陵。在奔袭巴蜀。嬴驷为求稳妥将此策与上大夫樗里疾商量了一下，对该计划又有了点不放心。
    • 所以在司马错临行前又特意跑过来再询问一遍他的大计安排 以图安心。当此之时，苏秦也到了秦地，求见秦君要建言献策。

  4. 雄心说长策

    • 以"王霸新途"为名，献亲王以 一统中国 之长策。建策的内容多涉及 秦要灭他国,秦能灭他国,秦灭了他国统一中国之后要以一国之法来治国。却始终没有说到 秦王所最关心的问题：秦现在该怎么做？
    • 秦君与 犀首和苏秦这类名士 之间最大的分歧在于：秦君觉得你们这些人只会夸夸而谈，你们真的了解我大秦吗？不过如此而已~
    • 苏秦因为自己的建言没有触及秦君G点而受到冷落，自己也只能悻然离去。

  5. 命乖车生祸 - 屋漏偏逢连夜雨·初出茅庐首战完败的苏秦

    • 苏秦驾着自己的王车气呼呼的来到了秦国最豪华的渭风古寓，喝酒泄愤喝的酩酊大醉。却没成想自己的王车被富商 猗矛 给看上了。
    • 当自己离开古寓后被这些商人们带到了北坂小树林，先礼后兵的将其抢劫了。这时的苏秦也真算得上是倒霉悲催啊，不仅仕途不顺，而且没想到在秦地还会遭人暗算。

  6. 孑然一身出咸阳

    • 苏秦遭劫后回到客栈，没成想客栈也是世态炎凉，虎落平阳被犬欺。
    • 有钱时你是大爷，没钱时你就得遭人白眼，受人眼色。最终苏秦不得不将自己随身所带所有物件全数抵押给了客栈放得结账脱身。自己最终也就落得个'仰天大笑，踹开房门愤然离去'孑然一身灰溜溜的离开秦国了。

    @思考：纵然大才如苏秦，师从名家大师鬼谷子，出山之后也不见得就能混得顺风顺水，所以 不要以为自己出身名校，就有多么牛逼了，做人还是要脚踏实地，谨记：唯有能立得住事，方能立得住身！

+ - "第四章 谈兵致祸" 张仪的齐地之行没想到也因自己在兵事上的大意而夭折

  1. 十六字诀震撼了齐威王
    • 视线转移到 张仪 这边，张仪魏国受挫后，来到了自己的下一个目标齐国临淄。
    • 没想到张仪在齐这边的进展如此顺利，见到齐王然后凭借自己事先已想明白之'十六字霸业'长策："联魏锁秦，和秦敬魏，北结燕赵，南遏楚韩"！
    • 一下抓住了齐王的心，同时又献策谋亲自出马以解齐王燃眉之急-越国北征之祸，齐王大喜，当即以丞相位许诺之。

  2. 一席说辞 大军调头

    • 张仪 以天下征战已由争霸大战转为利市大战，每战必以夺取土地,人口和财货，而这些齐无有 为理由 智劝越王姒无疆 弃齐攻楚。
    • 越王被他说动了，不仅大为振奋，而且还许之以名剑-龙泉剑一把，以及越国未来之上大夫位一职。

    @由此看来：不得不说 张仪的口活 真的不是盖的：'骂倒孟子，征服齐王，玩弄越王', 这些都是张仪那一张利嘴再加之胸怀天下的才华成就的。

  3. 策士与君王的交换

    • 张仪说服了越军攻楚之后，又转而跑到楚国给楚君说了说越国的家底，然后说服楚国一举灭了越，还说这算是自己给楚的大礼。
    • 其实张仪是想百尺竿头更进一步，到楚国想 寻回田忌和孙膑 与自己一起回到齐国达成'张田孙'铁三角共谋大业。
    • 张仪面见楚威王要他同意放走田忌，楚王就势答应以 田忌帅楚君灭越 作为交换。张仪也只能同意了。

    @联系 '吴楚越' 三国之间的恩恩怨怨：( 楚 - 吴 - 越 - 楚 ) 话说楚平王时期，国君昏聩无道，竟夺自己长子太子建新婚之妻，太子傅伍奢据理力争被处以极刑，伍奢的两个儿子伍尚和伍员不得以逃到了吴国。当时吴国对楚国还是臣服关系，但吴王却看准了机会，只交出了伍尚却谎称伍员逃窜无着，实际委以重任让其秘密练兵。三年后，伍员化名伍子胥作为吴国将军率领大军复仇大败楚军，楚平王自此声名狼藉，竟在盛年之期被活活气死了。
    之后吴国内部也发生了政变，公子光刺杀吴王僚，自立为吴王，也就是后来赫赫有名的吴王阖闾。在吴王阖闾在位期间兵家名士孙武也来到了吴国，吴国拜孙武为上将军，孙武联手伍子胥奔袭楚国腹地，要不是秦国秦哀公发兵救楚，楚可能真的就亡国了。此后楚国也再不敢小视吴国了，扶植越国对抗吴国，后来没成想还是不敌吴国，楚国只得东迁都城暂时避难。
    吴国至此成为真正的江南霸主。之后吴王夫差灭了越国，越王勾践卧薪尝胆，无所不用其极的讨好吴王取得吴王信任(包括现两大美女-西施和郑妃)，最终得以有机会东山再起 恢复了越国又一举灭了吴国。至此越国成为了江南新秀，不仅不感激楚国的扶植，却也是不断找楚国麻烦。

  4. 云梦泽访出了逃隐名将

    • 张仪来到云梦泽以'共图大业'为名请田忌出山回齐国再创不朽之功业。同时也向田忌说明了楚王的交换条件，让其带兵灭了越。
    • 张仪谈及天下纵横策略才华横溢豪气云天，但谈及兵事却总令田忌心里觉得很不踏实。

  5. 昭关大战 老军灭越

    • 田忌出山先在楚国任 大将军 统帅楚君，在昭关打败越军。
    • 由于张仪在兵事上的不谨慎，没成想却丢了楚国最为重要的数百座粮仓府库房陵，秦军远走奔袭房陵一举拿下之。楚威王大怒，立即捉拿张仪，想要治以重罪，但此时田忌却站出身来为张仪扛罪，愿再率楚军收复房陵。楚威王这才放了张仪。

  6. 错也数也 不堪谈兵

    • 张仪没想到自己 '一谋之功，由人而神；一谋之过，由神而鬼'！真正的体会到了权力场上天使与魔鬼之间的一线之隔。
    • 张仪也认识到了自己 对兵家战事之学 只是皮毛尔耳，却轻率谈兵，犯了兵家之大忌，可谓成败皆由己一念而至。
    • 此时的张仪被这老秦从背后这么偷袭一下搞得颜面尽失，只想着算了算了，楚国齐国都回不了了，还是快点回家吧。

    @总结：至此 苏秦张仪 这两大鬼谷奇才的首战都以惨败告终！苏秦是因为自己缺乏对秦国君臣的了解，张仪则是功亏于自己在兵事上的大意，一子错满盘皆输。

+ - "第五章 天地再造" 中原乱了秦国却强了富了

  1. 异数中山狼
    • 苏秦初入秦地惨败，自己现在真是穷困潦倒，现在的他只有一个念头：不要见人，赶紧回家(回洛阳)。
    • 中山国依靠中山狼又被称为'狼国'，这中山狼于中山国而言无异于十万大军。它对别国而言是实实在在的灾难，匪夷所思的天地异数。老猎人对苏秦穿越河西高原给了一个生存诀窍：'有树上树，无树钻洞，无洞无树，便装死'。遇到中山狼切记：打狼绝无胜算，只有装死。中山狼从来不吃死物的。
    • 苏秦在即将离开河西高原之际，无奈身体太过困顿便大意的就地睡下了，没想到自己却遇到了中山头狼-白毛老狼。
    • 情急之下苏秦大喊引得秦军诱狼小队注意，并既之喊来秦军大歼中山狼群于河西高原之上。不得不说秦军真的是够胆气够义气。
    • 救了人之后，还想要举荐他，却又是不图任何回报，这个千夫长甚至是连自己的姓名也不想告知，人家纯粹就没把这事当成救你命的大事。我觉得苏秦这次真可谓又被秦国军民那不图回报的忠肝义胆给折服了。
