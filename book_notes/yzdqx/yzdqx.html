作者： [美]布莱恩·格林


超弦，隐藏的维度 及 对宇宙终极理论的探索。
这本书解释了爱因斯坦，海森堡以来的主要物理学成就，超弦理论。

超弦理论撒开了一张大网，它是一个深广的主题，它融合了许多重要的物理学发现，这个理论统一了大与小的定律，

大到统领宇宙的尽头，小到深入物质的核心。

    ========= PART 1: 知识的边缘 ===================
    Date: 2015-11-02
  • + - 第一章 绷紧的弦
    •     =  现在物理学的两大支柱：爱因斯坦的相对论[大尺度宇宙]，量子力学[微观宇宙]。

              但是从广义相对论和量子力学今天的形式来看：它们不可能都是正确的。

          = 物理学史上的三次大冲突
              Conflict 1. 牛顿运动定律预言可以赶上或超过光速 vs. 麦克斯韦电磁场理论预言谁也跑不过光
              Solution1. 爱因斯坦“狭义相对论”(1905)，推翻了原有的时空观。
              Conflict 2. 狭义相对论预言任何物体都不可能跑过光 vs. 牛顿引力理论却可以使得物体的瞬时速度会超过光速
              Solution2. 爱因斯坦“广义相对论”(1915)，时空会随着物质和能量的出现发生弯曲。
              Conflict 3. 广义相对论 vs. 量子力学
              Solution3. 弦理论的提出(1980s) 与 发展。
          
          = 自然界中的四种力：引力，电磁力，弱力 和 强力。     
              引力取决于物体的质量；电磁力取决于粒子所带的电荷量；
              强力：是一种将上夸克下夸克组合成质子和中子，又把质子和中子塞在一起组成原子核的这种强大的核力。

          =  弦理论

              基本思想：物质由原子组成，而原子由夸克和电子组成，根据弦理论，所有这些粒子实际上是振动的一根闭合的弦。

                                万物在最微观的层次上是由振动的“绳子”组合在一起的。

             弦理论的弦 在共振频率处的振动，会产生粒子，而粒子的质量和电荷由弦的振荡行为决定。

                                就目前来看，把广义相对论和量子力学和谐的统一起来，是弦理论的主要成功。     


          = 相对论：是关于时空和引力的基本原理，依据研究对象的不同分为狭义相对论和广义相对论。
                           相对论的基本假设是“相对性原理”，也即物理定律与参照系的选择无关。
             狭义相对论：讨论的是匀速直线运动下的参照系(惯性参照系)下的物理定律。[爱因斯坦相对性原理 && 光速不变性原理]
             广义相对论：推广到了具有加速度的非惯性参照系，它发展了牛顿力学，推动物理学发展走向一个新的高度。
             量子力学：是研究微观粒子的运动规律的物理学分支学科。
                               量子力学的基本原理包括 量子态的概念，运动方程，理论概念和观测物理量之间的对应规则和无力原理。

  • + - 第二章 时间，空间 和 观察者的眼睛
    •     = Conflict 1 牛顿和麦克斯韦：

              根据麦克斯韦的电磁场理论，光总是以光速运动，它永不停歇也永不减缓脚步；但是按照牛顿的运动定律，如果我们以光的

              速度来追光，光相对于我们就相对静止了，光就好像停在了那里。这就引起了一个问题--谁也不曾抓一把光在手里。     

             Solution 1 最终爱因斯坦于1905年用 狭义相对论 解决了这一冲突，同时也颠覆了我们对时间和空间的概念。

              狭义相对论认为：不同观察者所看到的现象是不同的，而这又蕴藏着微妙而深刻的意义。     


          = 直觉和错觉
              时间膨胀(时间上)    洛伦兹收缩(空间上)
          = 相对性原理
              牛顿发现：运动是相对的，爱因斯坦发现：时间也是相对的。
              所有的观察者都同意：光是以每秒30 0000km的速度在运动，光的速度是不变的(相对于任何参照系)。
                  -“光速是不变的”这一事实已然改变了我们已知的宇宙观！
                  - 爱因斯坦发现：不变的光速意味着牛顿物理学的崩溃。
          = 事实和结论
              光的存在，不仅是对 我们寻常对速度这一概念 的挑战，它实际上也在挑战着我们原有的时空观。
          = 时间的同时性
          = 时间的膨胀
              使用理论上的“光子钟”来进行解释。
              运动的物体，速度越快，所经历的时间变慢了。从静止的观察者的角度来看，可能他已经经历了1年的时间，
              而这个高速运动的物体却只经历了1个月，甚至是更短，所看在他看来，高速运动的物体时间是膨胀了的。
      Date: 2015-11-04 
         = 运动的生命【54/433】
              运动者的生命会比静止的生命活的更长，运动者的时间慢下来了。
              其实不是人的寿命延长了，而是来自微观世界的某种粒子(μ子)的寿命延长了。【联想电影《星际穿越》里面的情景】
             【但是并不是说寿命延长了人可以干更多的事，因为运动者的时间变慢，同时他做任何事也会变慢。】
          = 谁在运动
              又是一个由相对性原理导致的逻辑困惑。
          = 空间的收缩
              运动不仅使得物体所经历的时间变短，同时它在空间上也变短了。
              观察者会发现 运动的物体在运动方向上缩短了。狭义相对论的方程已经证明，如果物体以90%的光速运动，
              那么静止的观察者将发现他比静止时短了80%，也即：只有静止时长度的20%。

          = 在时空里运动
              狭义相对论还提供了一个更为深刻的统一观点，它能够囊括所有这些运动时的现象。
              爱因斯坦狭义相对论 惊人的物理学事实基础——运动不仅能够在空间维上尽心分解，而且 时间维上也能“分享”运动。

              重要现象：当物体相对于我们在空间运动时，它的钟比我们走得慢，这就是说：它在时间里的运动速度变慢了。
              爱因斯坦由此宣布：宇宙间的一切事物总是以一个固定的速度(光速)，在时空中运动。

              当我们以四维时空的视角来看时，物体在空间运动，原先在时间里的部分运动转移到了空间，这样，在原始时间轴
              上的运动速度就会降低，所以时间走的就变慢了。


  • + - 第三章 卷曲与波澜
    •     = Conflict 2: 狭义相对论 与 牛顿引力理论之间的冲突
              Solution 2: 广义相对论，再一次革新了我们对时空的概念，时空是卷曲的，而引力就是那卷曲的波澜。

             冲突详情：
                  狭义相对论认为，任何物体包括各种信号或干扰从一个地方传到另一个地方，都不可能比光速更快，所以它们的
                  传播总会有一定的时间上的延迟；
                  而牛顿的万有引力定律，则说明引力只与两个物体的质量以及他们之间的距离相关，而与时间无关，所以一旦
                  物体的质量或距离变化了，那么它们之间的引力也就会立即发生变化，这种变化是瞬时的，不会有任何时间上的
                  延迟。所以两者之间就矛盾了。

          = 牛顿的万有引力理论
          = 牛顿引力与狭义相对论的不相容
              狭义相对论的一个重要特征：光所限定的绝对速度，任何物体的速度都不可能超越光速。
              这导致了牛顿引力与狭义相对论之间的不相容。

          = 爱因斯坦将 加速运动与引力的不可分辨 的性质称做为 “等效原理”，这在 广义相对论 中起着核心的作用。
              爱因斯坦用旋转的环的加速度来模拟引力。
              得出结论：在所有的加速运动的情形中，时空都是弯曲的。

          = 广义相对论基础
              当没有任何物质或能量存在时，空间应该是平直的。但当有了质量和能量的存在之后，时空就会发生弯曲。
              爱因斯坦 同时也确定了引力的传播机理：空间的弯曲。

              质量牵着空间，告诉空间如何弯曲；空间牵着质量，告诉质量如何运动。
              
              想象当太阳爆炸的时候，大约在8分钟之后，地球上的我们才知道太阳灭了，而在这同一时刻我们也会
              瞬间感到来自太阳的引力的消失，而在这8分钟之内，地球上的我们完全不可能知道太阳已经发生了变化，
              这是由于引力扰动与光同步，他们一起以光速在宇宙中传播。

      Date: 2015-11-06
           = 再谈时间弯曲
              质量不仅会导致空间的弯曲，也会导致时间的弯曲。引力越大，时间也会变得越慢。
             【这也可以联想电影《星际穿越》里面在一颗引力很大的行星上呆了只有几分钟，当再次回到飞船上时发现
                已经过去了几十年 (这种时间变慢就是由引力导致的，而不是速度导致的)。】

              当我们讨论黑洞时，由于引力场过于强大，所以在这里 时间和空间 都失去了意义。

          = 黑洞，大爆炸和空间膨胀
              狭义相对论：在高速运动的情形下 会显著的表现出来；
              广义相对论：在物体质量大，空间和时间相应地弯曲剧烈时就能发挥自己的作用。
          
              黑洞(或者是质量很大的横星) 可以被用作时光机器，因为在这附近存在着巨大的引力场从而导致时间变慢了，
              我们在这里呆一段时间后再回到地球，会发现地球上已经过去了很多年，我们穿越到了地球的未来。

  • + - 第六章 万物都是音乐：超弦理论的基础
    •     超弦理论(或简称为"弦理论") 革命性的修正了我们对宇宙超微观性质的理论描述，而且这修正也正是我们想要的，
          它使得爱因斯坦的广义相对论与量子力学完全相容了。
          根据弦理论宇宙的基本组成要素不是点粒子，而是上下振动着的一堆丝线(弦)，根据该理论，弦 是构成原子的粒子
          的超微观组成元。这些弦的长度大约就是 普朗克常数长度的尺寸。
          【联想电影《蚁人》中男主角所进入的那种次原子级别的空间之后所看到的原子核的超微观组成。】

      Date: 2015-11-08
          =  弦是做什么的？

              它是真正基本的东西。

              一根基本弦的不同振动模式可以生成不同的质量和力荷。

              一个粒子就是一根弦，不同的基本粒子实际上是在同一根基本弦上弹出的不同“音调”。     

          =  基本粒子

              基本粒子 已经可以用 “一根根振动的弦” 来理解。

              在亚原子尺度上，量子概念取代了经典逻辑。


          = 如何调和 广义相对论 与 量子力学之间的矛盾？

             从某种意义上来说，假想的普朗克尺度下汹涌的量子波浪是不存在的。

             也就是说：弦理论指出了那种导致相对论与量子力学相互矛盾的 假想的普朗克尺度以下的微观空间 实际上是没有的，

                               从而平息了疯狂的量子颤栗，调和了广义相对论与量子力学之间的矛盾。

              十一维的宇宙空间，其中有10个空间维和1个时间维。

          = 看不见的卷缩起来的空间维度 的物理意义
              隐藏维度 的几何 决定着我们在寻常三维展开空间里所观察到的那些粒子的基本物理属性：比如质量，电荷等。

              囊括四方的框架将这一切都包容了进来，这种框架就是“M 理论”。

              一种科学方法：微扰方法，先给出一个大致的估计然后再对这个估计进行一点点的微调(微扰)，从而使得我们原先

  • + - 十三章 从 弦/M 理论看黑洞
    •     = 黑洞 与 基本粒子
              黑洞和基本粒子从某种角度来看，并不像我们想的那么悬殊。
              所谓“黑洞无毛”，一种看法就是除了少数几个可以区别的特征外，所有黑洞看起来都是相像的。
          
              黑洞和基本粒子 原来是同一弦物质的不同相，弦理论在 黑洞和基本粒子之间架起了桥梁。
          
          = 黑洞熵
              熵是衡量无序和随机的量度。高度无序意味着高熵。
              热力学第二定律：事物都是朝着更加无序的状态演化，熵增原理。
              
              黑洞质量越小，它的温度越高，从而辐射也越强；黑洞质量越大，它的熵也就越大，同时也就
              代表着它所包含的无序是巨大的。但是这种熵(无序)到底是什么导致的呢？这个问题一直困扰着
              很多人。知道使用弦理论给出了黑洞的微观组成以及其相应的熵。
          
          = 法国拉普拉斯所提出的 “决定论”：
                  理性 能认识某一时刻所有令自然洋溢生机的力 和 它的存在物的状态，如果理性足够强大，可以
                  将那些数据拿来分析，那么它能将一切运动，从宇宙中最大的物体到最小的原子，都包含在同一
                  个公式里。对这样的理性来说：没有什么不确定的东西，将来与过去一样，它都能看得见。
              换句话说：
                  如果我们知道宇宙每个粒子在某一刻的位置和速度，我们就可以用牛顿运动定律或其他的定律
                  来确定它们在过去或未来任何时刻的位置和速度【也就是说这一切都是注定的】。

              这种宇宙观使得哲学陷入了关于自由意志的泥潭。但后来还是量子力学中的不确定性原理缓解了
              这种一切由天注定所带来的尴尬局面。

          = 黑洞中的未解之谜
              1. 黑洞中的信息是否真会丢失，是今天理论物理学的一个主要研究问题。
              2. 黑洞中心点的时空本性问题。
