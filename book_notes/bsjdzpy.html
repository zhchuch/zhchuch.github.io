把时间当作朋友

作者：李笑来

  • 运用心智，获得解放。

    改变人生命运的无非两样东西：一个是你读过的书，一个是你遇到的人。

    最节省时间的方法：学习；

    效率低下的根源：回避困难；

    提高效率的最有效方法：克服拖拉；

    最浪费时间的句型："要是 .... 就好了！"

    自卑 是浪费时间的罪魁祸首之一。

     

    第 0 章 所有人的困境

    • 总是感到"时间不够了" (时间恐惧症)

      = 我最初发现，所有的学生提问都是源自于所有人共有的弱点：懒惰！

      • 后来发现，这些学生每天坚持上课等等，表现出来的又不是懒惰之人，他们真的是 "既勤奋又懒惰"，

        后来我明白了，这种奇怪现象来自于他们的“时间压力”感受，总是有 "没有时间了" 或 "时间不够了"的感受和恐惧，使得他们超乎寻常地勤奋，而这种同样的感受，也使得他们终日寻找捷径，最好是能不费吹灰之力，所以又养成了他们的懒惰。

        长期来看，我们都是要死的。可是，明显有一些人，尽管数量上并不是大多数，在用另外一种生活状态。他们从容而又优雅。他们善于化解各种压力，安静地去做他们认为应该做的事情，并总是有所成就。他们最终甚至可以达到常人所无法想象的境界——不以物喜，不以己悲。

       

      相信我，你并不孤独。

      要管理的不是时间，而是我们自己。

      人生的幸运在于能够“用正确的方式做正确的事情。”而什么是正确的或者更好的方式，什么事情真的值得去做，需要培养出良好的心智才能做出尽可能准确的判断。

      人生是一场马拉松比赛，胜者不一定是跑得最快的。

      硅谷是创新的沃土，也是全世界精英的汇集地。以我个人的体会，我非常鼓励每个年轻人能花上几年时间到国外走一走，学一学。

      教育的根本其实只不过是 "见多识广"。然后再用5-10年时间定义一个属于自己的人生方向，为之奋斗，为之坚持。

      一切都靠积累，一切都可提前准备，越早醒悟越好。做人要勤奋，做人要执着，但做事不一定要图快。得按你想的去生活，否则，你迟早会按你生活的去想。

     

    + - 第 1 章 心智的力量

    • = 不管什么方法，都要通过实践才能获得效果。有没有用，只有通过实践才能得出结论。

      = 不知道学习有什么用，有的人就选择放弃了学习；但是还有一些人心里在想：正因为我不知道学习有什么用，它才可能变的更有价值。

      = 有的时候换一个环境，意味着放弃过去所有的积累而彻底重新来过。

      = 兴趣真的那么重要么？

      • 人们常常抱怨对自己现在从事的事情并不感兴趣，而对另一些事情充满兴趣。他们觉得自己正在做自己不喜欢做的事情。

        然而，真的是他们以为的那样么？事实上，首先，这些人并不是对正在做的事情没有兴趣，而是没能力把目前正在做的事情做好。

        最终：没有人喜欢自己做不好的事情。

        往往并不是有兴趣才能做好，而是做好了才有兴趣。

        没兴趣，往往只不过是结果而已(证明你在这件事情上是loser的结果)，如果你把它当做不去做好的理由，最终的惩罚就是大量的时间白白浪费！

      = 方法真的很重要么？

      • 所有学习上的成功，都只靠两件事：策略和坚持。而坚持本身就应该是最重要的策略！
      • 所以“坚持就是胜利”这句话不无道理。相对于坚持，方法有多重要呢？哪怕说是不重要到几乎可以忽略不计的地步，其实也并不是特别过分。

     

    + - 第 2 章 开启自己的心智

    • = 一个人的心智是指：他的各项思维能力的总和。
      • 当一个孩子被夸奖为 "聪明的孩子"的时候，应该被夸奖的很可能或者说更可能是这个孩子的父母。
      • 拥有 "更聪明" 的父母的孩子，在随后的成长中每时每刻都有更多的机会耳濡目染并学习模仿 "更聪明" 的行为，而最终使得自己变得更聪明。
    • = 用自己的大脑 控制 自己的大脑。

      • 一个人的智商很高，可以被形容为 "聪明"；而一个人的心智能力非常强的时候，我们往往用 "睿智"来形容。

        小聪明(智商很高) 与 大智慧(心智很强)。

        这个世界上的事情或道理，大抵可以分成以下四类：

        1. 实际上是正确的，我们也认为是正确的；(安全的决策)

        2. 实际上是错误的，我们却认为是正确的；(认知发生错误)

        3. 实际上是正确的，我们却认为是错误的；(认知发生错误)

        4. 实际上是错误的，我们也认为是错误的。(安全的决策)

        由以上这四类可以看出——我们的认知不仅可能也确实往往会背离现实。所以我们需要反省机制。

    • = 控制自己的大脑
      • 得按你想的去生活，否则，你迟早会按你生活的去想。
      • 无意识的受大脑控制【直接根据自己的感觉做出反应】 vs. 有意识的控制大脑【用自己的意识作用于大脑，和自己的感觉一起共同决定正确的反应，并在得出反应之后反思一下再去做】
    • = 控制自己的情绪

      • 先说如何控制痛苦，在正常情况下，我们对于痛苦往往并不十分了解。最常见的误解就是：当我们觉得自己痛苦时，总是不自觉地把自己想象成全世界最痛苦的人。这也是非常自然的，因为我们自己的痛苦是亲身感受，而别人的痛苦我们做不到感同身受，所以，我们当然会觉得我们自己最痛苦。

        认清这种关系，让我们知道：我再痛苦，在目前这种状况下，肯定还不是最痛苦的人。

        世界上比较痛苦失意的人举例：阿伯拉罕·林肯。

        --------------------------------------------------------------- 阿伯拉罕·林肯 的经历 ------------------------------------------------------------------------------------------------

        有这么个人。在1832年，他失业了，这显然使他很伤心，但他下定决心要当政治家，当州议员，糟糕的是他竞选失败了。在一年里遭受

        两次打击，这对他来说无疑是痛苦的。他着手自己开办企业，可一年不到，这家企业又倒闭了。在之后的17年里，他不得不为偿还企业

        倒闭时所欠下的债务而到处奔波，历尽磨难。他再一次参加竞选州议员，这次他成功了。他内心萌发了一丝希望，以为自己的生活有望

        了："可能我可以成功了"。第二年，即1835年，他订婚了，但是离结婚还差几个月的时候，未婚妻不幸离世。这对他的精神打击实在太

        大了，他心力交瘁，数月卧床不起。在1836年，他还得了精神衰弱症。1938年他觉得身体状况好些，于是决定竞选州议会的议长。可是

        他又失败了。1843年，他又参加竞选美国国会议员，但是这次仍然没有成功。直到17年后的1960年，历经更多的磨难和失败之后，这个

        人当选了美国总统，他的名字是阿伯拉罕·林肯。可是他的噩运并没有结束，1865年，他被暗杀了！

        我们的大脑具有 遗忘痛苦 的功能。

        要知道，你所面临的所有尴尬，最终肯定有一部分原因是你自己造成的。所以，没有必要找借口，没必要抱怨别人，没必要觉得这世界

        对你一个人不公平。要记得 "你并不孤独"，肯定有别人也在不同的地方，不同的时间遭遇过同样的尴尬和痛苦。

        在做所有的必须记住大量信息的工作(比如：单词或诗词)的时候，一定要想办法由衷的把这件事当做快乐的事情来做。因为如果你觉得

        这件事是痛苦的，大脑就会把它列入遗忘名单。

        而已)。想到这里，背单词也会变成一件特别快乐的事情，而且也不会轻易被大脑遗忘。

    + - 第 3 章 通过管理自己 与时间做朋友

    • = 许多人都曾不由自主地产生过自杀的念头，所以请你记住 "你并不孤独"。

      = "事件-时间日志" (event-time log) —— 时间账本

      • 这种基于过程的记录，不仅更加详尽，还有另一大好处就是——当遇到不好的结果的时候，更容易找到缘由。
      • 通过这种记录，可以让自己更加精确的感知时间。

      = 最好的工具——纸笔

      • 但是这种方法最大的敌人就是“半途而废”，大多数人连一次“坚持到底最终成功”的经验都没有。

        好记性不如烂笔头，克服懒惰进而导致的遗忘的最有效方法往往也是最简单的方法就是：用纸笔记录下来，让自己随时都可以看到。

        养成一个新的习惯是很不容易的，一般都是在五年左右的时间。

        每天把自己做的事情，时间开销记录下来，一方面可以培养自己的成就感，另一方面可以避免轻易的原谅自己(避免轻易遗忘痛苦的记忆)。

        如果你坚持记下来，哪怕你不去翻阅，只须看着本子边缘变得稍微有点黑的部分越来越厚，都会很有成就感；反之，如果你不记录，仅凭

        那靠不住的记忆，是不可能有这种实在的感觉的！

      = 养成记录你的时间开销的习惯吧！

      • 开始记录时间开销。

        如果你养成了每晚睡前用纸笔 记录你的时间开销的习惯，那么可以保证你在第二天早上一醒来就会为新的一天进行着有意义的规划！

        这样你每天的生活中就会增多许多有意义的事情。

      = 如何判断一项任务是否真的重要 —— 主要就是看这项任务的完成是否 对你的目标达成 确实有意义(有益处)。

      • 往往审视一下自己的生活，就有可能发现自己常常仅仅因为一件事非常的有趣就去做，那不去看这件事到底有没有用。

        ** 真正重要的任务只有一个：那个真正对你的目标实现有帮助的任务。

       

      = 在参加一个会议之前，应该【准备一个列表】记录自己应该提出的种种设想，这是合理的。

      • 这个时候列表可以让你表现的更加自信。
      • 最好的列表记录工具仍然是 纸和笔。

      = 制作列表最重要的一点就是：一定要随手可及[自己要能够随时看到，让列表来提醒我们]，否则列表将会失去意义。

      = 一事无成 的最根本原因就是因为放弃。放弃的方法有很多种，最常见的一种就是 "换一个更好的方向试试"。

      • 有一个很好的比喻：走向成功的过程大抵上就像是 你的起点在南极，而成功路径的终点在北极。
      • 那么，无论你往哪个方向走，只要中途不改变方向，最终就会到达北极。
      • 但是，如果你中途改变了方向，甚至经常改变方向，你就几乎无法到达北极，甚至可能会返回到起点。

      ☆ 计划成功的前提：目标现实可行。成功的定义就是：达到预期目标！所以目标一定要现实可行，才有可能成功。

      ☆ 如何判断一个目标现实可行：1.已经有人做到了；2.我与那个人没有太大差距。

      ☆ 时间的浪费，往往是因为：1.目标不现实 或者 暂时尚不可行；2.为了达到目标而制定的实施策略有误。

      大多数人都知道 "凡事要三思而后行"，但却不知道要具体思考些什么，而以上者两条标准就是最值得思考的问题，它也许可以帮你省下来不少的时间。

       

      = 一般来说，越是短期的目标，越容易清晰，而越清晰的目标就越容易实现。

      • 另外 计划固然重要，但是行动更加重要。行动是改变自己的众多方法中最有效，最直接的方法。

        很多时候，只要你开始行动，哪怕事先没有计划，也往往会有所收获。

        如果你在坚持做一件事情的时候，其间有不愿意去的想法，记住这只是你大脑的想法，而不是你自己的想法。

        你要做的就是控制你的大脑，而不是被你的大脑所控制！只要意识到这个，就可以坚持的做下去。

        失败并不可怕，因为你总要失败许多次才会有结果，并且又不是全天下只有你一个人失败，所以不用怕；

        真正可怕的是失败之后不能吸取教训，如果能够吸取教训，然后做自我调整，那就是进步。

     

    + - 第 4 章 开拓我们的心智

    • = 获取知识 几乎是我们开拓自身心智的唯一手段

      • 获取知识的途径：体验、试错、观察、阅读(最重要的手段)、思考。

        鲁迅说过："第一个吃螃蟹的人是令人敬佩的，第一个吃过螃蟹的人也吃过蜘蛛，因为两者外形极为相似，只是他觉得螃蟹的味道可口而蜘蛛的味道不可口"。这种知识的获得就是通过试错。

      = 本科教育的根本应该在于培养学生的自学能力！

      • 理论上讲，本科毕业之后，学生应该有能力自学自己需要的任何科目！
      • (所以在公司任职之后，上司对给你一大堆资料，应该学会自学，不要抱怨！因为抱怨是无助于改变现状的，只会显得自己更low)
      • -> 阅读理解能力：是自学能力的基础。
      • -> 检索能力：当我看到很多本科毕业生甚至研究生不善于用图书馆资源，不会用搜索引擎，而仅仅因此就无法找到实际上唾手可得的有效信息的时候，无法不感到悲哀。
      • -> 写作能力：在自学能力中占据重要地位！文字还有更多的责任——传递信息，积累经验，共享知识等。
      • -> 实践能力：是自学能力真正体现出价值的根本。(比如学英语，很多人只知道背单词，却不知道去用英语。很多学生没有进步本质上只是因为懒惰，他们在第一步就已经放弃实践了。)
      • ☆ 如何检验自己的自学能力？

        • 完全靠自己学一样东西吧，管它是什么。并且一定要学好，学到比相当数量的人都好，如果你真的做到这个地步，那，你不仅已经是一个完整的人，还是一个相当优秀的人。

      •  
      • 学习成果与所花费的时间，往往不是线性关系，而是一种由量变到质变再到量变再到质变的过程(所以不要期望在量变的过程中就产生较大的进步，这是不现实的)。

       

      = 对于老师

      • 注意：不要因为讨厌老师而拒绝学习。

        • 像我之前就是太害怕英语老师，导致我一直持续到现在英语都是拖我后腿的东西，这是非常吃亏的。

        记住：无论如何，永远不要也不应该用别人的错误来惩罚自己，这么做不仅不对，而且很愚蠢。

        ☆ 不应该过分依赖老师，往往越好的学生对于自己的老师更加无所谓(独立于老师)。

        对待一位位的老师，就像去书店买书一样，怎么可能买回来的每本书都同样精彩呢？就算那些精彩的书中，又怎么可能都一字千金，字字珠玑呢！

        事实上，买回来的一本书里，哪怕有一句话能给我们带来惊喜，带来思考，带来改变，那么就已经值回书价了！

         

      = 通过改变态度来改变世界

      • 对待考试：考试意味着要么是得到"通行证"，要么是变成"拦路虎"。

        我们每个人都喜爱拿到"通行证"，所以我们要热爱考试，热爱考试的理由很简单——获得通行证，考试意味着机会，这是其他没有此通行证的人无法获得的机会。

        大一大二的时候，所有学校里的考试都没有显得那么重要，如果你有一个这样的成绩单：

        • 大一各科平均70分(意味着你参加了更多的社会活动)，大二各科平均80分(意味着你在努力)，大三各科均分都在85分甚至95分(意味着你对这个专业掌握的比较牢固)，那么，你在申请留学的时候将获得极大的优势。

        很重要的考试，就是那些终将对你来讲成为通行证的考试，比如：高考、考研，托福,GRE(意味着留学通行证)，

        对待这些考试，不要想着智取、走捷径，其实很多重要的考试本质上就是看你准备了多久，准备的有多充分。

         

        学习上的终极技巧就是——把你学会的教给别人。教是最好的学习方法。

        记住：教别人等于自己也在学，只有学好的人才可能教会别人。另外，随着时间的推移，你在知识上不吝分享的经历，最终会让你明白这是最好的助人为乐的方法，并且获得的永远都是尊重！(想想我大三时怎么当上班长的，就是因为这一条！)

      = 耐心培养记忆力

      • 记笔记：好记性不如烂笔头！
      • 在任何情况下，无论是听课还是参加会议，或是进行研讨，记笔记最起码的几大好处：1. 可以使自己保持参与状态； 2. 提供一个完整的捕捉灵感、疑惑的机制； 3. 可以用来与其他参与者沟通，讨论正确的信息。
      • 最实用的电子产品：录音笔 —— 可以用来从容的整理清楚笔记。

     

    + - 第 5 章 小心所谓成功学

    • = 成功的定义

      • 现在有很多"成功学"的核心缺陷就是错误的理解了成功的定义。
      • 幸福感来源于两个方面：一种是需要与别人比较所得到的幸福(这种建立在比较的基础上的幸福往往都会使自己感觉到痛苦)，另一种则是不需要比较就可得到的幸福(比如：寒冬里晒太阳所体会到的幸福等)。

      = 不要相信 "机不可失，失不再来"

      • 天下所有的骗子下手的时候，第一步就是想尽一切办法让受害者相信"机不可失失不再来"，而受害者一旦相信这个说法，就会在这种想法的压力下做出非理性的选择。
      • 请记住：当年没有准备好的时候，对你来说，是不存在任何机会的。机会时时刻刻都会出现在你身边，关键在于，你有没有足够努力，有没有做好准备。
      • 关于一个人的财富取决于他的 "出身、运气和努力"，而对这个人而言 "努力" 最终可以改变他的 "运气"，进而改变他们的孩子的 "出身"问题，所以 "努力" 是很重要的。

      = 真正的朋友意味着"公平交换"的关系

      • 建立人脉交朋友也要注意到这点，不要成为一个"索取者"，这样就不会形成真正的朋友。人脉固然重要，但是对于某个具体的个人来说，更重要的是他自己所拥有的资源。
      • 这些资源包括：金钱，地位，名誉 (这些更多取决于出身和运气)；才华，知识，学识 (这些则可以通过努力来获得) 等。
      •  
      • 关于建立人脉的建议：
        • - 专心做可以提升自己的事情，学习并拥有更多更好的技能，使自己成为一个值得交往的人。
        • - 学会独善其身，以不给他人制造麻烦为美德(这也就是素质的体现)，用你的独立赢得尊重。
        • - 真正关心一个朋友的意思是说，你情愿在他身上花费甚至浪费更多的时间。

      = 人不可能以百分百的效率工作 (要学会做一个正常人)

      • 在做计划的时候切忌把时间排的太满。除了工作，学习之外，一定要保证自己有足够的其他活动。
      • 我常常劝大学生一定要在大学毕业之前轰轰烈烈地爱一场，或者至少偷偷摸摸地恋一次，给自己一些工作外的其他活动。

      如何判断自己是不是自己大脑的主人？

      • 可以找一些无趣但是很重要的事情，并且在未来的一个月里(或更多)能够做到每天坚持做下去，每天专注于这件事儿不低于2小时。
      • 在这两个小时内，也要做到有张有弛，可以划分成6个时间段，每段20分钟；在每段时间内专注完成之后，给自己放松一下。
      • 这样原本120分钟的预计时间，就需要规划出150分钟左右的时间开销，这个方法非常简单，但是确实也非常有效。

      = 意外总是发生 绝对不是因为你的运气格外差，而往往只不过是因为你考虑的不够周全！

      = 睡眠时间

      • 新生儿                20 h/Day

        <= 3                        14-15 h/Day

        <= 6                        12 h/Day

        15 <=  18        8 h/Day

        >= 18                        6-7 h/Day                (也就是说我这个读研的年龄，每天只需要睡 6.5 h就足够了，而且是最健康的生活方式。) [ 01:00 - 07:30 ]

        不要盲目的试图减少睡眠时间；

        尽量不要减少与家庭成员和亲属交流的时间；(尤其是一家人生活的比较离散的时候，你要知道哪怕是偶尔打个电话问候一下，你的家人都会很开心——很多时候说的究竟是什么都可能并不重要。)

        最好不要放弃你的社交时间。

      = 真正的朋友是等价交换

      • 在你需要别人的时候，别人在陪伴你的时候就要花费他的时间，人家为什么会心甘情愿地花时间在你的身上呢？
        • - 你要学会不停的帮助身边人；
        • - 你要想尽一切办法运用你自己的资源为值得帮助的人提供帮助；(记得一次李建求我帮他一个忙，我推脱了，导致关系就疏散了！)

      = 不要浪费时间去"证明自己"

      • 如果你是出色的，不需要你去证明自己，别人自然会看到；如果你去证明了，可能会招致不必要的麻烦(比如引来小人的嫉妒)；
      • 如果你是平庸的，也不需要你去证明自己平庸，别人也会看到；如果你去证明了，在别人看到的你就是一个小丑而已。

      = 如何看待 "时间就是金钱" 的问题

      • 越是收入高的人越是吝惜自己的时间；越是收入低的人越是不爱惜自己的时间。
      • 每个人的一天是24小时(86400秒)，你可以算一下自己的时间究竟可以标价多少！这就是你决心不再浪费时间的最有效的起点和动力。

      = 被低估了的今天

      • 给你两个选项 "a.今天给你1000元 ；b. 一年后的今天给你1000元"，你会毫不犹豫的选 a；如果b选项换成2000元，你可能还是会因为

        经验而选择 a选项；但是如果b选项变成了10000元，那么没想到的是还有人会选 a 。这就是低估了未来回报的价值！

        由此考虑 "大学生利用课余时间打工"的问题实际上也是基于这种对于未来回报的价值的低估，实际上如果家庭条件允许，大学生

        尽量不要把自己宝贵的青春时间花费在 "打工" 上，因为，在大多数情况下，实际的回报实在太少(相对于把这些时间精力花费在

        恰当的学习上而因此在将来可以获得的回报来看)。

      当你遇到一些看似无趣但是很重要的事情的时候，你的大脑通常会告诉你还有时间不要急，这就是你做事拖拉的原因(你被你的大脑控制了)；

      如果你想明白了要控制自己的大脑，突然会发现自己比身边的人多了一个玩具，而且是玩不腻的玩具 —— 你的大脑。

     

    + - 第 7 章 从此时此刻开始改变

    • = 大多数事情都需要提前准备，也都可以提前准备。认识到这一点本身就几乎是改变一切的起点。

        这样的话，你为了能够按时提交材料，起码要在今年6月份前考完GRE的作文考试，并且在10月份前后考完GRE的其他部分，

        托福考试，最迟也要在今年10月份考完，当然，比较理想的是在今年年初就已经拿到成绩。

        而托福也好，GRE也好，根据多年的教学经验，我坚定的认为，一个拿得出手的，能够换得来名校高额奖学金的成绩，需要

        至少两年的时间进行认真准备，并且还需要在此之前有相当扎实的基础(这也就是为什么北大清华的毕业生可以比较容易的

        就拿到世界名校的Offer的原因)。扎实的基础+多年的准备！

        万事皆可提前准备，甚至不夸张的讲：男人的性能力也可以提前准备。【所谓性能力往往与心肺功能紧密相关。】所以应该坚持每周至少长跑三四次！

       

      = 如何与时间做朋友

      • 答案只能是你要提前做准备。比别人更早一些开始准备，早一点，再早一点。越早一点，你与时间这个朋友之间就会变得越亲密。
      • 与投入的时间相比，聪明、勤奋之类的因素实在是微不足道。所谓时间的投入，也就是坚持不懈。

      ☆ 为什么人随着年龄的 增长会觉得时间过的越来越快？

      • 心理学家的解释：对一个5岁的孩子来说，未来的一年相当于他已经度过的人生的 20%，而对于一个50岁的人来说，未来的一年只相当于他已经度过的人生的 1/50，也即 2%。所以，从感觉上来看，随着年龄的增加，时间好像越来越快。

        听一下罗大佑的《同桌的你》 "那时候天总是很蓝，日子总是过得太慢。" 看是否会产生共鸣。

      = 几乎所有失败的青少年教育，都是因为简单粗暴而又愚昧地无视孩子们所拥有的 "现存经验" 以及 "现存经验的作用"。

      ☆ 教育孩子的方法其实很简单

      • 在某一天，做一件让他印象无比深刻的事情。而后，你自己在日记里记录清楚这件事情，当然包括时间、地点与事件，
      • 而且千万要妥善保存。大约两三年后，把这个记录翻出来，给他看，让他体会时间的快慢。

      无论如何，反正我们早晚要为自己的过去买单，这个是逃不了的。
